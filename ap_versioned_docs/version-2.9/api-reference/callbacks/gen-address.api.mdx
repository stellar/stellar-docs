If the platform is configured with `depositInfoGeneratorType: api`, the Platform make this request to the Anchor backend every time a transaction is initiated by a client application.

The Anchor must guarantee that the memo returned in this request is unique, so it can be mapped 1-to-1 to a Platform resource such as a [SEP-31](https://stellar.org/protocol/sep-31) transaction.
