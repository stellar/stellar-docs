Soroban contracts are written in [Rust]. To setup your development environment to develop contracts, you need to:

1. Install Rust
2. Create a new Rust project
3. Update the project to import the `soroban-sdk`
4. Write a test

## Install Rust

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

```sh
$ rustup update
```

[Rust]: https://www.rust-lang.org/
