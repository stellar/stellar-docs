title: Soroban Smart Contracts
---

Soroban is a smart contracts platform that is designed to integrate with and work alongside the Stellar blockchain. It is currently in Phase 1 of its phased launch on Mainnet. Read more about what to expect in each phase in this [blog post](https://stellar.org/blog/developers/the-stellar-network-s-phased-rollout-of-smart-contracts-the-road-to-mainnet).

Learn more about Soroban and access the docs here: [Soroban docs](https://soroban.stellar.org/docs).
