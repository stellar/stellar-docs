# Networks

:::info

To see the current status of Mainnet and Testnet, please see the [Stellar.org Dashboard](https://dashboard.stellar.org/) and the [Stellar Status page](https://status.stellar.org/) for network and service uptime.

:::

Read more about the different networks in the [Networks section](../learn/fundamentals/networks.mdx).

:::caution

Testnet and Futurenet can be reset at any time. A reset erases all accounts, contract data, and contract deployments from the network.

The current Testnet reset dates for 2025 are:

- March 19, 2025
- June 18, 2025
- September 17, 2025
- December 17, 2025

:::
