Reports endpoints return PDFs for wallet statements and transaction notices. The backend generates these PDFs with the [gofpdf](https://github.com/jung-kurt/gofpdf) library: A4 layout, organization name and logo, shared fonts (Inter, Google Sans Code). The statement PDF is multi-page and ledger-style; the transaction notice PDF is a single-page notice per payment.
