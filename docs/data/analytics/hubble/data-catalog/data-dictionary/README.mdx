Hubble data is organized into bronze, silver, and gold sections.

- Bronze - raw, untransformed data from the Stellar network (e.g., history_contract_events)
- Silver - decoded, transformed, and filtered data (e.g. token_transfers)
- Gold - Curated analytics with friendly aggregate tables (e.g. tvl_agg)
