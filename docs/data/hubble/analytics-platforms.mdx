## Ecosystem Data Analytics Platforms

The following is a list of data analytics platforms that make a complete historical record of Pubnet Stellar network data available.

|  | SDF Provided Data Analytics (Native) | Dune | Artemis | Nansen |
| --- | --- | --- | --- | --- |
| Description | SDF provides a publicly available dataset with a complete historical record of Stellar network data available. | Raw dataset for the stellar network maintained by Dune. This is available in Dune portal | Aggregated dataset for the Stellar network maintained by Artemis. This is available in Bigquery as well as Artemis portal | Aggregated dataset for the Stellar network maintained by Nansen. This is available in Nansen portal |
| Home page |  | https://dune.com/ | https://www.artemis.xyz/ | https://app.nansen.ai/ |
| Unique feature | Raw dataset layer of the stellar network updated every 15 mins | Raw dataset layer of the stellar network updated every hour. You can compare different chains on the platform | Provides dashboard to compare stellar chain activity with ecosystem chains. | Provides dashboard to compare stellar chain activity with ecosystem chains |
| Pricing |  |  |  |  |
| Signup page |  | [link](https://dune.com/auth/register) |  | [link](https://app.nansen.ai/auth/signup) |
| How to access raw/transformed Stellar data | Please see the [Analyst Guide](./analyst-guide) for more information. | To access datasets, click [here](https://dune.com/queries?category=canonical&namespace=stellar) | For accessing artemis transformed stellar dataset, upgrade to enterprise plan and then star [this](https://console.cloud.google.com/bigquery?ws=!1m4!1m3!3m2!1sartemis-bigquery-share!2sshared_us) dataset | For accessing nansen transformed stellar dataset, upgrade to Pioneer plan and then star [this](https://console.cloud.google.com/bigquery?project=nansen-query) dataset |
| How to access dashboard / metrics |  | Explore community dashboards, network overview and datasets [here](https://dune.com/blockchains/stellar) |  Explore [chain compare](https://app.artemisanalytics.com/chains?selectedChains=stellar) and [application activity](https://app.artemis.xyz/application-activity/stellar) dashboard | Explore [chain activity, entity and ecosystem](https://app.nansen.ai/macro/blockchains?chain=stellar&tab=overview) |
