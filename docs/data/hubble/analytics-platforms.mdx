## Ecosystem Data Analytics Platforms

The following is a list of data analytics platforms that make a complete historical record of Pubnet Stellar network data available.

| Platform | Description | Home page | Unique feature | Pricing | Signup page | Data Access | Dashboard access | Example query |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| SDF Provided Data Analytics (Native) | SDF provides a publicly available dataset with a complete historical record of Stellar network data available. |  | Raw dataset layer of the stellar network updated every 15 mins | End users incur the cost to execute queries. Visit the [BigQuery Pricing ](https://cloud.google.com/bigquery/pricing#analysis_pricing_models)Page to learn more |  | Please see the [Analyst Guide](./analyst-guide) for more information. |  |  |
| Dune | Raw dataset for the stellar network maintained by Dune. This is available in Dune portal | https://dune.com/ | Raw dataset layer of the stellar network updated every hour. You can compare different chains on the platform | Free for simpler queries. Need to pay for complex queries | [link](https://dune.com/auth/register) | To access datasets, click [here](https://dune.com/queries?category=canonical&namespace=stellar) | Explore community dashboards, network overview and datasets [here](https://dune.com/blockchains/stellar) |  |
| Artemis | Aggregated dataset for the Stellar network maintained by Artemis. This is available in Bigquery as well as Artemis portal | https://www.artemis.xyz/ | Provides dashboard to compare stellar chain activity with ecosystem chains | Free dashboards. Need to pay for accessing raw dataset | [link](https://app.artemisanalytics.com/artemis-terminal-auth?flow=SIGN_UP) | For accessing artemis transformed stellar dataset, upgrade to enterprise plan and then star [this](https://console.cloud.google.com/bigquery?ws=!1m4!1m3!3m2!1sartemis-bigquery-share!2sshared_us) dataset | Explore [chain compare](https://app.artemisanalytics.com/chains?selectedChains=stellar) and [application activity](https://app.artemis.xyz/application-activity/stellar) dashboard |  |
