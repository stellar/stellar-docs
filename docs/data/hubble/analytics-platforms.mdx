## Ecosystem Data Analytics Platforms

The following is a list of data analytics platforms that make a complete historical record of Pubnet Stellar network data available.

| Platform | Description | Home page and Signup | Pricing | Data Access | Dashboard access |
| --- | --- | --- | --- | --- | --- |
| SDF Provided Data Analytics (Native) | SDF provides a publicly available raw dataset with a complete historical record of Stellar network data, updated every 15 minutes. | N/A | End users incur the cost to execute queries. Visit the [BigQuery Pricing](https://cloud.google.com/bigquery/pricing#analysis_pricing_models) page to learn more. | Please see the [Analyst Guide](./analyst-guide) for more information. | N/A |
| Dune | Raw dataset for the Stellar network available in the Dune portal. Updated every hour.
