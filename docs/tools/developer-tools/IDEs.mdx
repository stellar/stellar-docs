# IDEs

### [Okashi](https://okashi.dev)

A web IDE for building, deploying, and testing contracts.

### [Sorobix](https://sorobix.vercel.app/)

An IDE and a web GUI where you can write, deploy, and invoke contracts on the Stellar network using Soroban.
