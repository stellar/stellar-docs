# AI Bot Stella

Stella, an AI assistant created by the SDF to help developers and general users with the Stellar blockchain and Stellar's smart contracts. Stella's knowledge base covers various aspects of the Stellar ecosystem, including:

- Stellar blockchain: Its architecture, functionality, and use cases
- Soroban: Stellar's smart contract execution environment and its capabilities
- Development resources: Documentation, SDKs, and tools for building on Stellar
- Technical concepts: Blockchain fundamentals, consensus mechanisms, and more

To access Stella, you must agree to the disclaimer first in [#stella-disclaimer](https://discord.com/channels/897514728459468821/1265827552375738369/1266084686053249094) and visit Stella bot in [#stella-help](https://discord.gg/CK36xY2C). Prompt Stella with, "Stella, ....".
