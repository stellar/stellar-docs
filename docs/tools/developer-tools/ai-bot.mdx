# Meet Stella

Your AI Assistant for Stellar development. Stella is an AI assistant to help developers and users with questions about the Stellar blockchain and smart contracts.

**Now in Beta!** We're piloting Stella to make Stellar development easier. Your [feedback](#help-us-improve) helps us improve this experience for the entire community.

You can chat with Stella right here by clicking the yellow icon at the bottom of this page, or join the #stella-help channel in the [Stellar Developer Discord](https://discord.gg/stellardev).

## Knowledge sources

Stella draws from an extensive knowledge base, including:

- Stack Overflow: Real-world solutions and community-validated answers
- SCF Handbook: Deep insights into the Stellar Community Fund processes
- YouTube content: Tutorials, demos, and conference talks from the Stellar ecosystem
- GitHub repositories: Code examples and implementation patterns
