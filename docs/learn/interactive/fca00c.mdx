Fast, Cheap, and 0ut 0f Control (FCA00C) is a collection of games and tutorials designed to familiarize you with smart contracts on the Stellar network. The Tutorial walks you through the basics of smart contract development in a series of six quests. This is a great place to get started learning about Stellar smart contracts and smart contract development.

Learn more and play all games at the [fca00c site](https://fastcheapandoutofcontrol.com).
