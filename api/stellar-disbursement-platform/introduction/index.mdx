The Stellar Disbursement Platform (SDP) enables organizations to disburse bulk payments to recipients using Stellar.

This section includes the API reference for the Stellar Disbursement Platform. To learn more about the Stellar Disbursement Platform and how to run it, check out the [documentation](/docs/stellar-disbursement-platform).
