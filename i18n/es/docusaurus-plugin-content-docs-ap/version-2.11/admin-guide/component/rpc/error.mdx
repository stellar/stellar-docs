:::tip

También haremos referencia a una variable `$transaction_id`. Esta es una identificación de la transacción que se devuelve desde el Anchor Platform en una solicitud de inicio de retirada de fondos o depósito. Puedes obtener el ID de la transacción conectando la billetera de prueba a tu instancia local de Anchor Platform.

:::
