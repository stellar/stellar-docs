# Anchor Platform

Anchor Platform es un conjunto de herramientas y API que permite a los desarrolladores y empresas crear sus propios servicios de acceso y salida para la red Stellar. Proporciona una interfaz estandarizada, incluida la implementación de varias Propuestas del Ecosistema Stellar (SEPs), para facilitar la integración de empresas con billeteras y exchanges basados en Stellar.

La plataforma también incluye funciones para gestionar activos, transacciones y cuentas de usuario, y admite una variedad de opciones de implementación, incluyendo el uso de Docker o Kubernetes. En general, Anchor Platform tiene como objetivo simplificar el proceso de crear y gestionar un servicio financiero basado en Stellar, permitiendo a las empresas centrarse en ofrecer valor a sus clientes.

Aprende sobre la integración con Anchor Platform en la [Guía del Administrador](./admin-guide/README.mdx) o obtén información de la API en la [Referencia de la API](./api-reference/README.mdx).
