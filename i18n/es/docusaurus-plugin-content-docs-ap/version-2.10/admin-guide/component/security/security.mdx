:::caution

Por defecto, los endpoints de la API de la Plataforma, como `GET /transactions` y `GET /transactions/:id`, no están protegidos y son accesibles para cualquier persona que tenga acceso al servidor, incluidas las aplicaciones de billeteras.

:::

:::info

Se recomienda mantener el servidor de la Plataforma accesible solo desde la red privada. Sin embargo, puede que desees añadir una capa adicional de protección mediante la seguridad de la API.

:::
