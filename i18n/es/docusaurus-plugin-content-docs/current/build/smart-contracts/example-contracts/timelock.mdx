El [ejemplo de tiempo de bloqueo] demuestra cómo escribir un tiempo de bloqueo e implementa un saldo reclamable muy simplificado similar a la característica de [saldo reclamable] disponible en Stellar.

[![Open in Codespaces](https://github.com/codespaces/badge.svg)][open-in-github-codespaces]

[![Abrir en Codeanywhere](https://codeanywhere.com/img/open-in-codeanywhere-btn.svg)][open-in-code-anywhere]

El contrato acepta depósitos de una cantidad de un token y permite a otros usuarios reclamarlo antes o después de un punto en el tiempo.
