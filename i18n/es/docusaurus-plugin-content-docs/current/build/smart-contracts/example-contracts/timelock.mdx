title: Tiempo de bloqueo
description: Escribir un tiempo de bloqueo que implemente una característica de saldo reclamable simplificada que acepte un depósito de una cantidad de un token y permita a otros usuarios reclamarlo.
---
El [ejemplo de tiempo de bloqueo](https://github.com/stellar/soroban-examples/tree/v23.0.0/timelock) demuestra cómo escribir un tiempo de bloqueo e implementa un saldo reclamable muy simplificado similar a la característica de [saldo reclamable](../../guides/transactions/claimable-balances.mdx) disponible en Stellar.

[![Abrir en Codespaces](https://github.com/codespaces/badge.svg)][open-in-github-codespaces](https://github.com/codespaces/new?repo=stellar/soroban-examples&editor=web)

[![Abrir en Codeanywhere](https://codeanywhere.com/img/open-in-codeanywhere-btn.svg)][open-in-code-anywhere](https://app.codeanywhere.com/#https://github.com/stellar/soroban-examples)

El contrato acepta depósitos de una cantidad de un token y permite a otros usuarios reclamarlo antes o después de un punto en el tiempo.
