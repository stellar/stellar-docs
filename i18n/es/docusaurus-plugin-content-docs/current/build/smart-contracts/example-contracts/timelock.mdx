El [ejemplo de timelock] demuestra cómo escribir un timelock e implementa un equilibrio muy simplificado similar a la característica [saldo reclamable] disponible en Stellar.

[![Abrir en Gitpod](https://gitpod.io/button/open-in-gitpod.svg)][oigp]

El contrato acepta depósitos de una cantidad de un token, y permite a otros usuarios reclamarlo antes o después de un punto.
