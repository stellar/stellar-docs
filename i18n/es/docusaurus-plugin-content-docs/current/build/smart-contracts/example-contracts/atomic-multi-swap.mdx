El [ejemplo de agrupación de intercambios atómicos](https://github.com/stellar/soroban-examples/tree/v23.0.0/atomic_multiswap) intercambia un par de tokens entre los dos grupos de usuarios que autorizaron la operación de `intercambio` del ejemplo de [Intercambio Atómico](atomic-swap.mdx).

Este contrato básicamente agrupa los múltiples intercambios mientras sigue algunas reglas simples para igualar a los participantes del intercambio.

Sigue los comentarios en el código para más información.
