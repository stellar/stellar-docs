El [ejemplo de intercambio atómico por lote] cambia un par de fichas entre los dos grupos de usuarios que autorizaron la operación `swap` del ejemplo [Atomic Swap].

Este contrato básicamente agrupa los múltiples swaps siguiendo algunas reglas simples para coincidir con los participantes de swap.

Siga los comentarios en el código para más información.
