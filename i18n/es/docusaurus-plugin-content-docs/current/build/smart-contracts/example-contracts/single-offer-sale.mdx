title: Venta de oferta única

sidebar_position: 120

El [ejemplo de venta de oferta única](https://github.com/stellar/soroban-examples/tree/v23.0.0/single_offer) demuestra cómo escribir un contrato que permite a un vendedor establecer una oferta para vender el token A por el token B a varios compradores. Los comentarios en el [código fuente](https://github.com/stellar/soroban-examples/tree/v23.0.0/single_offer/src/lib.rs) explican cómo se debe utilizar el contrato.
