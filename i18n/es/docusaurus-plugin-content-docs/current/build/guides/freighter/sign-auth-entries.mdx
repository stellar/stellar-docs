---

Para aprovechar la [autorización de contrato](../../../learn/encyclopedia/security/authorization.mdx), puedes usar la API de Freighter para firmar una entrada de autorización. Un buen ejemplo de cómo firmar una entrada de autorización puede encontrarse en el [`authorizeEntry` ayudante de `stellar-sdk`](https://github.com/stellar/js-stellar-base/blob/e3d6fc3351e7d242b374c7c6057668366364a279/src/auth.js#L97).

Como en el ayudante, puedes construir una [`HashIdPreimageSorobanAuthorization`](https://github.com/stellar/js-stellar-base/blob/a9567e5843760bfb6a8b786592046aee4c9d38b2/types/next.d.ts#L6895) y usar la representación xdr de esa estructura para llamar `await freighterApi.signAuthEntry(preimageXdr)`. Esta llamada devolverá un `Buffer` del hash firmado de la `HashIdPreimageSorobanAuthorization` pasada, que se puede utilizar para enviar a la red durante un flujo de trabajo de autorización de contrato.

Para un ejemplo completo de cómo usar la autorización de contratos, [la demo de scaffold-soroban para un intercambio atómico](https://github. om/stellar/soroban-react-atomic-swap) hace uso de la API `signAuthEntry` de la autenticación del contrato y del carguero.
