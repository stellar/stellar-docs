title: Prompt Freighter para firmar transacciones como desarrollador de dapp JS

description: Firma transacciones de contratos inteligentes usando la extensión del navegador Freighter.
---

Si estás construyendo un dapp JS, firma fácilmente las transacciones de Soroban usando la [extensión del navegador de Freighter](https://www.freighter.app/) y su biblioteca de clientes correspondiente [@stellar/freighter-api](https://www.npmjs.com/package/@stellar/freighter-api):

1. Siga las instrucciones de configuración para [conectar a Testnet](./connect-testnet.mdx), si es necesario durante el desarrollo.

2. Ahora, puedes usar `signTransaction` [method](https://docs.freighter.app/docs/guide/usingFreighterWebApp) de `@stellar/freighter-api` en tu dapp para firmar los XDR de Soroban usando la cuenta en Freighter.

3. Al llamar a `signTransaction`, Freighter abrirá y pedirá al usuario que firme la transacción. Aprobar la transacción devolverá el XDR firmado a la aplicación solicitante.
