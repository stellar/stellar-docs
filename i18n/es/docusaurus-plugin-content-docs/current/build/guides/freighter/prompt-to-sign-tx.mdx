---
title: Indicar a Freighter que firme transacciones como desarrollador de dapp JS
hide_table_of_contents: true
description: Firmar transacciones de contratos inteligentes utilizando la extensión del navegador Freighter
---

Si estás desarrollando un dapp JS, firma fácilmente transacciones Soroban utilizando la [extensión del navegador Freighter](https://www.freighter.app/) y su correspondiente biblioteca cliente [@stellar/freighter-api](https://www.npmjs.com/package/@stellar/freighter-api):

1. Sigue las instrucciones de configuración para [conectar al Testnet](./connect-testnet.mdx), si es necesario durante el desarrollo.

2. Ahora, puedes utilizar el `signTransaction` [método](https://docs.freighter.app/docs/guide/usingFreighterWebApp) de `@stellar/freighter-api` en tu dapp para firmar XDRs Soroban usando la cuenta en Freighter.

3. Al llamar a `signTransaction`, Freighter se abrirá y pedirá al usuario que firme la transacción. Aprobar la transacción devolverá el XDR firmado a la dapp solicitante.
