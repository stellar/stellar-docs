title: Firmar Soroban XDRs
hide_table_of_contents: true
description: Firmar XDRs de contratos inteligentes utilizando dapps integradas con Freighter
---

Con una cuenta de Testnet financiada, ahora puedes firmar Soroban XDRs utilizando dApps que están integradas con Freighter. Un ejemplo de una dApp integrada es el [Laboratorio](https://lab.stellar.org/transaction/sign?$=network$id=testnet&label=Testnet&horizonUrl=https:////horizon-testnet.stellar.org&rpcUrl=https:////soroban-testnet.stellar.org&passphrase=Test%20SDF%20Network%20/;%20September%202015;;) de Stellar.

1. En el firmador de transacciones del Laboratorio, introduce un Soroban XDR en el campo del formulario.

2. Haz clic en `Firmar con Freighter`.

3. Freighter se abrirá con los detalles del XDR. Haz clic en `Aprobar` para firmar o `Rechazar` para descartar sin una firma.

4. Si se aprueba, Freighter transmitirá un XDR firmado de vuelta al Laboratorio.
