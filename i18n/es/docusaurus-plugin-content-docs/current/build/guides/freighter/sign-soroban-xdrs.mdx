---
title: Firmar XDRs de Soroban
hide_table_of_contents: true
description: Firma XDRs de contrato inteligente usando dapps integrados con Freighter.
---

Con una cuenta Testnet financiada, ahora puede firmar XDR Soroban usando dApps integrados con Freighter. Un ejemplo de un dApp integrado es la [Laboratory]de Stellar (https://laboratory.stellar.org/#txsigner).

1. En el firmador de transacciones del laboratorio, introduzca un XDR Soroban en el campo del formulario.

2. Haz clic en `Firmar con Freighter`.

3. El carguero se abrirá con los detalles del XDR. Haz clic en `Aprobar` para firmar o `rechazar` para descartar sin una firma.

4. Si se aprueba, Freighter transmitirá un XDR firmado de vuelta al laboratorio.
