Las carteras son una parte esencial de cualquier dapp. Permiten a los usuarios interactuar con la cadena de bloques y firmar transacciones. En esta sección, aprenderá cómo integrar el monedero de carga en sus dapps React.

### Componente WalletData

En el [ejemplo de crowdfund dapp](https://github.com/stellar/soroban-example-dapp), el componente `WalletData` juega un papel clave en la integración de carteras. Vamos a desglosar el código y entender su funcionalidad:

```tsx title="/components/moleculres/wallet-data/index.tsx"
import React from "react";
import { useAccount, useIsMounted } from "../../../hooks";
import { ConnectButton } from "../../atoms";
import styles from "./style.module.css";

export function WalletData() {
  const mounted = useIsMounted();
  const account = useAccount();

  return (
    <>
      {mounted && account ? (
        <div className={styles.displayData}>
          <div className={styles.card}>{account.displayName}</div>
        </div>
      ) : (
        <ConnectButton label="Connect Wallet" />
      )}
    </>
  );
}
```

Aquí hay un desglose del código:

- La variable `mounted` se obtiene usando el [`useIsMounted`](https://github.com/stellar/soroban-example-dapp/blob/main/hooks/useIsMounted.ts), indicando si el componente está montado o no.
- El [hook `useAccount`](https://github.com/stellar/soroban-example-dapp/blob/main/hooks/useAccount.ts) se utiliza para obtener los datos de la cuenta del usuario, y la propiedad `data` es destruida del resultado.
- El renderizado condicional se utiliza para mostrar diferentes contenidos basados en el estado de montaje del componente y la disponibilidad de los datos de la cuenta.
- Si el componente está montado y los datos de la cuenta están disponibles, se mostrarán los datos de la cartera del usuario. Esto incluye el nombre de la cuenta.
- Si el componente no está montado o los datos de la cuenta no están disponibles, el [componente `ConnectButton`](https://github.com/stellar/soroban-example-dapp/blob/main/components/atoms/connect-button/index.tsx) es renderizado, permitiendo al usuario conectarse con Freighter.
