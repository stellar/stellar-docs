El [`contractmeta! ] macro provista en el SDK de Rust permite a los usuarios escribir dos cadenas - una `clave` y un `val` - dentro de un objeto XDR serializado `SCMetaEntry::SCMetaV0` en la sección personalizada de los contratos de Wasm. El nombre de sección para estos metadatos es `contractmetav0`. Los desarrolladores pueden utilizar esta macro para escribir metadatos, y entonces las herramientas pueden leer y mostrar esta información a los usuarios.

El [ejemplo de la batería de liquidez] proporciona una demostración clara de cómo utilizar la macro `contractmeta!`:

```rust
// ¡Metadatos que se agregan en la sección personalizada de Wasm
contractmeta!
    clave = "Descripción",
    val = "Producto constante AMM con un . % swap fee"
);

pub trait LiquidityPoolTrait {...
```
