Estas guías describen el modo "típico" en el que algo se podría conseguir en un contrato de Polvo. Estas guías no están pensadas como _prescriptiva_ como algunas otras; En cambio, sirven para destacar algunas de las normas que hemos visto cuando surgen en el desarrollo de contratos.
