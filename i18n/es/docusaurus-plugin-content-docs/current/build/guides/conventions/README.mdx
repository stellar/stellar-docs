Estas guías describen la forma "típica" en que algo podría lograrse en un contrato de Rust. Estas guías no pretenden ser tan _prescriptivas_ como algunas otras; en cambio, sirven para resaltar algunas de las normas que hemos observado cuando surgen en el desarrollo de contratos.
