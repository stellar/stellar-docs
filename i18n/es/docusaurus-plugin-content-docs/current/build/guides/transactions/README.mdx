Las transacciones de Stellar están compuestas por operaciones. Estas guías te llevarán a través de diversas operaciones de Stellar, transacciones sin contrato y transacciones con contrato inteligente. Lee más sobre operaciones y transacciones en la [sección de Operaciones y Transacciones](../../../learn/fundamentals/transactions/README.mdx).
