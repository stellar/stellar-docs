La `Dirección` es un tipo opaco que podría representar una "cuenta" en la red estelar (es decir, un par de claves) o un "contrato". Desde el punto de vista de Soroban, no importa cuál es. La variedad "cuenta" de estas direcciones se muestra típicamente como una dirección pública `G...` y la variante "contrato" se muestra normalmente como una dirección `C...`.
