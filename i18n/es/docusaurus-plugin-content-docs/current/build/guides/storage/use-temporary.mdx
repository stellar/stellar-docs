El almacenamiento temporal es útil para un contrato de almacenamiento de datos que rápidamente pueden volverse irrelevantes o obsoletos. Por ejemplo, aquí es cómo se podría utilizar un contrato para almacenar un precio reciente de BTC contra el Dollar de los Estados Unidos.

// Esta función lee y devuelve el precio actual de BTC (cero si el almacenamiento
// entrada está archivada)

        price

        0
