En el [`increment` ejemplo contrato] almacena un valor entero en una entrada del libro de valores que es identificada por una clave con el símbolo `COUNTER`. El valor de esta clave de ledger puede derivarse usando los siguientes fragmentos de código.

```python
from stellar_sdk import xdr, scval, Address

def get_ledger_key_symbol(contract_id: str, symbol_text: str) -> str:
    ledger_key = xdr. edgerKey(
        type=xdr. edgerEntryType.CONTRACT_DATA,
        contract_data=xdr. edgerKeyContractData(
            contract=Dirección(contract_id). o_xdr_sc_address(),
            key=scval. o_symbol(symbol_text),
            durabilidad=xdr.ContractDataDurability. ERSISTENTE
        ),
    )
    return ledger_key. o_xdr()

print(
    get_ledger_key_symbol(
        "CPYZFKEAXHHS5VVW5J45TOU7S2EODJ7TZNJIA5LKDVL3PESCES6FNCI",
        "COUNTER"
    )
)
```
