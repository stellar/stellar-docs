---
title: Desplegar el contrato de activos estelares para un activo estelar
hide_table_of_contents: true
description: Despliegue un SAC para un activo Stellar para que pueda interactuar con contratos inteligentes.
---

El CLI estelar puede desplegar un [Contrato de Activos Estelares] para un activo estelar de manera que cualquier contrato inteligente estelar pueda interactuar con el activo.

Cada activo Stellar ha reservado un contrato que cualquiera puede desplegar. Una vez desplegado, cualquier contrato puede interactuar con ese activo manteniendo un saldo del activo, recibiendo el activo o enviando el activo.

Desplegar el Contrato de Activos Estelares para un activo Estelar permite que ese activo se utilice en contratos inteligentes.

El Contrato de Activos Estelares puede ser desplegado para cualquier posible activo Estelar, o bien activos ya utilizados en Stellar o activos que nunca han visto ninguna actividad. Esto significa que el emisor no necesita haber sido creado, y nadie necesita tener el activo en Stellar.

Para realizar el despliegue utilice el siguiente comando:

```bash
stellar contract asset deploy \
    --source S... \
    --network testnet \
    --asset USDC:GCYEIQEWOCTSA72VPZ6LYIZIK4W4KNGJR72UADIXUXG45VDFRVCQTYE
```

El argumento `asset` corresponde al símbolo y su dirección de emisor, que es la forma en que los activos se identifican en Stellar.

Lo mismo puede hacerse con el activo nativo [Lumens]\:

```bash
stellar contract asset deploy \
    --source S... \
    --network testnet \
    --asset native
```

:::note

El despliegue del activo nativo fallará en testnet o mainnet ya que ya existe un Contrato de Activos Estelares.

:::

Para cualquier activo, la dirección del contrato se puede obtener con:

```bash
stellar contract id asset \
    --source S... \
    --network testnet \
    --asset native
```
