title: Instalar e implementar un contrato inteligente
hide_table_of_contents: true
description: Combine los comandos de instalación e implementación en el CLI Stellar para completar ambas tareas.
---

Puedes combinar los comandos `install` y `deploy` del CLI Stellar para completar ambas tareas:

```bash
stellar contract deploy \
    --source S... \
    --network testnet \
    --wasm ../relative/path/to/soroban_contract.wasm
```
