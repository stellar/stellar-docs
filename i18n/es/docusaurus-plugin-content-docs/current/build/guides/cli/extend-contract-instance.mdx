---
title: Extender la TTL de una instancia de contrato desplegada
hide_table_of_contents: true
description: Utilice el CLI para ampliar el tiempo de vida (TTL) de una instancia de contrato.
---

Puede utilizar la CLI Stellar para extender la TTL de una instancia de contrato así:

```bash
contrato estelar extender \
    --source S... \
    --network testnet \
    --id C... \
    --ledgers-to-extend 535679 \
    --durabilidad persistente
```

Este ejemplo utiliza 535,679 ledgers como el nuevo archivo TTL. Este es el valor máximo permitido para este argumento en la CLI. Esto corresponde a aproximadamente 30 días (promedio de 5 segundos de tiempo de cierre del contable).

Cuando extienda una instancia de contrato, esto incluye:

- la instancia del contrato en sí misma
- cualquier entrada `env.storage().instance()` en el contrato
- el código Wasm del contrato
