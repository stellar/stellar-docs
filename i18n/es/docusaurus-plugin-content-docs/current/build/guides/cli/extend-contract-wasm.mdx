---
title: Extiende el código de Wasm de un contrato desplegado TTL
hide_table_of_contents: true
description: Utilice Stellar CLI para extender el código de byte Wasm del contrato TTL, con o sin binario local.
---

Puede utilizar la CLI Stellar para extender la TTL de un código de byte Wasm de un contrato. Esto se puede hacer en dos formas: si usted tiene o no tiene el contrato compilado localmente. Si tiene el binario compilado en su máquina local:

```bash
contrato estelar extender \
    --source S... \
    --network testnet \
    --wasm ../relative/path/to/soroban_contract.wasm \
    --ledgers-to-extend 535679 \
    --durabilidad persistente
```

Este ejemplo utiliza 535,679 ledgers como el nuevo archivo TTL. Este es el valor máximo permitido para este argumento en la CLI. Esto corresponde a aproximadamente 30 días (promedio de 5 segundos de tiempo de cierre del contable).

Si no tiene el binario compilado en su máquina local, todavía puede utilizar el CLI para extender el TTL de código de bytes. Necesitarás conocer el hash Wasm del código del contrato instalado:

```bash
contrato estelar extender \
    --source S... \
    --network testnet \
    --wasm-hash <hex-encoded-wasm-hash> \
    --ledgers-to-extend 535679 \
    --durabilidad persistente
```
