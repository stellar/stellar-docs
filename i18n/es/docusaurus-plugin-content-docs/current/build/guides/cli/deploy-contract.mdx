Para desplegar una instancia de un contrato inteligente compilado que ya ha sido instalado en la red Stellar, utilice el comando `stellar contract deploy`:

```bash
stellar contract deploy \
    --source S... \
    --network testnet \
    --wasm-hash <hex-encoded-wasm-hash>
```
