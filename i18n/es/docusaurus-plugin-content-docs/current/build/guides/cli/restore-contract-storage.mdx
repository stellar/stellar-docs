---
title: Restaurar los datos del contrato archivados usando la CLI estelar
hide_table_of_contents: true
description: Restaurar las entradas de almacenamiento por contrato archivadas usando Stellar CLI.
---

Si la entrada de almacenamiento persistente de un contrato ha sido archivada, puede restaurarla utilizando el CLI Stellar. Para una entrada de almacenamiento que utiliza un simple `Symbol` como clave de almacenamiento, puedes ejecutar un comando así:

```bash
stellar contract restore \
    --source S... \
    --network testnet \
    --id C... \
    --key COUNTER \
    --durabilidad persistente
```

Si su entrada de almacenamiento usa una clave de almacenamiento más avanzada, como `Balance(Address)` en un contrato de tokens, necesitarás proporcionar la clave en un formulario XDR codificado en base64:

```bash
stellar contract restore \
    --source S... \
    --network testnet \
    --id C... \
    --key-xdr AAAAAHXkotywnA8z+r365/0701QSlWouXn8m0UOoshCtNHOYQAAAAAAAAAHQmFsYW5jZQAAAAAAB \
    --durabilidad persistente
```

:::info

Asegúrate de revisar nuestra [guía sobre la creación de claves de ledger XDR](../rpc/generate-ledger-keys-python.mdx) para ayudarte a generarlas.

:::
