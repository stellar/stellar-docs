Para usar el CLI de Stellar y subir un contrato inteligente compilado al ledger, usa el comando `stellar contract upload`:

```bash
stellar contract upload \
    --source S... \
    --network testnet \
    --wasm ../relative/path/to/soroban_contract.wasm
```

:::note

Nota que este comando devolverá el ID de hash del bytecode Wasm, en lugar de una dirección para una instancia de contrato.

:::
