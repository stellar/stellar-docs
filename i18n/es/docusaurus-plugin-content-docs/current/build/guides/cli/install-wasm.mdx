---

Para utilizar la CLI Stellar para instalar un contrato inteligente compilado en el contador, utilice el comando `stellar contract install`:

```bash
stellar contract install \
    --source S... \
    --network testnet \
    --wasm ../relative/path/to/soroban_contract.wasm
```

:::note

Tenga en cuenta que este comando devolverá el identificador hash del bytecode de Wasm, en lugar de una dirección para una instancia de contrato.

:::
