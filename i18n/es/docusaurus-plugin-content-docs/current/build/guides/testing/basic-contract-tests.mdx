Las funciones de prueba de un contrato se pueden utilizar como una forma simple de asegurar que las funciones de un contrato se comporten como se esperaba. El [contrato de ejemplo de incremento](../../smart-contracts/getting-started/deploy-increment-contract.mdx) tiene una función que incrementa un contador por uno en cada invocación. La prueba correspondiente invoca que funciona varias veces, asegurando con `assert_eq!(...)` el recuento aumenta como se esperaba.

```rust
#![cfg(test)]

use super::{IncrementContract, IncrementContractClient};
use soroban_sdk::{Env};

#[test]
fn test() {
    // Casi todas las pruebas comenzarán de la misma manera. Se crea un entorno Soroban predeterminado
    // y el contrato (junto con su cliente) está registrado en él.
    let env = Env::default();
    let contract_id = env. egister_contract(None, IncrementContract);
    let client = IncrementContractClient::new(&env, &contract_id);

    assert_eq!(cliente. ncrement(), 1);
    assert_eq!(client.increment(), 2);
    assert_eq!(client.increment(), 3);
}
```
