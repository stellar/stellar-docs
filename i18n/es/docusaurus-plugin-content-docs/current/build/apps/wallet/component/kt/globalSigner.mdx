Por último, con el enfoque anterior definimos el dominio del firmante y del cliente por petición. Si quieres definirlo una vez y usarlo para cada llamada de autenticación que tu aplicación está haciendo, puedes hacerlo cambiando la configuración:

<CodeExample>

```kotlin
val appCfg = ApplicationConfiguration(WalletSigner.DomainSigner("https://my-domain.com/sign"), "my-domain.com")
```

</CodeExample>

Esto es especialmente útil para la integración con múltiples fondos.
