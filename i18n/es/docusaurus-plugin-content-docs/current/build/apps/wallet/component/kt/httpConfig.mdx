Hay una configuración más disponible para una cartera que le permite configurar la lógica interna del SDK. Por ejemplo, para probar con servidores locales en un protocolo HTTP, HTTP puede ser habilitado manualmente.

<CodeExample>

```kotlin
val walletCustom = Wallet(
  StellarConfiguration.Testnet,
  ApplicationConfiguration { defaultRequest { url { protocol = URLProtocol.HTTP } } }
)
```

</CodeExample>
