### Configurando el cliente

El SDK de la cartera Typescript utiliza el [axios client](https://axios-http.com/docs/intro) para todas las solicitudes de red. Puedes leer más sobre cómo configurar el cliente [here]de los axios (https://axios-http.com/docs/instance).

Por ejemplo, podemos configurar nuestro cliente de axios para que se configure globalmente con un tiempo de espera:

<CodeExample>

```typescript
const customClient: AxiosInstance = axios. reate({
  timeout: 1000,
});
let appConfig = new ApplicationConfiguration(DefaultSigner, customClient);
let wal = new Wallet({
  stellarConfiguration: StellarConfiguration. estNet(),
  Configuración: appConfig,
});
```

</CodeExample>

Puedes encontrar más [opciones de configuración aquí.](https://axios-http.com/docs/req_config)
