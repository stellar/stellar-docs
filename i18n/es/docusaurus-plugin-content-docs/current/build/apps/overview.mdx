# Resumen

Stellar es un ledger distribuido open-source que puedes utilizar como backend para potenciar varias aplicaciones y servicios, como billeteras, aplicaciones de pago, exchanges de divisas, servicios de micropagos, plataformas para compras en el juego, y más — consulta los proyectos que se están creando en Stellar: [Proyectos del Ecosistema Stellar](https://stellar.org/ecosystem/projects#Projects).

Stellar tiene lógica incorporada para el almacenamiento de claves, crear cuentas, firmar transacciones, rastrear saldos y realizar consultas a la base de datos de Stellar, y cualquier persona puede usar la red para emitir, almacenar, transferir e intercambiar activos.

Esta documentación incluye secciones sobre cómo crear aplicaciones sin contratos inteligentes con el [Wallet SDK](./wallet/overview.mdx) o el [JS SDK](./example-application-tutorial/overview.mdx), construir con contratos inteligentes con el [tutorial de frontend dapp](./dapp-frontend.mdx), y toda la información relacionada con la experimentación con [billeteras inteligentes](./smart-wallets.mdx).

## Anclas

Muchos activos de Stellar se conectan a monedas del mundo real, y Stellar tiene protocolos abiertos para integrar depósitos y retiradas de estos activos a través de la [red de anclajes](https://stellar.org/learn/anchor-basics). Debido a esto, una aplicación basada en Stellar puede aprovechar las vías bancarias reales y conectarse a dinero real.

Lee más sobre los anclajes en nuestra [sección de Anclajes](../../learn/fundamentals/anchors.mdx).

Configura un anclaje utilizando la [Anchor Platform](/platforms/anchor-platform).

Integra MoneyGram Ramps en una aplicación existente con el [tutorial Integrar con MoneyGram Ramps](https://developer.moneygram.com/moneygram-developer/docs/integrate-moneygram-ramps).

## Propuestas del Ecosistema Stellar (SEPs)

Los productos y servicios basados en Stellar interoperan implementando varias Propuestas del Ecosistema Stellar (SEPs), que son documentos abiertos, creados públicamente, que se encuentran en un [repositorio de GitHub](https://github.com/stellar/stellar-protocol/tree/master/ecosystem#stellar-ecosystem-proposals-seps) y definen cómo interactúan entre sí los emisores de activos, anclajes, billeteras y otros proveedores de servicios.

Como billetera, los SEPs más importantes son [SEP-24: Depósito y Retirada Alojada](https://github.com/stellar/stellar-protocol/blob/master/ecosystem/sep-0024.md), y [SEP-31: API de Pagos Transfronterizos](https://github.com/stellar/stellar-protocol/blob/master/ecosystem/sep-0031.md), [SEP-10: Autenticación Stellar](https://github.com/stellar/stellar-protocol/blob/master/ecosystem/sep-0010.md), [SEP-12: API de KYC](https://github.com/stellar/stellar-protocol/blob/master/ecosystem/sep-0012.md), y [SEP-38: API RFQ de Anclaje](https://github.com/stellar/stellar-protocol/blob/master/ecosystem/sep-0038.md).
