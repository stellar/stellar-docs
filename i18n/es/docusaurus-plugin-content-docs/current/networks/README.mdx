# Redes

:::info

Para ver el estado actual de Mainnet y Testnet, consulta el [Tablero de Stellar.org](https://dashboard.stellar.org/) y la [página de estado de Stellar](https://status.stellar.org/) para el tiempo de actividad de la red y del servicio.

:::

:::caution

Testnet y Futurenet pueden ser reiniciados en cualquier momento. Un reinicio borra todas las cuentas, datos de contratos y despliegues de contratos de la red.

Las fechas actuales de reinicio de Testnet para 2025 son:

- 19 de marzo de 2025
- 18 de junio de 2025
- 14 de agosto de 2025
- 17 de diciembre de 2025

:::
