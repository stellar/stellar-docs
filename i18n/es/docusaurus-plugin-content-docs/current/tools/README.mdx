Esta sección de los documentos proporcionará enlaces a los SDKs y herramientas de desarrollo para el desarrollo en Stellar, así como documentación para varias plataformas mantenidas por SDF, como la Plataforma de Anchor y la Plataforma de Desembolso Estelar (SDP).

## [SDKs](./sdks/library.mdx)

Los Kits de desarrollo de software (SDKs) de Stellar proporcionan a los desarrolladores las herramientas, bibliotecas y documentación para interactuar y desarrollar en la cadena de bloques. Simplifican tareas como la creación y el despliegue de contratos inteligentes y el envío de transacciones, al tiempo que ofrecen API para acceder a datos e integrar funcionalidades en las aplicaciones.

## [Herramientas de desarrollador](./develop-tools/README.mdx)

Encuentre herramientas de desarrollo mantenidas por el ecosistema y SDF que ayuden a agilizar el proceso de desarrollo de aplicaciones y contratos inteligentes en Stellar.

## Plataformas SDF

### [Plataforma de Anchor](/plataforms/anchor-platform)

La Plataforma de Anchor es un conjunto de herramientas y APIs que permiten a desarrolladores y empresas construir sus propios servicios para la red Stellar. Proporciona una interfaz estandarizada, incluyendo la implementación de varias propuestas estelares del ecosistema (SEPs), para facilitar a las empresas la integración con las carteras y los intercambios basados en Stellar.

### [Plataforma de desembolso estelar (SDP)](/platforms/stellar-disbursement-platform)

La Plataforma de Desembolso Estelar (SDP) es una herramienta creada para que las organizaciones realicen pagos masivos a un grupo de destinatarios a través de la red Stellar.
