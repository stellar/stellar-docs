# Resumen de Herramientas

Esta sección de la documentación proporcionará enlaces a los SDK y herramientas para desarrolladores para su uso al desarrollar en Stellar, así como documentación para varias plataformas mantenidas por SDF, como la Anchor Platform y la Stellar Disbursement Platform (SDP).

## Herramientas para Desarrolladores

### [SDKs](./sdks/README.mdx)

Los Kits de Desarrollo de Software (SDK) de Stellar proporcionan a los desarrolladores las herramientas, bibliotecas y documentación para interactuar y desarrollar en la blockchain. Simplifican tareas como crear y desplegar contratos inteligentes y enviar transacciones, además de ofrecer APIs para acceder a datos e integrar funcionalidades en aplicaciones.

### [Stellar CLI](./cli/README.mdx)

La interfaz de línea de comandos para contratos inteligentes de Soroban. Te permite crear, desplegar e interactuar con contratos inteligentes; configurar identidades; generar pares de claves; gestionar redes; y más.

### [Laboratorio](./lab/README.mdx)

Stellar Lab es nuestra nueva herramienta de referencia para el desarrollo, experimentación y pruebas, así como para explorar APIs que los desarrolladores utilizan para interactuar con la red Stellar.

### [Inicio Rápido](./quickstart/README.mdx)

Inicio rápido es un entorno de red Stellar local (nodo) que permite a los desarrolladores ejecutar una versión local de la red Stellar para el desarrollo y pruebas.

### [Contratos OpenZeppelin](./openzeppelin-contracts.mdx)

OpenZeppelin Stellar Contracts es una colección de contratos auditados y utilidades para Stellar. Los contratos son desarrollados por OpenZeppelin en colaboración con la comunidad Stellar y la Stellar Development Foundation (SDF), con el objetivo de ofrecer una biblioteca de contratos de alta calidad y auditados que puedan usarse para construir aplicaciones en la red Stellar.

### [Scaffold Stellar](./scaffold-stellar.mdx)

Scaffold Stellar es un kit de herramientas para desarrolladores para construir aplicaciones descentralizadas (dApps) y contratos inteligentes en Stellar. Proporciona herramientas CLI, plantillas de contratos reutilizables, un registro de contratos inteligentes y un frontend moderno.

### [Más Herramientas para Desarrolladores](./developer-tools/README.mdx)

Encuentra otras herramientas para desarrolladores mantenidas por SDF y el ecosistema que ayudan a agilizar el proceso de desarrollo para aplicaciones y contratos inteligentes en Stellar.

## Plataformas de SDF

### [Plataforma Anchor](../platforms/anchor-platform/README.mdx)

La Anchor Platform es un conjunto de herramientas y APIs que permiten a los desarrolladores y empresas crear sus propios servicios de acceso y salida para la red Stellar. Proporciona una interfaz estandarizada, incluyendo la implementación de varias Propuestas del Ecosistema Stellar (SEPs), para facilitar a las empresas la integración con billeteras y exchanges basados en Stellar.

### [Stellar Disbursement Platform (SDP)](../platforms/stellar-disbursement-platform/README.mdx)

La Stellar Disbursement Platform (SDP) es una herramienta desarrollada para que las organizaciones realicen pagos masivos a un grupo de beneficiarios a través de la red Stellar.
