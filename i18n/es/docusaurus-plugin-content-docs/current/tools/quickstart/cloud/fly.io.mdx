# Fly.io

Desplegar en Fly.io siguiendo los pasos a continuación.

Instala la CLI de Fly. Consulta [fly.io/docs/flyctl/install](https://fly.io/docs/flyctl/install) para las instrucciones.

Inicia la imagen:

<Tabs groupId="platform" defaultValue={getPlatform()}>
<TabItem value="unix" label="macOS/Linux">

```shell
fly launch \
    --image docker.io/stellar/quickstart:latest \
    --ha=false \
    --env NETWORK=local \
    --env LIMITS=testnet \
    --env NETWORK_PASSPHRASE="My Network" \
    --internal-port 8000
```

</TabItem>
<TabItem value="windows" label="Windows (PowerShell)">

```powershell
fly launch `
    --image docker.io/stellar/quickstart:latest `
    --ha=false `
    --env NETWORK=local `
    --env LIMITS=testnet `
    --env NETWORK_PASSPHRASE="My Network" `
    --env RANDOMIZE_NETWORK_PASSPHRASE=false `
    --internal-port 8000
```

</TabItem>
