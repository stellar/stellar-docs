# Grifo

Quickstart utiliza Friendbot como un grifo de activos nativos para testnet, futurenet y redes locales, igual que se usa Friendbot en el testnet público y futurenet.

Friendbot está disponible en `:8000/friendbot` y se puede usar para financiar una nueva cuenta. Por ejemplo:

```sh
curl http://localhost:8000/friendbot?addr=G...
```

En modo de red local, se está ejecutando un Friendbot local. En modos de red testnet y futurenet, las solicitudes al `:8000/friendbot` endpoint se redirigirán a las implementaciones de Friendbot para la red respectiva.
