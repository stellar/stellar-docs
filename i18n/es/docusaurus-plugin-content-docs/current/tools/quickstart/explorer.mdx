Cuando ejecutas **Stellar Lab** mediante **Quickstart**, un explorador ligero de bloques/transacciones está disponible automáticamente junto a Lab. Esto facilita inspeccionar la actividad de la red en tu red local de Quickstart mientras construyes y pruebas.

### URLs

**Lab (inicio):** `http://localhost:8000/lab`

**Explorador:** `http://localhost:8000/lab/transactions-explorer`

Si usas un host o puerto personalizados al iniciar Quickstart, reemplaza `localhost:8000` por tu dirección elegida.

### Qué puedes hacer con el Explorador

- Examinar las transacciones recientes en tu red Quickstart.
- Explorar una transacción para ver sus detalles (hash, resultado, operaciones involucradas, etc.).
- Usarlo junto con las herramientas de Lab (por ejemplo, construir/firma de transacciones) para verificar rápidamente los resultados de la red local.

### Resumen

1. Inicia Quickstart (según las instrucciones del repositorio).
2. Abre Lab: `http://localhost:8000/lab`.
3. Abre el Explorador: `http://localhost:8000/lab/transactions-explorer`.
4. Construye y envía transacciones en Lab, luego cambia al Explorador para confirmar que se registraron.
