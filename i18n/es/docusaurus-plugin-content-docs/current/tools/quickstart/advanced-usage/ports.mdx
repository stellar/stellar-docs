# Puertos

Quickstart expone un puerto principal a través del cual los servicios proporcionan sus APIs:

| Puerto | Servicio                         | Descripción           |
| ------ | -------------------------------- | --------------------- |
| 8000   | horizon, stellar-rpc, friendbot  | puerto http principal |

Quickstart también expone varios otros puertos que la mayoría de los desarrolladores no necesitan, pero están disponibles:

| Puerto  | Servicio                    | Descripción                         |
| ------- | --------------------------- | ----------------------------------- |
| 5432    | postgresql                  | puerto de acceso a la base de datos |
| 6060    | horizon                     | puerto de administración            |
| 6061    | stellar-rpc                 | puerto de administración            |
| 11625   | stellar-core                | puerto de nodo usuario              |
| 11626   | stellar-core                | puerto http principal               |
| 11725   | stellar-core (horizon)      | puerto de nodo usuario              |
| 11726   | stellar-core (horizon)      | puerto http principal               |
| 11825   | stellar-core (stellar-rpc)  | puerto de nodo usuario              |
| 11826   | stellar-core (stellar-rpc)  | puerto http principal               |
