# Uso Avanzado

Quickstart funciona directamente, y cuando se lanza usando el CLI de Stellar, no se requiere configuración. Aunque la configuración estándar puede funcionar para muchos casos de uso, puede que no se ajuste a las necesidades de todos. Si se necesita una configuración personalizada, el contenedor de Quickstart se puede ejecutar usando el CLI de docker.
