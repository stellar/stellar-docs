# Opciones del Servicio

Todos los modos de red ejecutan todos los servicios (Core, Horizon y RPC) por defecto, pero la opción --enable se puede usar para habilitar solo algunos de los servicios. El comportamiento de la opción habilitar es ligeramente diferente en modo de red local.

La opción toma una lista de nombres de servicio separados por comas para habilitar. Para habilitar todos los servicios, que es el comportamiento por defecto, usa:

```sh
--enable core,horizon,rpc
```

Para ejecutar solo servicios seleccionados, simplemente especifica solo esos servicios. Por ejemplo, para habilitar el RPC, usa:

```sh
 --enable rpc
```

:::info

En modo de red local, el servicio principal siempre se ejecuta sin importar qué opciones se pasen, el servicio de grifo Friendbot se ejecuta siempre que Horizon está en funcionamiento, y Horizon se ejecuta cuando se solicita RPC para que Friendbot esté disponible.

:::
