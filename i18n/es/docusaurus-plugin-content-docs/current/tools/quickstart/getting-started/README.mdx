# Comenzar

En esta guía, configurarás Quickstart usando el Stellar CLI. Quickstart usará la configuración estándar, que funcionará para muchos casos de uso. Para una configuración más personalizada de Quickstart, consulta la sección de Uso Avanzado.

## Requisitos Previos

Antes de seguir los pasos de esta guía, deben estar instalados y configurados los siguientes elementos en tu sistema local:

- [Stellar CLI](../../../build/smart-contracts/getting-started/setup.mdx#install-the-stellar-cli)
- [Docker](https://docs.docker.com/get-started/get-docker/)

## Iniciar Quickstart

Cuando uses el Stellar CLI para iniciar Quickstart, la imagen de docker se descargará automáticamente, y se aplicará la configuración estándar. Lo único que necesitas hacer para empezar es ejecutar este comando:

```sh
stellar container start local
```

En este ejemplo, se elige el modo de red local, consulta la sección de Modos de Red para más información sobre los modos de red disponibles. Después de ejecutar el comando, Quickstart estará funcionando en tu sistema local. En el siguiente paso, usarás Stellar Lab para verificar que Quickstart está funcionando correctamente.
