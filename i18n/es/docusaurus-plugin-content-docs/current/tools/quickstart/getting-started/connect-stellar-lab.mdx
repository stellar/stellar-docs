# Conectar Stellar Lab

Una forma de confirmar que Quickstart se inició correctamente y está disponible es conectar a Quickstart desde [Stellar Lab](https://lab.stellar.org).

Navega a [Stellar Lab](https://lab.stellar.org) y en la esquina superior derecha, usa el menú desplegable para seleccionar la red personalizada. Esto te permitirá proporcionar los detalles de conexión — utiliza estos detalles de testnet:

- URL RPC: `http://localhost:8000/rpc`
- URL Horizon: `http://localhost:8000`
- Frase de contraseña de red: `Standalone Network ; February 2017`

![Stellar Lab](/assets/quickstart/stellar_lab_screenshot.png)

Cambia a la red personalizada y puedes comenzar a interactuar con Quickstart. Una buena primera tarea podría ser navegar a la pestaña de Cuenta, crear una cuenta y financiarla con Friendbot (faucet).
