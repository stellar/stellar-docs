# Exploradores de Bloques

Los exploradores de bloques existen para mostrar públicamente los datos de la blockchain de una manera fácilmente digerible. Se pueden explorar con un navegador web ordinario y no requieren habilidades especiales de desarrollador para usarlos. Los exploradores de bloques disponibles para Soroban indexan datos relacionados con pagos, cuentas, contratos desplegados, historial de transacciones y más.

### [StellarExpert](https://stellar.expert/explorer/public)

Explora transacciones y actividad de la red con StellarExpert. Consulta las estadísticas específicas de un código de activo, hash de transacción, dirección de cuenta o número de secuencia de ledger. No disponible para Futurenet.

### [StellarChain](https://stellarchain.io/)

Explora transacciones y actividad de la red para las redes de Stellar, incluyendo Futurenet.

### [Stellar Explorer](https://steexp.com/)

Consulta datos relacionados con pagos, cuentas, contratos desplegados y más para el Futurenet, Testnet y Mainnet de Stellar.
