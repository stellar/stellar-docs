# Estado de la Red

### [Tablero](https://dashboard.stellar.org/)

Muestra el estado actual de Testnet y Mainnet. Monitorea estadísticas de tarifas, operaciones recientes, suministro de lumen y más.

### [Stellarbeat](https://stellarbeat.io/)

Visualiza los nodos de la red Stellar y el consenso.

### [Página de Estado](https://status.stellar.org/)

Rastrea incidentes de la red y mantenimiento programado para Testnet y Mainnet. Suscríbete a actualizaciones para ser notificado sobre eventos importantes, incluyendo mejoras de protocolo y reinicios de Testnet.
