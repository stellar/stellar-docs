# Herramientas para desarrolladores

## Herramientas de Anclaje

### [Directorio de Anclas](https://anchors.stellar.org/?)

Ver todos los anclos de Stellar, sus monedas y donde operan.

### [Monedero Demo](https://demo-wallet.stellar.org/)

Una aplicación para probar interactivamente servicios de fondeo. Permite que los desarrolladores de aplicaciones financieras prueben sus integraciones y aprendan cómo funcionan los protocolos de los ecosistemas Stellar (SEP); usar la cartera demo para probar Activos Regulados (SEP-8), Depósito Alojado y Retiro (SEP-24), y Cross-Border Payments (SEP-31) con cualquier dominio de casa que tenga un archivo de información estelar (también conocido como SEP-1, o un estelar. archivo oml).

### [Polaris](https://django-polaris.readthedocs.io/es/stable/)

Una aplicación extensible django que facilita a los ancladores facilitar los pagos transfronterizos y habilitar depósitos y retiros; ejecutar un servidor web que soporte cualquier combinación de SEP-1, 6, 10, 12 y 24.

## Herramientas de activos

### [Sandbox de activos](https://stellar.cheesecakelabs.com/?utm_source=stellar&utm_medium=landing-page&utm_campaign=stellar-asset-sandbox)

Un sandbox soportado por SDF y Cheesecake Labs para que las empresas experimenten con la emisión de activos en la red de pruebas de Stellar.

## Bloquear exploradores

Los exploradores de bloques existen para mostrar públicamente los datos de la cadena de bloques de una manera fácilmente digestble. Pueden ser consultados con un navegador web ordinario, y no requieren ninguna habilidad especial de desarrollador para usar. Los exploradores de bloque disponibles para los datos del índice Soroban relacionados con pagos, cuentas, contratos desplegados, historial de transacciones y más.

### [StellarExpert](https://stellar.expert/explorer/public)

Explora las transacciones y la actividad de la red con StellarExpert. Compruebe las estadísticas específicas de un código de activo, el hash de la transacción, la dirección de la cuenta o el número de secuencia del contador. No disponible para Futurenet.

### [StellarChain](https://stellarchain.io/)

Explore las transacciones y la actividad de la red para las redes de Stellar, incluyendo Futurenet.

### [Explorador soroban](https://soroban.nownodes.io/)

Construido específicamente para Soroban, usted puede explorar las transacciones y la actividad de red para Stellar Testnet y Futurenet (Mainnet próximamente).

### [Explorador Stellar](https://steexp.com/)

Revisar datos relacionados con pagos, cuentas, contratos desplegados y más para Futurenet, Testnet y Mainnet de Stellar.

### [Ortege](https://www.ortege.ai/)

Ortege es una herramienta de análisis y análisis de cadena de bloques impulsada por AIs para Stellar, haciendo que los datos complejos sean simples y accesibles. Accede gratis registrándote [here](https://app.ortege.ai/login/).

## CLI

### [Stellar CLI](https://github.com/stellar/stellar-cli)

La interfaz de línea de comandos a los contratos inteligentes de Soroban. Permite construir, desplegar e interactuar con contratos inteligentes; configurar identidades; generar pares de claves; administrar redes y mucho más.

Instale Stellar CLI como se explica en [Setup](../../build/smart-contracts/getting-started/setup.mdx#install-the-stellar-cli). Para ver ejemplos sobre cómo usar el CLI Stellar, por favor vea la [Guía Stellar CLI](../../build/guides/cli/README.mdx).

La documentación de referencia completa generada automáticamente está disponible [here](../stellar-cli.mdx).

## Indexadores de Datos

Para potenciar sus aplicaciones, estará disponible una gama de servicios de datos desde los índices de datos con interfaces de consultas. Estos índices le permitirán almacenar y recuperar fácilmente los datos del contador descodificado, las emisiones de eventos por contrato y más. Puedes concentrar más energía en hacer que tu proyecto tenga éxito, y pierda menos tiempo averiguando exactamente cómo obtener los datos importantes de la red.

### [SubQuery](https://subquery.network/)

Un índice de datos de código abierto que soporta tanto Stellar como Soroban. SubQuery permite a los usuarios crear una API personalizada para su dapp o contrato inteligente y proporciona datos indexados limpios a cualquier servicio. Usted puede indexar contadores, transacciones, operaciones y efectos de Stellar y transacciones y eventos de Soroban a su propia base de datos de Postgres. Lea la guía de inicio rápido aquí.

### [Mercury](https://mercurydata.app/)

Un servicio de indexación de red que proporciona un conjunto de herramientas para ayudarle a comenzar a trabajar con los datos de la red Stellar.

### [BlockEden](https://blockeden.xyz/stellar-soroban/)

Un portal web de todo Soroban RPC, indexador API GraphQL, y análisis de datos.

### [Goldsky](https://goldsky.com/)

Mirror e indexador próximamente.

### [Ankr](https://www.ankr.com/)

También ofrece instancias RPC y Horizon.

## GUIs

### [Laboratory](https://laboratory.stellar.org/#?network=test)

Una GUI donde se puede utilizar y aprender sobre la red Stellar en el Testnet y Pubnet. Puede crear cuentas, construir y enviar transacciones, y consultar puntos finales de Horizon sin necesidad de conocer código.

### [Okashi](https://okashi.dev)

Una GUI web para construir, desplegar y probar contratos.

## IDEs

### [Sorobix](https://sorobix.vercel.app/)

Un IDE donde puede escribir, desplegar e invocar contratos en la red Stellar usando Soroban.

## Estado de la red

### [Dashboard](https://dashboard.stellar.org/)

Muestra el estado actual de la red Testnet y la red principal. Seguimiento de estadísticas de tasas, operaciones recientes, suministro de lumen y más.

### [Stellarbeat](https://stellarbeat.io/)

Ver nodos de red estelar y visualizar consenso.

### [Página de Estado](https://status.stellar.org/)

Seguimiento de incidentes de red y mantenimiento programado para Testnet y Mainnet. Suscríbete a las actualizaciones para ser notificado sobre eventos importantes, incluyendo actualizaciones de protocolo y reinicios de Testnet.

## Recursos de Contrato Inteligente

### [Soroban Copilot](https://github.com/mootz12/soroban-copilot)

Una colección de paquetes no_std Rust enfocados en facilitar el desarrollo de Soroban.

### [Soroban React](https://github.com/esteblock/soroban-react)

Un framework simple y potente para construir aplicaciones modernas de Soroban usando React.

### [Dev](https://sorobandev.com/)

Guía de un desarrollador web para Soroban. Contiene guías para aprender Soroban y Rust, herramientas de desarrollo, aplicaciones de ejemplo y más.

## Herramientas de Operador de Núcleo y Horizón

### [GitHub Repository](https://github.com/stellar/go/tree/master/tools)

Un repositorio de GitHub con herramientas como Archivista Stellar (para mantenimiento del archivo Stellar Coree) y Cmp Horizon (compara las respuestas de dos servidores Horizon).

## Pruebas

### [Keizai](https://keizai.dev/)

Keizai es una aplicación de código abierto inspirada por postman, diseñada para simplificar y elevar el proceso de pruebas para desarrolladores que trabajan con contratos inteligentes de Soroban.

## Carteras

### [Stellar Wallet Kit](https://github.com/Creit-Tech/Stellar-Wallets-Kit/)

Un kit de cartera para manejar la integración a múltiples carteras del ecosistema estelar con una API simple. Aprende más sobre cómo integrar esta biblioteca en [la documentación del Kit del Monedero Stellar](https://stellarwalletskit.dev/).

### [Visor de cuenta](https://accountviewer.stellar.org/)

Una billetera reducida donde puede comprobar el saldo XLM de una cuenta y enviar pagos sencillos en Testnet y Mainnet.

### [Albedo](https://albedo.link/)

Una cartera web para la firma de transacciones estelares y la verificación de identidad.

### [Freighter](https://www.freighter.app/)

La extensión del monedero no custodial de SDF que permite a los usuarios firmar transacciones Stellar a través de su navegador.
