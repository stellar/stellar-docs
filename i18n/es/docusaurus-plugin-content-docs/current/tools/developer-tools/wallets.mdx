# Integración de Billetera

### [Stellar Wallet Kit](https://github.com/Creit-Tech/Stellar-Wallets-Kit/)

Un kit de billetera fácil de usar para gestionar la integración a múltiples billeteras del ecosistema Stellar. Aprende más sobre cómo integrar esta biblioteca desde [la documentación del Kit de Billeteras Stellar](https://stellarwalletskit.dev/).

![Kit de Billeteras Stellar](/assets/dev-tools/stellar-wallet-kit.gif)

El Kit de Billeteras Stellar admite muchas billeteras, incluyendo:

- Albedo
- Freighter
- Hana
- Billetera Hardware Ledger
- Billetera Hardware Trezor
- Lobstr
- Rabet
- WalletConnect
- xBull
- Billetera caliente

### [Account Viewer](https://accountviewer.stellar.org/)

Una billetera simplificada donde puedes verificar el saldo de XLM de una cuenta y enviar pagos simples en Testnet y Mainnet. El [código fuente](https://github.com/stellar/account-viewer-v2) de Account Viewer también es útil e instructivo para los desarrolladores que están desarrollando funciones relacionadas con billeteras en sus aplicaciones.

### [Billetera Freighter](https://www.freighter.app/)

La extensión de billetera no custodial insignia de SDF que permite a los usuarios firmar transacciones Stellar a través de su navegador. El código de la billetera Freighter es de código abierto aquí: https://github.com/stellar/freighter.

### [Dfns](https://www.dfns.co/)

Una plataforma de billetera como servicio que optimiza las operaciones de activos digitales, ofreciendo soporte completo para funciones de billetera, incluyendo la detección automática de saldos de activos y NFT (cuando corresponda), historial de transferencias on-chain, transferencias de activos, transmisión de transacciones y generación de firmas.

Consulta la [documentación](https://docs.dfns.co/d/api-docs/wallets/broadcast-transaction/stellar).

### [Privy](https://docs.privy.io/wallets/overview/chains)

Infraestructura de billetera que permite a los usuarios crear programáticamente una nueva billetera dentro de sus aplicaciones.

Sigue esta [guía](https://docs.privy.io/wallets/wallets/create/from-my-server) para crear una billetera Stellar.
