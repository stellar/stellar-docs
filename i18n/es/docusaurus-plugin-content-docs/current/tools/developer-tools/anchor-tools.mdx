# Herramientas de Anchor

### [Directorio de Anclajes](https://anchors.stellar.org/?)

Visualiza todos los anclajes en Stellar, sus monedas y dónde operan.

### [Billetera de Demostración](https://demo-wallet.stellar.org/)

Una aplicación para probar interactivamente los servicios de anclaje. Permite a los desarrolladores de aplicaciones financieras probar sus integraciones y aprender cómo funcionan los protocolos del ecosistema Stellar (SEPs); utiliza la billetera de demostración para probar Activos Regulados (SEP-8), Depósito y Retirada de Fondos (SEP-24), y Pagos Transnacionales (SEP-31) con cualquier dominio local que tenga un Archivo de Información de Stellar (también conocido como SEP-1, o un archivo stellar.toml).

### [Suite de Pruebas de Anchor](https://anchor-tests.stellar.org/)

Una suite de pruebas para validar los servidores de transferencia SEP6, SEP24, SEP31.
