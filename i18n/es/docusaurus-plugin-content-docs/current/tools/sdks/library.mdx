Interactúe con la red Stellar utilizando el SDK en su idioma preferido. Los SDKs de JavaScript, Go y Rust son mantenidos por SDF y el resto son mantenidos por desarrolladores dedicados a la comunidad. Todos los SDKs son de código abierto; archivan un problema de GitHub o pull request en el repositorio SDK específico si tiene preguntas o sugerencias.

Cada SDK tiene su propio código fuente y documentación. Aprenda cómo utilizar un SDK específico refiriéndose a la documentación- la mayoría de los documentos ofrecen ejemplos prácticos que demuestran cómo construir y enviar transacciones e interactuar con los puntos finales de Horizon.

### Polvo Soroban SDK

[Rust SDK](https://github.com/stellar/rs-soroban-sdk) | [Docs](https://docs.rs/soroban-sdk)

La caja `soroban-sdk` Rust contiene el Soroban Rust SDK.

Reporta problemas y comparte comentarios sobre el `soroban-sdk` [here](https://github.com/stellar/rs-soroban-sdk/issues/new/choose).

**Añade `soroban-sdk` como dependencia** usando [crates.io](https://crates.io/crates/soroban-sdk) para encontrar la versión de la versión más reciente de SDK.

Añade las siguientes secciones al `Cargo.toml` para importar el `soroban-sdk` y reemplaza `$VERSION` con la versión liberada.

```toml
[dependencies]
soroban-sdk = $VERSION

[dev_dependencies]
soroban-sdk = { version = $VERSION, features = ["testutils"] }
```

### Colmena SDK

[SDK](https://github.com/Soneso/as-soroban-sdk)

El `as-soroban-sdk` es un SDK de código abierto que soporta la escritura de programas para la plataforma Soroban mediante el uso del lenguaje de programación AssemblblyScript.

La Colmena Soroban SDK es mantenida por Soneso. Informar problemas y compartir comentarios [here](https://github.com/Soneso/as-soroban-sdk/issues/new).

### SDK JavaScript

[JavaScript SDK](https://github.com/stellar/js-stellar-sdk) | [Docs](https://stellar.github.io/js-stellar-sdk/) | [NPM](https://www.npmjs.com/package/@stellar/stellar-sdk)

`stellar-sdk` es la biblioteca JavaScript para comunicarse con un servidor RPC Soroban, comunicándose con la API Horizon, y construyendo transacciones en la red Stellar. Se utiliza para construir aplicaciones Stellar, ya sea en Node.js o en el navegador.

Proporciona:

- Una API de capa de red para los métodos RPC Soroban y la API Horizon.
- Instalaciones para construir y firmar transacciones, para comunicarse con una instancia RPC, para comunicarse con una instancia de Horizon y para enviar transacciones o consultar el estado de la red.

### Python SDK

[Python SDK](https://github.com/StellarCN/py-stellar-base) | [Docs](https://stellar-sdk.readthedocs.io/en/latest/) | [Examples](https://github.com/StellarCN/py-stellar-base/tree/master/examples)

`py-stellar-base` es una biblioteca de Python para comunicarse con un servidor Stellar Horizon. Se utiliza para construir aplicaciones Stellar en Python. Soporta Python 3.7+ y PyPy 3.7+.

Este SDK es mantenido por un desarrollador dedicado a la comunidad.

Proporciona:

- Una API de capa de red para los puntos finales de Horizon.
- Instalaciones para construir y firmar transacciones, para comunicarse con un Horizon Estelar, y para enviar transacciones o consultar el historial de la red.

[iOS SDK](https://github.com/Soneso/stellar-ios-mac-sdk) | [Docs](https://github.com/Soneso/stellar-ios-mac-sdk/tree/master/docs) | [Smart Contract Docs](https://github.com/Soneso/stellar-mac-sdk/blob/master/soroban.md)

### Ir

Este SDK se divide en paquetes separados, todos los cuales puedes encontrar en [Go monorepo README](https://github.com/stellar/go/blob/master/docs/reference/readme.md). Las dos bibliotecas clave para interactuar con Horizon son `txnbuild`, que permite la construcción, firma y codificación de transacciones Stellar, y `horizonclient`, que proporciona un cliente web para interactuar con los endpoints de REST del servidor Horizon para recuperar información de ledger y enviar transacciones construidas con `txnbuild`.

- txnbuild: [SDK](https://github.com/stellar/go/tree/master/txnbuild) | [Docs](https://godoc.org/github.com/stellar/go/txnbuild)
- Horizonclient: [SDK](https://github.com/stellar/go/tree/master/clients/horizonclient) | [Docs](https://godoc.org/github.com/stellar/go/clients/horizonclient)
