# Métodos RPC

Los métodos RPC son un conjunto de funciones que permiten a los desarrolladores interactuar directamente con la red Stellar. Puedes aprender más sobre todos los métodos RPC [aquí](../../../data/apis/rpc/api-reference/methods). El Laboratorio proporciona una manera fácil de utilizar estos métodos desde la interfaz de usuario. Para acceder a los elementos "RPC Methods", haz clic en el enlace "API Explorer" en la navegación principal, luego haz clic en el enlace del submenú "RPC Methods", que abrirá otro submenú que enumera todos los métodos RPC por nombre. Haz clic en el nombre del método para abrir esa página.

:::warning

Se requiere la URL de RPC para enviar estos métodos. Puedes actualizar o establecer la URL RPC en el selector de red en la esquina superior derecha.

:::

## Ejemplo

Seleccionaremos el método `getLedgerEntries` como ejemplo. En la parte superior de la página hay un enlace "Ver Documentos" para aprender más sobre este método. El campo de entrada de solo lectura muestra la URL de RPC (verás una advertencia para proporcionar la URL de RPC si el campo de entrada está vacío). Junto al campo de entrada hay un botón de Enviar, que estará desactivado hasta que todos los datos requeridos estén llenos y sean válidos. Los botones Compartir y Guardar están junto al botón de Enviar. Más sobre ellos abajo.

Hay dos maneras de ingresar la Clave del Ledger: cadena XDR Base64 o entradas manuales. Puedes cambiar el modo de entrada haciendo clic en el botón "Cambiar entrada". Aquí hay un ejemplo al ingresar la cadena XDR `AAAABgAAAAHMA/50/Q+w3Ni8UXWm/trxFBfAfl6De5kFttaMT0/ACwAAABAAAAABAAAAAgAAAA8AAAAHQ291bnRlcgAAAAASAAAAAAAAAAAg4dbAxsGAGICfBG3iT2cKGYQ6hK4sJWzZ6or1C5v6GAAAAAE=`. Las entradas manuales se generan como de solo lectura (la cadena XDR se generaría si se selecciona y completa correctamente el modo de entrada manual).

![Laboratorio: ejemplo getLedgerEntries](/assets/lab/rpc-methods-item.png)

Puedes agregar múltiples claves de ledger al hacer clic en el botón "Agregar otra clave de ledger".

Puedes cambiar el formato de respuesta en el menú desplegable "Formato XDR": `base64` (XDR) o `json` (más legible para humanos).

La sección "Cuerpo" genera una vista previa del cuerpo que se enviará.

Para enviar la solicitud, haz clic en el botón Enviar en la parte superior (debería estar habilitado si todos los campos son correctos). Una vez que se procesa la solicitud, la respuesta se mostrará en la parte inferior de la página.

## Función de compartir

El laboratorio Stellar te permite compartir métodos RPC con datos rellenados. Simplemente haz clic en el botón Compartir (junto al botón Enviar) para copiar el enlace para compartir. ## Solicitudes guardadas

Puedes guardar métodos en el almacenamiento local del navegador haciendo clic en el botón Guardar (junto al botón Compartir). Una vez que hagas clic en ese botón, aparecerá un modal donde necesitarás ingresar el nombre y hacer clic en guardar. Para ver tus métodos RPC guardados en el Laboratorio, haz clic en el enlace "Saved" en el menú principal, luego haz clic en el enlace del submenú "Requests". Luego haz clic en la pestaña "RPC Methods" en la esquina superior derecha.

:::info

Solo se muestran los elementos para la red seleccionada. Si no ves el elemento que buscas, intenta cambiar de red.

:::

![Laboratorio: Métodos RPC guardados](/assets/lab/rpc-methods-saved.png)

Cada elemento de solicitud guardada tiene lo siguiente:

1. Nombre del método RPC - identifica rápidamente el método que necesitas.
2. Nombre - facilita encontrar el método RPC que buscas. Puedes actualizar el nombre en cualquier momento haciendo clic en el botón de edición y guardando el nuevo nombre.
3. URL RPC - donde se envía la solicitud.
4. Comparte la URL para ver el método con todos los datos en el laboratorio.
5. Ve el elemento en el API Explorer, donde puedes enviarlo.
6. Ver cuerpo.
7. La última fecha y hora guardadas.
8. Elimina el elemento si ya no es necesario.
