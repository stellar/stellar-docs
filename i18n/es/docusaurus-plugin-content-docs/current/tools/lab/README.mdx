# Laboratorio

### [Laboratorio Stellar](https://lab.stellar.org)

Laboratorio Stellar es nuestra nueva herramienta para el desarrollo, la experimentación y las pruebas, así como para explorar las APIs que los desarrolladores utilizan para interactuar con la red Stellar. Ya seas un desarrollador que busca probar transacciones, explorar métodos RPC o puntos finales de Horizon, o profundizar en el ecosistema, Laboratorio Stellar proporciona una interfaz moderna y fácil de usar que hace que el proceso sea fluido e intuitivo.

![Laboratorio: Página de inicio](/assets/lab/lab.png)

### Características de Laboratorio Stellar

- Crea Cuentas Fácilmente: Crea cuentas en Mainnet, Testnet y Futurenet utilizando una interfaz web. Puedes usar Friendbot para financiar esas cuentas directamente en Laboratorio para Testnet y Futurenet.
- Accede a Métodos RPC y Puntos Finales de Horizon: Aprovecha potentes métodos RPC de Stellar y puntos finales de Stellar Horizon en una interfaz web para interactuar con la red Stellar y obtener datos cruciales. Prueba los métodos RPC para obtener estados del ledger como cuentas, trustlines, contrato wasm, y más.
- Simular y Enviar: Laboratorio admite la capacidad de usar proveedores RPC personalizados para que puedas simular transacciones, guardar transacciones y enviar transacciones directamente usando Laboratorio. También puedes enviar transacciones usando Horizon.
- Guarda y Comparte Solicitudes API: Guarda fácilmente tus solicitudes y transacciones para el futuro o compártelas con compañeros de equipo para crear más rápido juntos.
- Soporte XDR ⇔ JSON: Hemos introducido un mapeo canónico [XDR a JSON](https://www.npmjs.com/package/@stellar/stellar-xdr-json-web) que se utiliza en Laboratorio Stellar (también utilizado en el [CLI de Stellar](https://github.com/stellar/stellar-cli)). Puedes verlo en acción en el Laboratorio donde [XDR se puede convertir a JSON](https://lab.stellar.org/xdr/view).
- Compatible con Móviles: Escuchamos que usas Laboratorio en móviles, así que optimizamos Laboratorio para tener un diseño responsivo para móviles.

![Laboratorio XDR a JSON](/assets/lab/xdr-json-lab.png)

Estas son las características que están disponibles ahora. Habrá más características próximas para apoyar contratos inteligentes. Esperamos poder mostrarte el futuro de Laboratorio Stellar.

### ¿Qué pasa con el Antiguo Laboratorio?

En caso de que lo necesites, la versión anterior de Laboratorio Stellar aún es accesible [aquí](https://old-lab.stellar.org). Sin embargo, ya no se mantendrá activamente. Te animamos a explorar el nuevo Laboratorio y si crees que falta algo, contáctanos en Stellar Github.
