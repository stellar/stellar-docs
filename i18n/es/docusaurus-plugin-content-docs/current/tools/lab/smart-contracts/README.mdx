# Contratos Inteligentes

En esta sección, puedes explorar contratos inteligentes de Stellar por ID y ver una lista de contratos inteligentes en la red Stellar seleccionada.

:::tip

Si no conoces ningún ID de contrato inteligente de Stellar, mira la lista y haz clic en cualquiera de los elementos. Esto te llevará a la página "Explorador de Contratos", donde el ID seleccionado aparece prellenado. ¡Carga el contrato y examínalo!

:::
