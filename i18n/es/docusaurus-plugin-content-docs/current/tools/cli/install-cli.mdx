# Instalar el Stellar CLI

### Stellar CLI

Hay varias formas de instalar la versión más reciente del Stellar CLI.

Instalar con Homebrew (macOS, Linux):

```text
brew install stellar-cli
```

Instalar con winget (Windows):

```text
winget install --id Stellar.StellarCLI
```

Instalar con cargo desde la fuente ([github.com/stellar/stellar-cli](https://github.com/stellar/stellar-cli)):

```text
cargo install --locked stellar-cli
```

:::note

También puedes utilizar la herramienta de terceros [SVM (Stellar Version Manager)](https://www.npmjs.com/package/svm-cli), un administrador de versiones para Stellar CLI que te permite instalar y cambiar entre diferentes versiones de stellar-cli.

:::

## Configurar autocompletado

El Stellar CLI admite autocompletado. Para configurar, ejecuta los siguientes comandos:

```text
stellar completion --shell <SHELL>
```

Los valores posibles de SHELL son `bash`, `elvish`, `fish`, `powershell`, `zsh`, etc.

Para habilitar el autocompletado en la shell bash actual, ejecuta:

```text
source <(stellar completion --shell bash)
```

Para habilitar el autocompletado permanentemente, ejecuta:

```text
echo "source <(stellar completion --shell bash)" >> ~/.bashrc
```

## Recetario del Stellar CLI

Para entender cómo aprovechar al máximo el Stellar CLI, consulta el [Recetario del Stellar CLI](https://github.com/stellar/stellar-cli/tree/main/cookbook) para recetas y una colección de recursos que te enseñarán cómo usar el CLI. Ejemplos de recetas incluidas en el recetario del CLI son: enviar pagos, gestionar el ciclo de vida del contrato, extender instancia/almacenamiento/wasm del contrato, y más.
