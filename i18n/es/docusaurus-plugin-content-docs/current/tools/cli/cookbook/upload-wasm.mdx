Para usar el CLI de Stellar y subir un contrato inteligente compilado en el ledger, utiliza el comando `stellar contract upload`:

```bash
stellar contract upload \
    --source S... \
    --network testnet \
    --wasm ../relative/path/to/soroban_contract.wasm
```

:::note

Ten en cuenta que este comando devolverá el hash ID del bytecode Wasm, en vez de una dirección para una instancia de contrato.

:::
