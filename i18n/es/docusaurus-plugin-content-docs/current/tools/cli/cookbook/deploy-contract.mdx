Para desplegar una instancia de un contrato inteligente compilado que ya ha sido cargado en la red Stellar, usa el comando `stellar contract deploy`:

```bash
stellar contract deploy \
    --source S... \
    --network testnet \
    --wasm-hash <hex-encoded-wasm-hash> \
    --alias <alias>
```

:::tip

Opcionalmente, asigna un alias reemplazando `<alias>` con el nombre de alias que desees para el contrato. El alias es una asignación almacenada localmente a la dirección del contrato, y puede usarse en otros comandos de stellar-cli en lugar de la dirección.

:::
