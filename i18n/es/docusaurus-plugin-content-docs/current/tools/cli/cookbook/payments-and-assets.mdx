Para enviar pagos y trabajar con activos usando la CLI de Stellar, sigue estos pasos:

1. Configura tu red preferida. Para esta guía, usaremos `testnet`. Puedes encontrar una lista de redes disponibles [aquí](https://developers.stellar.org/docs/networks)

```bash
stellar network use testnet
```

2. Financia las cuentas:

```bash
stellar keys generate alice
```

```bash
stellar keys fund alice
```

```bash
stellar keys generate bob
```

```bash
stellar keys fund bob
```

3. Obtén el ID del contrato del activo stellar:

```bash
stellar contract id asset --asset native
```

4. Obtén la clave pública de Bob:

```bash
stellar keys address bob
```

5. Envía 100 XLM de Alice a Bob:

```bash
stellar keys use alice
```

```bash
stellar contract invoke --id <ASSET_CONTRACT_ID> -- transfer --to bob --from alice --amount 100
```

6. Consulta el saldo de la cuenta:

```bash
stellar contract invoke --id <ASSET_CONTRACT_ID> -- balance --id bob
```

Para más información sobre las funciones disponibles en el contrato de activo stellar, consulta el [código de la interfaz del token](https://developers.stellar.org/docs/tokens/token-interface#code).
