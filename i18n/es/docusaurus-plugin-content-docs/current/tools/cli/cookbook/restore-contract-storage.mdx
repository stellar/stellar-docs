Si la entrada de almacenamiento persistente de un contrato ha sido archivada, puedes restaurarla usando la CLI de Stellar. Para una entrada de almacenamiento que use un `Symbol` simple como clave de almacenamiento, puedes ejecutar un comando así:

```bash
stellar contract restore \
    --source S... \
    --network testnet \
    --id C... \
    --key COUNTER \
    --durability persistent
```

Si tu entrada de almacenamiento usa una clave de almacenamiento más avanzada, como `Balance(Address)` en un contrato de token, necesitarás proporcionar la clave en forma XDR codificada en base64:

```bash
stellar contract restore \
    --source S... \
    --network testnet \
    --id C... \
    --key-xdr AAAABgAAAAHXkotywnA8z+r365/0701QSlWouXn8m0UOoshCtNHOYQAAAA4AAAAHQmFsYW5jZQAAAAAB \
    --durability persistent
```

:::info

Asegúrate de revisar nuestra [guía para crear claves de libro mayor XDR](../../../build/guides/rpc/generate-ledger-keys-python.mdx) para obtener ayuda generando las claves.

:::
