# Stellar CLI

La interfaz de línea de comandos para contratos inteligentes Soroban. Te permite crear, implementar e interactuar con contratos inteligentes; configurar identidades; generar pares de claves; gestionar redes; y más.

Instala Stellar CLI como se explica en [Configuración](../../build/smart-contracts/getting-started/setup.mdx#install-the-stellar-cli). Para ejemplos sobre cómo usar Stellar CLI, consulta [Guía de Stellar CLI](../../build/guides/cli/README.mdx).
