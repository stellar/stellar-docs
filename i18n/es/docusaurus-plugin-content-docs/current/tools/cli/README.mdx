# Stellar CLI

La interfaz de línea de comandos para contratos inteligentes Soroban. Te permite crear, implementar e interactuar con contratos inteligentes; configurar identidades; generar pares de claves; gestionar redes; y más.

Instala Stellar CLI como se explica en [Configuración](../../build/smart-contracts/getting-started/setup.mdx#install-the-stellar-cli). Para ejemplos sobre cómo usar Stellar CLI, consulta [Guía de Stellar CLI](../../build/guides/cli/README.mdx).

La documentación completa de referencia generada automáticamente está disponible [aquí](stellar-cli.mdx).

:::note

[Sora] es una GUI multiplataforma mantenida por un tercero que está diseñada para proporcionar una interfaz gráfica a la Stellar CLI. Ofrece una interfaz fácil de usar para gestionar proyectos, identidades, redes, métodos de contrato, eventos, registros y mucho más en una interfaz gráfica en lugar de una interfaz de línea de comandos.

:::

[Sora]: https://github.com/tolgayayci/sora
