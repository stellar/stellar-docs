MoneyGram Ramps es un producto de MoneyGram que permite a los usuarios de aplicaciones de terceros, como billeteras criptográficas y exchanges, depositar (cash-in) y retirar (cash-out) USDC en Stellar.

[Explora la documentación de MoneyGram Ramps](https://developer.moneygram.com/moneygram-developer/docs/integrate-moneygram-ramps) para conocer los requisitos técnicos para integrar MoneyGram Ramps en una billetera existente o para crear una nueva aplicación de billetera.
