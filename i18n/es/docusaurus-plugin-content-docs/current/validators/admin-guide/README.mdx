Stellar Core es el uso de nodos del programa para comunicarse con otros nodos para crear y mantener la red de par a par estelar. Es una implementación del Protocolo de Consenso Estelar configurado para construir una cadena de ledgers garantizados de estar de acuerdo en todos los nodos participantes en todo momento.

Estas páginas describen varios aspectos de instalar, configurar y mantener un nodo `stellar-core`.
