Las cuentas son la estructura central de datos en Stellar- tienen balances, transacciones de signos y activos de emisión. Las cuentas sólo pueden existir con un par de claves válido y el saldo mínimo requerido de XLM.

Para aprender sobre los requisitos de balance mínimo, [ver nuestra sección en Lumens](../lumens.mdx#minimum-balance).

## Basar reservas y subentradas

Las cuentas almacenan datos en subentradas, y cada subentrada incrementa el saldo mínimo requerido de la cuenta.

### Reservas base

Una reserva base es una unidad de medida utilizada para calcular el saldo mínimo de una cuenta. Una reserva base es actualmente 0.5 XLM.

### Subentries

Los datos de las cuentas se almacenan en subentradas, cada una de las cuales aumenta el saldo mínimo de una cuenta en una reserva base (0.5 XLM). Una cuenta no puede tener más de 1.000 subentradas. Las subentradas posibles son:

- Líneas de confianza (incluye activos tradicionales y acciones de cobertura)
- Ofertas
- Firmadores adicionales
- Entradas de datos (incluye datos hechos con la operación `manageData`, no entradas inteligentes del libro de contadores)

## Líneas de confianza

Los fideicomisos son una opción explícita para que una cuenta posea y opere con un activo en particular. Para mantener un activo específico, una cuenta debe establecer una línea de confianza con la cuenta emisora usando la operación change_trust . Los Trustlines rastrean el saldo de un activo y también pueden limitar la cantidad de un activo que una cuenta puede tener.

Debe establecerse una línea de confianza para que una cuenta reciba cualquier activo excepto lumens (XLM). Puedes crear un saldo cobrable para enviar activos a una cuenta sin una línea de confianza, pero el destinatario tiene que crear una línea de confianza para reclamar ese equilibrio. Más información aquí: [Entrada de la Enciclopedia de Balances Claimables](../../encyclopedia/transactions-specialized/claimable-balances.mdx)

Una línea de confianza también rastrea pasivos. La compra de pasivos equivale a la cantidad total del activo ofrecido para comprar agregados sobre todas las ofertas propiedad de una cuenta, y los pasivos vendidos equivalen a la cantidad total del activo ofrecido para vender agregados sobre todas las ofertas de una cuenta. Una línea de confianza siempre debe tener un equilibrio lo suficientemente grande como para satisfacer sus obligaciones de venta y un saldo lo suficientemente por debajo de su límite para acomodar sus obligaciones de compra.
