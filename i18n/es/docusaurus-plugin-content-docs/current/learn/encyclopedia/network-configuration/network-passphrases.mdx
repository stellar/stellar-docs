La Mainnet, Testnet y Futurenet de Stellar tienen cada una su propia frase única. Estas se utilizan al validar firmas en una transacción dada. Si firmas una transacción para una red pero la envías a otra, no se considerará válida. Por convención, el formato de una frase es ‘`[Nombre de la Red] ; [Mes de Creación] [Año de Creación]`’.

Las frases actuales para la Mainnet, Testnet y Futurenet de Stellar son:

- Mainnet: '`Public Global Stellar Network ; Septiembre 2015`'
- Testnet: '`Test SDF Network ; Septiembre 2015`'
- Futurenet: '`Test SDF Future Network ; Octubre 2022`'

Las frases sirven para dos propósitos principales: (1) utilizadas como la semilla para la cuenta raíz (clave maestra de la red) en el génesis y (2) utilizadas para crear hashes de transacciones, que son finalmente lo que se firma con la clave secreta de cada firmante en un sobre de transacción; esto te permite verificar que una transacción fue destinada para una red específica por sus firmantes.

Muchos SDK tienen las frases codificadas para las redes de Stellar. Si estás ejecutando una red privada, tendrás que pasar manualmente una frase que se usará cada vez que se generen hashes de transacciones. Todos los SDK oficiales de Stellar permiten usar una red con una frase personalizada.
