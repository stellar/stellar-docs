---
title: Contraseña de red
sidebar_position: 10
---

La red principal de Stellar, Testnet y Futurenet tienen su propia contraseña única. Estos se utilizan para validar firmas en una transacción determinada. Si firma una transacción por una red pero la envía a otra, no se considerará válida. Por convención, el formato de una contraseña es ''[Nombre de red] ; [Mes de la creación] [Año de la Creación]'.

Las contraseñas actuales de Stellar Mainnet, Testnet y Futurenet son:

- Mainnet: 'Red pública mundial estelar ; septiembre de 2015'
- Testnet: '`Probar red SDF ; septiembre 2015`'
- Futurenet: '`Test SDF Future Network ; octubre 2022`'

Las contraseñas tienen dos propósitos principales: (1) usados como semilla para la cuenta raíz (clave de red maestra) en génesis y (2) usados para construir hash de las transacciones, que son en última instancia lo que está firmado por la clave secreta de cada firmante en un sobre de transacción; esto le permite verificar que una transacción fue destinada a una red específica por sus signatarios.

Muchos SDK tienen las frases de contraseña codificadas para las redes de Stellar. Si está corriendo una red privada, tendrá que pasar manualmente una contraseña para ser usada cuando se generen errores de transacción. Todos los SDKs oficiales de Stellar le permiten usar una red con una contraseña personalizada.
