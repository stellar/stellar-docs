title: Requisitos previos
sidebar_position: 5

Para crear utilizando el SDK de ingest, necesitas:

- **Compilador y tiempo de ejecución de Go (>=1,23)** – para crear y ejecutar los paquetes
- **Git** – para descargar el código fuente _(no necesario si se instala a través del gestor de paquetes)_
- **IDE** (por ejemplo, VS Code, Goland) _(opcional)_
