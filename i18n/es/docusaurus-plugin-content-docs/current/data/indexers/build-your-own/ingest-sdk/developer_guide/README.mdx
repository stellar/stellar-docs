El [SDK de ingestión](https://github.com/stellar/go-stellar-sdk/tree/main/ingest) de Stellar es un conjunto de paquetes para recuperar y procesar metadatos del ledger de la red Stellar. Los desarrolladores pueden utilizar este SDK para crear motores de ingestión personalizados.

Esta guía explica cómo configurar, configurar y utilizar el SDK de ingestión de manera efectiva.
