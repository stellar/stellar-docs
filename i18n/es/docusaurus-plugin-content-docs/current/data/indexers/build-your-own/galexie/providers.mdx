Varios proveedores de infraestructura han puesto a disposición el servicio y el lago de datos de Galexie, y ofrecen planes que van desde acceso gratuito hasta acceso de pago. Estos proveedores pueden usarse para desarrollo, pruebas y producción.

Estos proveedores permiten acceso a las redes Futurenet, Testnet y Mainnet.

| Proveedor                                                                       | Futurenet | Testnet | Mainnet | URI                                                |
| ------------------------------------------------------------------------------- | --------- | ------- | ------- | -------------------------------------------------- |
| [AWS Public Blockchain\*](https://registry.opendata.aws/aws-public-blockchain/) | ❌         | ❌       | ✅       | `s3://aws-public-blockchain/v1.1/stellar/ledgers/` |
| [Lightsail Network - Quasar](https://quasar.lightsail.network/)                 | ❌         | ❌       | ✅       | `https://galexie.lightsail.network/v1/`            |

\*AWS Public Blockchain poblado por Goldsky

## Ejecutar tu propio Galexie

Si te interesa ejecutar tu propio servicio y lago de datos Galexie, por favor consulta la [Guía del administrador](./admin_guide/README.mdx).
