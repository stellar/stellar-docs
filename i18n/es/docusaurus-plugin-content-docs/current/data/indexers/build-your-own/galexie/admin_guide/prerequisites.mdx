# Requisitos previos

### 1. Cuenta de Google Cloud Platform (GCP)

Galexie exporta metadatos del ledger de Stellar a Google Cloud Storage (GCS), por lo que necesitas una cuenta de GCP con:

- Permisos para crear un nuevo bucket de GCS, o
- Acceso a un bucket existente con permisos de lectura/escritura.

### 2. Docker (Recomendado)

> **_NOTA:_** Aunque es posible instalar Galexie de manera nativa (sin Docker), esto requiere una gestión manual de dependencias y se recomienda solo para usuarios avanzados

Galexie está disponible como una imagen de Docker, lo que simplifica la instalación y configuración. Asegúrate de tener Docker Engine instalado en tu sistema ([guía de instalación de Docker](https://docs.docker.com/engine/install/)).

## Requisitos de Hardware

Los requisitos mínimos de hardware para ejecutar Galexie son:\
**RAM**: 8 GB\
**CPU**: 2 vCPUs\
**Disco**: 100 GB con al menos 5K IOPS

### Exportación Completa de Historial

Exportar todo el historial (a partir de noviembre de 2024):

- Toma un estimado de 150 días usando una única instancia de Galexie
- Crea un lago de datos de aproximadamente 3 TB
- Para acelerar la exportación, se pueden ejecutar múltiples instancias de Galexie en paralelo, cada una trabajando en un rango diferente
- Por ejemplo, crear todo el historial para el lago de datos de Hubble usando alrededor de 40 instancias paralelas (cada una exportando aproximadamente <b>1,25</b> millones de ledgers) en _e2-standard-2_ (Google Cloud Platform) lleva alrededor de 5 días
