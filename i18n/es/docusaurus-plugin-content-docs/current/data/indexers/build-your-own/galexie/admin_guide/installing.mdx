# Instalando

## Kubernetes

Para instalar Galexie en un clúster de Kubernetes, sigue estos pasos:

- Ejecuta `helm repo add stellar https://helm.stellar.org/charts`
- Ejecuta `helm install <RELEASE_NAME> stellar/galexie --set datastore.params.path=<FULL_PATH_TO_GALEXIE_OBJECTS>`

La configuración adicional se puede establecer con las opciones `--set` o utilizando un [archivo de valores helm](https://helm.sh/docs/chart_template_guide/values_files/) con el chart.

## Instalar el contenedor en un host

Para descargar la imagen del contenedor Galexie desde el [registro Stellar Docker Hub](https://hub.docker.com/r/stellar/stellar-galexie) usa el siguiente comando docker o un comando similar compatible con OCI:

```shell
docker pull stellar/stellar-galexie
```
