## Indexadores de Datos

Para potenciar tus aplicaciones, estará disponible una gama de servicios de datos desde indexadores de datos con interfaces de consulta. Estos indexadores te permitirán almacenar y recuperar fácilmente datos de ledger decodificados, emisiones de eventos de contratos, y más. Puedes enfocar más energía en hacer que tu proyecto tenga éxito y perder menos tiempo descubriendo cómo obtener precisamente los datos importantes de la red.

### [Goldsky](https://goldsky.com/)

Una herramienta de canalización de datos que te permite combinar datos on-chain y off-chain como desees, transmitidos directamente a tu propia base de datos personalizada.

### [Mercury](https://docs.mercurydata.app)

Un servicio de indexación de red que proporciona un conjunto de herramientas para ayudarte a comenzar a trabajar con datos de la red Stellar.

### [SubQuery](https://subquery.network/)

Un indexador de datos open-source para Stellar que permite la creación de APIs personalizadas para dapps y contratos inteligentes con datos limpios e indexados. Admite la indexación de ledgers, transacciones, operaciones y efectos de Stellar, así como transacciones y eventos de Soroban en una base de datos Postgres.

### [The Graph](https://thegraph.com/)

Un protocolo de indexación descentralizado para consultar datos de blockchain. The Graph permite a los desarrolladores crear y publicar APIs abiertas, llamadas subgráficas, que hacen que los datos on-chain sean fácilmente accesibles y buscables para sus aplicaciones.
