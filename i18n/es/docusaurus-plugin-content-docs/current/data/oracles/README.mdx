# Oráculos

Los oráculos son servicios que conectan sistemas de blockchain con fuentes de datos externas y fuera de la cadena, permitiendo que los contratos inteligentes interactúen con información del mundo real. Actúan como intermediarios, obteniendo y verificando datos como precios de mercado, condiciones climáticas o resultados de eventos, y luego entregándolos a la blockchain de manera segura y confiable. Esto permite que las aplicaciones descentralizadas (dApps) se ejecuten en base a eventos del mundo real, ampliando su funcionalidad más allá de los datos en la cadena.
