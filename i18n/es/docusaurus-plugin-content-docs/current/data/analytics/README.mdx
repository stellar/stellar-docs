Aprende sobre análisis y proveedores de análisis en la red Stellar

## [Hubble](./hubble/README.mdx)

Hubble es un almacén de datos BigQuery de solo lectura y de código abierto mantenido por SDF que proporciona un registro histórico completo de la red Stellar, ideal para el análisis a gran escala de transacciones, tarifas y tendencias de la red. No es adecuado para necesidades de datos en tiempo real o búsquedas simples, y su documentación se centra en análisis exploratorios, métodos de conexión y optimización de consultas. Al ofrecer un conjunto de datos estructurado y listo para consultas, Hubble permite a los usuarios explorar datos de la red Stellar sin gestionar tuberías de datos en bruto.

Dentro de Hubble, el producto stellar-etl potencia la extracción y transformación de datos de la red Stellar en BigQuery. Este pipeline de código abierto permite a los desarrolladores extender o personalizar los flujos de trabajo de ingestión de datos según sus necesidades analíticas. Stellar-etl proporciona flexibilidad en cómo se procesan los datos, permitiendo a los usuarios extraer entradas de ledger específicas, operaciones o registros de eventos según sus requisitos.

## [Proveedores de análisis de datos](./analytics-providers/analytics-providers.mdx)

Los proveedores de análisis de datos proporcionan un registro histórico completo de los datos de la red Stellar de Pubnet para análisis y percepciones. Estas plataformas permiten a los usuarios explorar tendencias de la red, pero no están diseñadas para el acceso a datos en tiempo real o ejecución de transacciones. La diferencia clave es que estas plataformas del ecosistema operan de manera independiente de SDF y proporcionan métricas y tableros sobre la red Stellar.
