## Metadatos de la tabla

| Propiedad             | Configuración                                                                                                  |
| --------------------- | -------------------------------------------------------------------------------------------------------------- |
| Clave(s) natural(es)  | offer_id, closed_at                                                                                            |
| Campo(s) de partición | batch_run_date (partición por MES)                                                                             |
