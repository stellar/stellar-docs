## Metadatos de la Tabla

| Propiedad             | Configuración                                                                                                         |
| --------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Clave(s) Natural(es)  | asset_code, asset_issuer, asset_type                                                                                  |
| Campo(s) de Partición | batch_run_date (partición de MES)                                                                                     |
| Campo(s) Agrupados    | asset_code, asset_issuer, asset_type                                                                                  |
| Documentación         | [dbt_docs](http://www.stellar-dbt-docs.com/#!/source/source.stellar_dbt_public.crypto_stellar.history_assets_staging) |

## Detalles de la Columna

| Nombre          | Descripción                                                                                                                                                                                                                                | Tipo de Dato | Valores de Dominio                                                          | ¿Requerido? | Notas                                                                                                                                                                 |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------ | --------------------------------------------------------------------------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id              | Identificador único para la combinación de código de activo, tipo y emisor. Esto no es una clave primaria en la tabla                                                                                                                      | float        |                                                                             | Sí          | Obsoleto                                                                                                                                                              |
| asset_id        | Identificador único para asset_code, asset_issuer                                                                                                                                                                                          | integer      |                                                                             | No          |                                                                                                                                                                       |
| asset_type      | El identificador para el tipo de código de activo, puede ser un alfanumérico de 4 caracteres, 12 caracteres o el activo nativo de la red, XLM.                                                                                             | string       | <ul><li>credit_alphanum4</li><li>credit_alphanum12</li><li>native</li></ul> | Sí          | XLM es el activo nativo de la red. XLM no tiene código de activo ni representación de emisor y se mostrará con un tipo de activo 'native'                             |
| asset_code      | La representación del código del activo en la red de 4 o 12 caracteres                                                                                                                                                                     | string       |                                                                             | No          | Los códigos de activos no garantizan unicidad. La combinación de código de activo, emisor y tipo representa un activo distinto                                        |
| asset_issuer    | La dirección de la cuenta del emisor original del activo que creó el activo                                                                                                                                                                | string       |                                                                             | No          |                                                                                                                                                                       |
| batch_id        | Representación en cadena del id de ejecución para un DAG dado en Airflow. Toma la forma de `scheduled__<batch_end_date>-<dag_alias>`. Los batch ids son únicos para el batch y ayudan con el monitoreo y las capacidades de reejecución    | string       |                                                                             | Sí          |                                                                                                                                                                       |
| batch_run_date  | La fecha de inicio para el intervalo del batch. Tomada junto con la fecha en el batch_id, esta fecha representa el intervalo de ledgers procesados. La fecha de ejecución del batch puede verse como un proxy de closed_at para un ledger. | datetime     |                                                                             | Sí          | La tabla está particionada por batch_run_date. Se recomienda siempre incluir batch_run_date en el filtro si es posible para ayudar a reducir el costo de la consulta. |
| batch_insert_ts | El timestamp en UTC cuando un lote de registros fue insertado en la base de datos. Este campo puede ayudar a identificar si un batch se ejecutó en tiempo real o como parte de un relleno de datos (backfill)                              | timestamp    |                                                                             | Sí          |                                                                                                                                                                       |
