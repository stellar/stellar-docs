## Metadatos de la tabla

| Propiedad             | Configuración                                                                                         |
| --------------------- | ----------------------------------------------------------------------------------------------------- |
| Clave(s) natural(es)  | offer_id, closed_at                                                                                   |
| Campo(s) de partición | batch_run_date (partición MES)                                                                        |
| Campo(s) agrupado(s)  | selling_asset_id, buying_asset_id, last_modified_ledger                                               |
| Documentación         | [dbt docs](http://www.stellar-dbt-docs.com/#!/source/source.stellar_dbt_public.crypto_stellar.offers) |

| price                | Cuántas unidades del activo a comprar se necesitan para obtener 1 unidad del activo a vender. Este número es la forma decimal de pricen / priced                                                                                                         | flotante        |                                                                                                                                                                                                   | Sí          | Si una oferta quiere vender 10 XLM a cambio de 1 USD, el precio será 0.1                                                                                               |
| flags                | Indica la activación/desactivación de ciertos privilegios del emisor del activo                                                                                                                                                                          | entero          | <ul><li>0 - Ninguno, Valor predeterminado</li><li>1 - Pasivo (una oferta con esta flag no actuará ni aceptará una oferta inversa con precio igual)</li></ul>                                      | Sí          |                                                                                                                                                                        |
