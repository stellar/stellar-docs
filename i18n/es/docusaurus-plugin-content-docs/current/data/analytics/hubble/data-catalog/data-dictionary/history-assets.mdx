## Metadatos de la Tabla

| Propiedad             | Configuración                                                                                                         |
| --------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Clave(s) natural(es)  | asset_code, asset_issuer, asset_type                                                                                  |
| Campo(s) de Partición | batch_run_date (partición MESES)                                                                                      |
| Campo(s) agrupado(s)  | asset_code, asset_issuer, asset_type                                                                                  |
| Documentación         | [dbt_docs](http://www.stellar-dbt-docs.com/#!/source/source.stellar_dbt_public.crypto_stellar.history_assets_staging) |

## Detalles de las Columnas

| Nombre          | Descripción                                                                                                                                                                                                                               | Tipo de Dato | Valores del Dominio                                                         | ¿Requerido? | Notas                                                                                                                                                                    |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------ | --------------------------------------------------------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| id              | Identificador único para la combinación de código, tipo e emisor de activo. Esto no es una clave primaria en la tabla                                                                                                                     | float        |                                                                             | Sí          | Obsoleto                                                                                                                                                                 |
| asset_id        | Identificador único para asset_code, asset_issuer                                                                                                                                                                                         | integer      |                                                                             | No          |                                                                                                                                                                          |
| asset_type      | El identificador para el tipo de código de activo puede ser alfanumérico con 4 caracteres, 12 caracteres o el activo nativo de la red, XLM.                                                                                               | string       | <ul><li>credit_alphanum4</li><li>credit_alphanum12</li><li>native</li></ul> | Sí          | XLM es el activo nativo de la red. XLM no tiene representación de código o emisor de activo y se mostrará con un tipo de activo 'nativo'                                 |
| asset_code      | La representación en código de 4 o 12 caracteres del activo en la red                                                                                                                                                                     | string       |                                                                             | No          | Los códigos de activos no garantizan la unicidad. La combinación de código de activo, emisor y tipo representa un activo distinto                                        |
| asset_issuer    | La dirección de cuenta del emisor de activo original que creó el activo                                                                                                                                                                   | string       |                                                                             | No          |                                                                                                                                                                          |
| batch_id        | Representación en cadena del id de ejecución para un DAG dado en Airflow. Toma la forma de "scheduled\_\_\<batch_end_date>-\<dag_alias>". Los ids de lotes son únicos para el lote y ayudan con el monitoreo y capacidades de reejecución | string       |                                                                             | Sí          |                                                                                                                                                                          |
| batch_run_date  | La fecha de inicio para el intervalo de lotes. Tomada con la fecha en el batch_id, la fecha representa el intervalo de ledgers procesados. La fecha de ejecución del lote puede verse como un proxy de closed_at para un ledger.          | datetime     |                                                                             | Sí          | La tabla está particionada por batch_run_date. Se recomienda incluir siempre el batch_run_date en el filtro si es posible para ayudar a reducir el costo de la consulta. |
| batch_insert_ts | La marca de tiempo en UTC cuando se insertó un lote de registros en la base de datos. Este campo puede ayudar a identificar si un lote se ejecutó en tiempo real o como parte de un backfill                                              | timestamp    |                                                                             | Sí          |                                                                                                                                                                          |
