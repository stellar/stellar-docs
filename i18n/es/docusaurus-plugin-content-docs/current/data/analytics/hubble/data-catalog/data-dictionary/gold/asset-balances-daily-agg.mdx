## Metadatos de la Tabla

| Propiedad             | Configuración                                                                                           |
| --------------------- | ------------------------------------------------------------------------------------------------------- |
| Clave(s) Natural(es)  | día, tipo_de_activo, código_de_activo, emisor_de_activo                                                 |
| Campo(s) de Partición | día (partición DIA)                                                                                     |
| Campo(s) Clúster      | No Aplica                                                                                               |
| Documentación         | [dbt docs](http://www.stellar-dbt-docs.com/#!/model/model.stellar_dbt_public.asset_balances__daily_agg) |

## Detalles de Columnas

| Nombre                                     | Descripción                                                                                                                             | Tipo de Datos | Valores del Dominio | ¿Requerido? | Notas                                                                                                                                     |
| ------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------- | ------------- | ------------------- | ----------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| día                                        | Fecha hasta la cual se agregan todas las métricas.                                                                                      | FECHA         |                     | Sí          |                                                                                                                                           |
| tipo_de_activo                             | El identificador del tipo de código de activo, puede ser alfanumérico de 4 caracteres, 12 caracteres o el activo nativo de la red, XLM. | CADENA        |                     | Sí          | XLM es el activo nativo de la red. XLM no tiene código de activo ni representación de emisor y se mostrará con un tipo de activo 'nativo' |
| código_de_activo                           | La representación en código de 4 o 12 caracteres del activo en la red.                                                                  | CADENA        |                     | Sí          | Los códigos de activo no garantizan unicidad. La combinación de código de activo, emisor y tipo representa un activo único.               |
| emisor_de_activo                           | La dirección de cuenta del emisor original del activo que creó el activo.                                                               | CADENA        |                     | Sí          |                                                                                                                                           |
| liquidity_pool_balance                     | La suma de los saldos en todos los Fondos de Liquidez para un activo dado.                                                              | FLOTANTE      |                     | Sí          |                                                                                                                                           |
| offer_balance                              | La suma de saldos en todas las obligaciones de venta (solo SDEX) para un activo dado.                                                   | FLOTANTE      |                     | Sí          |                                                                                                                                           |
| trustline_balance                          | La suma del saldo de la trustline para un activo dado.                                                                                  | FLOTANTE      |                     | Sí          |                                                                                                                                           |
| total_accounts_with_liquidity_pool_balance | El recuento de titulares con saldo positivo en Fondos de Liquidez para un activo dado.                                                  | ENTERO        |                     | Sí          |                                                                                                                                           |
| total_accounts_with_offer_balance          | El recuento de titulares con saldo positivo de oferta para un activo dado.                                                              | ENTERO        |                     | Sí          |                                                                                                                                           |
| total_accounts_with_trustline_balance      | El recuento de titulares con saldo positivo en trustline para un activo dado.                                                           | ENTERO        |                     | Sí          |                                                                                                                                           |
