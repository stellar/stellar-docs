## Metadatos de la Tabla

| Propiedad             | Configuración                                                                                               |
| --------------------- | ----------------------------------------------------------------------------------------------------------- |
| Clave(s) Natural(es)  | key_hash, closed_at                                                                                         |
| Campo(s) de Partición | closed_at (partición por MES)                                                                               |
| Campo(s) Agrupado(s)  | last_modified_ledger, key_hash                                                                              |
| Documentación         | [documentos de dbt](http://www.stellar-dbt-docs.com/#!/source/source.stellar_dbt_public.crypto_stellar.ttl) |

## Detalles de la Columna

| Nombre                   | Descripción                                                                                                                                  | Tipo de Datos   | Valores de Dominio | ¿Requerido? | Notas |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------- | --------------- | ------------------ | ----------- | ----- |
| key_hash                 | Hash de clave del ledger usado para identificar datos de contratos caducados o entradas de código de contrato del ledger                     | cadena          |                    | Sí          |       |
| live_until_ledger_seq    | Número de secuencia del ledger hasta el cual vivirá el contrato o wasm                                                                       | entero          |                    | Sí          |       |
| last_modified_ledger     | El número de secuencia del ledger cuando la entrada del ledger (este firmante único para la cuenta) fue modificada                           | entero          |                    | Sí          |       |
| cambio_de_entrada_ledger | Código que describe el tipo de cambio de entrada del ledger que se aplicó a la entrada del ledger                                            | entero          |                    | Sí          |       |
| eliminado                | Indica si la entrada del ledger (id de saldo) ha sido eliminada o no. Una vez que una entrada es eliminada, no se puede recuperar            | booleano        |                    | Sí          |       |
| batch_id                 | Representación en cadena del id de ejecución para un DAG dado en Airflow                                                                     | cadena          |                    | Sí          |       |
| batch_run_date           | La fecha de inicio del intervalo de lote. Cuando se toma con la fecha en el batch_id, la fecha representa el intervalo de ledgers procesados | fecha y hora    |                    | Sí          |       |
| batch_insert_ts          | La marca de tiempo en UTC cuando un lote de registros se insertó en la base de datos                                                         | marca de tiempo |                    | Sí          |       |
| closed_at                | La marca de tiempo UNIX de la edad del número de secuencia                                                                                   | marca de tiempo |                    | Sí          |       |
| ledger_sequence          | El número de ledger no firmado de 32 bits de la edad del número de secuencia                                                                 | entero          |                    | Sí          |       |
