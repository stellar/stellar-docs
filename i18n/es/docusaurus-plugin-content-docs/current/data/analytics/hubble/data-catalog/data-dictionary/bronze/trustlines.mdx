## Metadatos de la tabla

| Propiedad             | Configuración                                                                                              |
| --------------------- | ---------------------------------------------------------------------------------------------------------- |
| Clave natural (s)     | account_id, asset_type, asset_issuer, asset_code, liquidity_pool_id, closed_at                             |
| Campo(s) de partición | batch_run_date (partición MES)                                                                             |
| Campo(s) agrupados    | account_id, asset_id, liquidity_pool_id, last_modified_ledger                                              |
| Documentación         | [dbt docs](http://www.stellar-dbt-docs.com/#!/source/source.stellar_dbt_public.crypto_stellar.trust_lines) |

## Detalles de columnas

| Nombre               | Descripción                                                                                                                                                                                                                                                 | Tipo de dato | Valores de dominio                                                                                                                                                       | ¿Requerido? | Notas                                                                                                                                                                                    |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ledger_key           | La clave única del ledger cuando el estado de la línea de confianza cambió por última vez                                                                                                                                                                   | string       |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| account_id           | La dirección de la cuenta                                                                                                                                                                                                                                   | string       |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| asset_type           | El identificador para el tipo de código de activo, puede ser alfanumérico de 4 caracteres, 12 caracteres o el activo nativo de la red, XLM.                                                                                                                 | integer      | <ul><li>credit_alphanum4</li><li>credit_alphanum12</li><li>native</li></ul>                                                                                              | Sí          |                                                                                                                                                                                          |
| asset_issuer         | La dirección de la cuenta del emisor original del activo que creó el activo que posee esta cuenta                                                                                                                                                           | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                          |
| asset_code           | La representación del código de 4 o 12 caracteres del activo que posee esta cuenta                                                                                                                                                                          | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                          |
| liquidity_pool_id    | Si el activo es parte de una participación en un pool de liquidez, el id único del pool del cual se origina el saldo del activo                                                                                                                             | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                          |
| balance              | El número de unidades de un activo que posee esta cuenta                                                                                                                                                                                                    | float        |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| trust_line_limit     | La cantidad máxima de este activo que esta cuenta está dispuesta a aceptar. El límite se especifica cuando se abre una línea de confianza                                                                                                                   | integer      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| buying_liabilities   | La suma de todas las ofertas de compra propiedad de esta cuenta para activos no nativos                                                                                                                                                                     | float        |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| selling_liabilities  | La suma de todas las ofertas de venta propiedad de esta cuenta para activos no nativos                                                                                                                                                                      | float        |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| flags                | Denota la habilitación y deshabilitación de ciertas privilegios del emisor del activo                                                                                                                                                                       | integer      | <ul><li>0 - Ninguno, Predeterminado</li><li>1 - Autorizado</li><li>2 - Autorizado para Mantener Pasivos</li><li>4 - Recuperación Activada</li></ul>                      | Sí          | Las flags se establecen en las cuentas de los emisores para un activo. Cuando las cuentas de los usuarios confían en un activo, las flags aplicadas al activo se originan en esta cuenta |
| last_modified_ledger | El número de secuencia del ledger cuando la entrada del ledger (este firmante único para la cuenta) fue modificada                                                                                                                                          | integer      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| ledger_entry_change  | Código que describe el tipo de cambio aplicado a la entrada del ledger                                                                                                                                                                                      | integer      | <ul><li>0 - Entrada de ledger creada</li><li>1 - Entrada de ledger actualizada</li><li>2 - Entrada de ledger eliminada</li><li>3 - Estado de entrada de ledger</li></ul> | Sí          | Los tipos de cambio de entrada válidos son 0, 1 y 2 para entradas de ledger del tipo `trust_lines`.                                                                                      |
| deleted              | Indica si la entrada del ledger (línea de confianza) ha sido eliminada o no. Una vez que una entrada es eliminada, no puede recuperarse                                                                                                                     | boolean      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| closed_at            | Marca de tiempo en UTC cuando este ledger se cerró y confirmó en la red. Se espera que los ledgers se cierren aproximadamente cada 5 segundos                                                                                                               | timestamp    |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| ledger_sequence      | El número de secuencia de este ledger. Representa el orden del ledger dentro de la blockchain de Stellar. Cada ledger tiene un número de secuencia único que incrementa con cada nuevo ledger, asegurando que los ledgers se procesen en el orden correcto. | integer      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| batch_id             | Representación en cadena del id de ejecución para un DAG dado en Airflow                                                                                                                                                                                    | string       |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| batch_run_date       | La fecha de inicio del intervalo de lote. Tomada junto con la fecha en batch_id, la fecha representa el intervalo de ledgers procesados                                                                                                                     | datetime     |                                                                                                                                                                          | Sí          | La tabla está particionada por batch_run_date. Se recomienda siempre incluir batch_run_date en el filtro si es posible para ayudar a reducir el costo de la consulta.                    |
| batch_insert_ts      | Marca de tiempo en UTC cuando un lote de registros fue insertado en la base de datos                                                                                                                                                                        | timestamp    |                                                                                                                                                                          | Sí          |                                                                                                                                                                                          |
| sponsor              | La dirección de la cuenta que patrocina las reservas base para la línea de confianza                                                                                                                                                                        | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                          |
| asset_id             | Identificador único para asset_code, asset_issuer                                                                                                                                                                                                           | integer      |                                                                                                                                                                          | No          |                                                                                                                                                                                          |
