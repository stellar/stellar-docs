## Metadatos de la tabla

| Propiedad             | Configuración                                                                                      |
| --------------------- | -------------------------------------------------------------------------------------------------- |
| Clave(s) natural(es)  | key_hash, closed_at                                                                                |
| Campo(s) de partición | closed_at (partición MES)                                                                          |
| Campo(s) agrupado(s)  | last_modified_ledger, key_hash                                                                     |
| Documentación         | [dbt docs](http://www.stellar-dbt-docs.com/#!/source/source.stellar_dbt_public.crypto_stellar.ttl) |

## Detalles de columnas

| Nombre                | Descripción                                                                                                                            | Tipo de datos   | Valores del dominio | ¿Requerido? | Notas |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------- | --------------- | ------------------- | ----------- | ----- |
| key_hash              | Hash de clave del ledger usado para identificar datos de contrato caducando o entradas de ledger de código de contrato                 | cadena          |                     | Sí          |       |
| live_until_ledger_seq | Secuencia del ledger hasta la cual el contrato o wasm permanecerá activo                                                               | entero          |                     | Sí          |       |
| last_modified_ledger  | Número de secuencia del ledger cuando se modificó la entrada de ledger (este firmante único para la cuenta)                            | entero          |                     | Sí          |       |
| ledger_entry_change   | Código que describe el tipo de cambio aplicado a la entrada del ledger                                                                 | entero          |                     | Sí          |       |
| deleted               | Indica si la entrada del ledger (ID de balance) ha sido eliminada o no. Una vez que una entrada es eliminada, no puede ser recuperada  | booleano        |                     | Sí          |       |
| batch_id              | Representación en cadena del ID de ejecución para un DAG dado en Airflow                                                               | cadena          |                     | Sí          |       |
| batch_run_date        | Fecha de inicio del intervalo por lote. Cuando se usa junto con la fecha en el batch_id, representa el intervalo de ledgers procesados | fecha y hora    |                     | Sí          |       |
| batch_insert_ts       | Marca de tiempo UTC cuando un lote de registros fue insertado en la base de datos                                                      | marca de tiempo |                     | Sí          |       |
| closed_at             | Marca de tiempo UNIX de la antigüedad del número de secuencia                                                                          | marca de tiempo |                     | Sí          |       |
| ledger_sequence       | Número entero sin signo de 32 bits del ledger correspondiente a la antigüedad del número de secuencia                                  | entero          |                     | Sí          |       |
