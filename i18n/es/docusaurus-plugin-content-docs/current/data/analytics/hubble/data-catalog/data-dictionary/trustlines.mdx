## Metadatos de la Tabla

| Propiedad             | Configuración                                                                                              |
| --------------------- | ---------------------------------------------------------------------------------------------------------- |
| Clave(s) natural(es)  | account_id, asset_type, asset_issuer, asset_code, liquidity_pool_id, closed_at                             |
| Campo(s) de Partición | batch_run_date (partición MONTH)                                                                           |
| Campo(s) agrupado(s)  | account_id, asset_id, liquidity_pool_id, last_modified_ledger                                              |
| Documentación         | [dbt docs](http://www.stellar-dbt-docs.com/#!/source/source.stellar_dbt_public.crypto_stellar.trust_lines) |

## Detalles de la Columna

| Nombre               | Descripción                                                                                                                                                                                                                                                 | Tipo de Dato | Valores de Dominio                                                                                                                                                       | ¿Requerido? | Notas                                                                                                                                                                          |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ledger_key           | La clave de ledger única cuando el estado de la línea de confianza cambió por última vez                                                                                                                                                                    | string       |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| account_id           | La dirección de la cuenta                                                                                                                                                                                                                                   | string       |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| asset_type           | El identificador para el tipo de código de activo, puede ser alfanumérico con 4 caracteres, 12 caracteres o el activo nativo de la red, XLM.                                                                                                                | integer      | <ul><li>credit_alphanum4</li><li>credit_alphanum12</li><li>native</li></ul>                                                                                              | Sí          |                                                                                                                                                                                |
| asset_issuer         | La dirección de la cuenta del emisor original de activo que creó el activo mantenido por esta cuenta                                                                                                                                                        | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                |
| asset_code           | El código de 4 o 12 caracteres que representa el activo mantenido por esta cuenta                                                                                                                                                                           | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                |
| liquidity_pool_id    | Si el activo mantenido es parte de una participación en un fondo de liquidez, el id único del fondo del que proviene el saldo del activo                                                                                                                    | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                |
| balance              | El número de unidades de un activo mantenido por esta cuenta                                                                                                                                                                                                | float        |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| trust_line_limit     | La cantidad máxima de este activo que esta cuenta está dispuesta a aceptar. El límite se especifica al abrir una línea de confianza                                                                                                                         | integer      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| buying_liabilities   | La suma de todas las ofertas de compra poseídas por esta cuenta para activos no nativos                                                                                                                                                                     | float        |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| selling_liabilities  | La suma de todas las ofertas de venta poseídas por esta cuenta para activos no nativos                                                                                                                                                                      | float        |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| flags                | Denota la habilitación y desactivación de ciertos privilegios del emisor de activo                                                                                                                                                                          | integer      | <ul><li>0 - Ninguno, Predeterminado</li><li>1 - Autorizado</li><li>2 - Autorizado a Mantener Pasivos</li><li>4 - Recaudación Habilitada</li></ul>                        | Sí          | Las banderas se configuran en las cuentas emisoras de un activo. Cuando las cuentas de usuario confían en un activo, las banderas aplicadas al activo provienen de esta cuenta |
| last_modified_ledger | El número de serie de ledger cuando la entrada de ledger (este firmante único para la cuenta) fue modificada                                                                                                                                                | integer      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| ledger_entry_change  | Código que describe el tipo de cambio de entrada de ledger que fue aplicado a la entrada de ledger                                                                                                                                                          | integer      | <ul><li>0 - Entrada de Ledger Creada</li><li>1 - Entrada de Ledger Actualizada</li><li>2 - Entrada de Ledger Eliminada</li><li>3 - Estado de Entrada de Ledger</li></ul> | Sí          | Los tipos de cambio de entrada válidos son 0, 1 y 2 para entradas de ledger del tipo `trust_lines`.                                                                            |
| eliminado            | Indica si la entrada de ledger (línea de confianza) ha sido eliminada o no. Una vez que una entrada es eliminada, no se puede recuperar                                                                                                                     | boolean      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| closed_at            | Marca de tiempo en UTC cuando este ledger se cerró y fue confirmado en la red. Se espera que los ledgers se cierren aproximadamente cada 5 segundos                                                                                                         | timestamp    |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| ledger_sequence      | El número de secuencia de este ledger. Representa el orden del ledger dentro de la blockchain Stellar. Cada ledger tiene un número de secuencia único que se incrementa con cada nuevo ledger, asegurando que los ledgers se procesen en el orden correcto. | integer      |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| batch_id             | Representación en cadena del id de ejecución para un DAG dado en Airflow                                                                                                                                                                                    | string       |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| batch_run_date       | La fecha de inicio para el intervalo de batch. Cuando se toma junto con la fecha en el batch_id, la fecha representa el intervalo de ledgers procesados                                                                                                     | datetime     |                                                                                                                                                                          | Sí          | La tabla se particiona por batch_run_date. Se recomienda incluir siempre el batch_run_date en el filtro si es posible para ayudar a reducir el costo de las consultas.         |
| batch_insert_ts      | La marca de tiempo en UTC cuando se insertó un batch de registros en la base de datos                                                                                                                                                                       | timestamp    |                                                                                                                                                                          | Sí          |                                                                                                                                                                                |
| sponsor              | La dirección de la cuenta que patrocina las reservas básicas para la línea de confianza                                                                                                                                                                     | string       |                                                                                                                                                                          | No          |                                                                                                                                                                                |
| asset_id             | Identificador único para asset_code, asset_issuer                                                                                                                                                                                                           | integer      |                                                                                                                                                                          | No          |                                                                                                                                                                                |
