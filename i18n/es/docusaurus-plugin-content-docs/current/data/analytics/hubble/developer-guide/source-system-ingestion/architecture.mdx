## Descripción general de la arquitectura

<img src="/assets/hubble/stellar-overall-architecture.png" width="600" alt="Arquitectura General Stellar" />

<img src="/assets/hubble/stellar-etl-architecture.png" width="300" alt="Arquitectura De Stellar ETL" />

En general, stellar-etl se ejecuta mediante:

- Leer datos en bruto de la red Stellar
- Esto se puede hacer ejecutando un comando de exportación de stellar-etl para exportar datos entre un ledger de inicio y uno de fin
  - stellar-etl tiene la capacidad de leer de dos fuentes diferentes:
    - Captive-core directamente para obtener LedgerCloseMeta
    - Un lago de datos de archivos comprimidos LedgerCloseMeta del Ledger Exporter
- Transforma el LedgerCloseMeta XDR en un formato JSON fácil de analizar
- Opcionalmente, sube los archivos JSON a GCS o a cualquier otro servicio de almacenamiento en la nube

Se puede encontrar información más detallada sobre stellar-etl y ejemplos en el repositorio [stellar-etl](https://github.com/stellar/stellar-etl/tree/master).
