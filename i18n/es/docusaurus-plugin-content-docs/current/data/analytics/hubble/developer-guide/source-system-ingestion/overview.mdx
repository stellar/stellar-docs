La ingesta de datos de la red Stellar en Hubble se realiza a través de [stellar-etl](https://github.com/stellar/stellar-etl/tree/master). stellar-etl lee y transforma datos de la red Stellar en archivos JSON compatibles con OLAP.

| Caso de uso                                                                                               | Solución recomendada                                                                                                                                          | Razón                                                                                                                                                                                                                                                                                                                              |
| --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Necesitas campos de datos comúnmente requeridos y no deseas asumir la carga de configurar toda la tubería | Usa la salida de la instancia stellar-etl de SDF, es decir, [Conectar a Bigquery y acceder a Hubble](../../developer-guide/connecting-to-bigquery/README.mdx) | Vale la pena notar que la mayoría de los usuarios no necesitarán levantar y ejecutar su propia instancia stellar-etl. La Stellar Development Foundation proporciona acceso público a datos de la red Stellar completamente transformados a través de los conjuntos de datos y tablas públicas en GCP BigQuery.                     |
| Necesitas datos personalizados pero no quieres asumir la carga de configurar toda la tubería              | Usa [JS-UDF](../../developer-guide/backfill/JS-UDF.mdx)                                                                                                       | Stellar-ETL no analiza cada campo disponible en crudo XDR, pero guarda la metadata de la transacción cruda. JS-UDF ayuda a extraer un campo directamente de XDR                                                                                                                                                                    |
| Necesitas datos personalizados y operativamente independientes                                            | Ejecuta [stellar-etl](https://github.com/stellar/stellar-etl/tree/master)                                                                                     | Ejecutar stellar-etl dentro de tu propia infraestructura ofrece una serie de beneficios. Puedes: <br></br> - Tener control operativo total sin depender de la Stellar Development Foundation para obtener datos de red <br></br> - Ejecutar tuberías ETL/ELT modificadas que se ajusten a las necesidades de tu negocio individual |
