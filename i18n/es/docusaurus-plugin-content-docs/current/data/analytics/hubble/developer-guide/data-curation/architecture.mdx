## Descripción general de la arquitectura

<img src="/assets/hubble/stellar-dbt-architecture.png" width="300" alt="Arquitectura Stellar DBT" />

En general, stellar-dbt-public se ejecuta por:

- Seleccionar un modelo de dbt para ejecutar
- Dentro de la ejecución del modelo:
  - Las fuentes se referencian y se utilizan para crear tablas de preparación
  - Las tablas de preparación luego pasan por varias transformaciones y se almacenan en tablas intermedias
  - Los toques finales y las uniones se realizan en las tablas intermedias, que producen las tablas finales amigables para el análisis

Tratamos de adherirnos a las prácticas recomendadas establecidas por la [dbt docs](https://docs.getdbt.com/docs/build/projects)

Información más detallada sobre stellar-dbt-public y ejemplos se puede encontrar en el repositorio de [stellar-dbt-public](https://github.com/stellar/stellar-dbt-public/tree/master).
