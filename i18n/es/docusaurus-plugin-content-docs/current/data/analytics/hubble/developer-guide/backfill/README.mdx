Una vez que hayas configurado tu programación y orquestación, es posible que encuentres los siguientes escenarios:

| Caso de Uso                          | Descripción                                                                                                                              | Solución                                                                                                                                                                                                                                                                                           |
| ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Relleno Inicial                      | Acabas de configurar Hubble y te gustaría ingerir datos históricos                                                                       | - Opción 1. [Importación de Datos](../../developer-guide/backfill/data-import.mdx) <br></br> **Pros:** Barato y rápido <br></br>- Opción 2: Reactivar los DAG para fechas pasadas <br></br> **Contras:** Lento y caro                                                                              |
| Corrección de Errores                | Has resuelto un error y necesitas reingerir una columna de datos específica o corregir una columna de datos                              | - Opción 1. [JS UDF](../../developer-guide/backfill/JS-UDF.mdx) <br></br> **Pros:** Barato y rápido <br></br> **Contras:** Puede necesitar escritura de consultas optimizadas y ejecución en lotes <br></br> - Opción 2: Reactivar los DAG para fechas pasadas <br></br> **Contras:** Lento y caro |
| Extracción de nueva columna de datos | Has añadido una nueva columna de datos como parte de una solicitud de funciones y necesitas rellenar datos para la nueva columna añadida | - Opción 1. [JS UDF](../../developer-guide/backfill/JS-UDF.mdx) <br></br> **Pros:** Barato y rápido <br></br> **Contras:** Puede necesitar escritura de consultas optimizadas y ejecución en lotes <br></br> - Opción 2: Reactivar los DAG para fechas pasadas <br></br> **Contras:** Lento y caro |
