## Proveedores de ecosistema

Múltiples proveedores de infraestructura han puesto a disposición los servicios de endpoint RPC Soroban y ofrecen planes que van desde puntos finales gratuitos hasta de alto rendimiento. Estos proveedores pueden ser utilizados para el desarrollo, las pruebas y la producción.

Estos proveedores permiten el acceso a las redes Testnet y Mainnet.

| Proveedor                                                                         | Acceder                                             | Futuro        | Testnet       | Mainnet    |
| --------------------------------------------------------------------------------- | --------------------------------------------------- | ------------- | ------------- | ---------- |
| [Blockdaemon\*](https://docs.blockdaemon.com/reference/how-to-access-soroban-api) | [Access](https://www.blockdaemon.com/apply/soroban) | No disponible | Disponible    | Disponible |
| [Validation Cloud\*](https://validationcloud.io/)                                 | [Access](https://app.validationcloud.io/)           | No disponible | Disponible    | Disponible |
| [QuickNode\*](https://www.quicknode.com/docs/stellar)                             | [Access](https://www.quicknode.com/chains/stellar)  | No disponible | Disponible    | Disponible |
| [NowNodes](https://nownodes.io/nodes/stellar-xlm)                                 | [Access](https://nownodes.io/nodes/stellar-xlm)     | Disponible    | Disponible    | Disponible |
| [Gateway](https://gateway.fm/)                                                    | [Access](https://gateway.fm/public-rpc/)            | No disponible | Disponible    | Disponible |
| [Ankr](https://www.ankr.com/rpc/advanced-api/)                                    | [Access](https://www.ankr.com/rpc/advanced-api/)    | No disponible | Disponible    | Disponible |
| [Infstones](https://infstones.com/)                                               | [Access](https://cloud.infstones.com/login)         | No disponible | No disponible | Disponible |

\*_Blockdaemon, Validation Cloud, y Quicknode combinan Horizon y RPC en el mismo punto final._

## SDF Proporcionado RPC (Futurenet y Testnet solamente) {#sdf-provided-rpc}

SDF ha puesto a disposición RPC Soroban con acceso a las redes Futurenet y Testnet. Estos servicios son de uso libre y son aptos para el desarrollo y la prueba.

SDF no proporcionará un endpoint RPC disponible públicamente para Mainnet. Los desarrolladores deben seleccionar un proveedor de ecosistemas que funcione para su proyecto antes de migrar a Mainnet. En algunos casos, los proyectos pueden elegir [ejecutar su propia instancia RPC](./admin-guide.mdx).

| Red         | Extremo RPC                           | Amigobot Endpoint                         | Proveedor                     |
| ----------- | ------------------------------------- | ----------------------------------------- | ----------------------------- |
| `Futurenet` | `https://rpc-futurenet.stellar.org`   | `https://friendbot-futurenet.stellar.org` | [SDF](http://www.stellar.org) |
| `Testnet`   | `https://soroban-testnet.stellar.org` | `https://friendbot.stellar.org`           | [SDF](http://www.stellar.org) |
