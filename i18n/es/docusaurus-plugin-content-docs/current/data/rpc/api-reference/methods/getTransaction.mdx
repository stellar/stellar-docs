### Guía de SDK

El ejemplo anterior es consultar los detalles de una transacción usando métodos RPC directamente. Si está utilizando el SDK Stellar para construir aplicaciones, puede utilizar las funciones nativas para obtener la misma información.

```python
# pip install --upgrade stellar-sdk
from stellar_sdk import SorobanServer, soroban_rpc


def get_transaction(hash: str) -> soroban_rpc.GetTransactionResponse:
    server = SorobanServer(server_url='https://soroban-testnet. tellar.org', client=Ninguno)
    tx = server.get_transaction(hash)
    return tx

tx = get_transaction("6bc97bddc21811c626839baf4ab574f4f9f7ddbebb44d286ae504396d4e752da")

print("result", tx.status)
```
