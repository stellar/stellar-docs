# DETÉN LO QUE ESTÁS HACIENDO

Si estás aquí para hacer algunas correcciones en la documentación de la API de Stellar (¡antes que nada, gracias!), probablemente no quieras estar en esta ubicación. La mayor parte de la documentación de la API ubicada aquí ha sido generada por los archivos `.yml` contenidos en el directorio `/openapi`. A menos que estés trabajando en algo que esté _fuera_ del directorio `/api/resources`, probablemente quieras revisar `/openapi` en lugar de aquí.
