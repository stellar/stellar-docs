Hubble publica metadatos que pueden ayudar a los usuarios a determinar qué tablas consultar, con qué frecuencia se actualizan los conjuntos de datos, e información general sobre el conjunto de datos.

Hay dos maneras de acceder a esta información:

## Explorador BigQuery

Al acceder a Hubble desde su enlace favorito, el panel Explorador cargará metadatos sobre el conjunto de datos `crypto-stellar.crypto_stellar`.

Utilice la tecla Alternar para ver el contenido del conjunto de datos. Al hacer clic en el nombre de una tabla se cargará lo siguiente:

- _Schema_ - información detallada sobre el esquema de la tabla, incluyendo definiciones de columnas y tipos de datos. Ver el esquema ayuda a escribir una consulta SQL
- _Details_ - información general sobre la tabla en sí, incluyendo particionamiento, clustering y tamaño de tabla. Ver detalles ayuda con la optimización de consultas
- _Vista previa_ - datos crudos de la tabla. Los datos presentados son el equivalente a ejecutar una sentencia `SELECT *`

## INFORMACION_ESCHEMA

BigQuery soporta vistas de sólo lectura definidas por el sistema que proporcionan información de metadatos sobre objetos BigQuery. Las vistas se pueden consultar a través de SQL desde la interfaz de usuario de BigQuery o las bibliotecas de clientes.

:::note

Las consultas ejecutadas contra `INFORMATION_SCHEMA` no pueden ser almacenadas en caché y **serán** incurrirán en cargos de procesamiento de datos por cada ejecución.

:::

Desde el editor de BigQuery, la siguiente consulta listará todas las tablas en Hubble:

<CodeExample>

```sql
# Lista todas las tablas en Hubble
#standardSQL
seleccione *
de `crypto-stellar.crypto_stellar`.INFORMATION_SCHEMA.TABLES;
```

</CodeExample>

Si desea detalles en una tabla en particular, puede devolver el esquema de la tabla:

<CodeExample>

```sql
# Lista todas las columnas de la tabla de cuentas
select table_name,
  column_name,
  is_nullable,
  data_type,
  is_partitioning_column
de `crypto-stellar. rypto_stellar`.INFORMATION_SCHEMA.COLUMNS
donde table_name = "cuentas";
```

</CodeExample>

Más en el `INFORMATION_SCHEMA` se puede encontrar [here](https://cloud.google.com/bigquery/docs/information-schema-intro).
