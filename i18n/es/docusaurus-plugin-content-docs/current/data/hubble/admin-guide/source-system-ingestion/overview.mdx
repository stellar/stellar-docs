La ingestión de datos de red estelar en Hubble se realiza a través de [stellar-etl](https://github.com/stellar/stellar-etl/tree/master). stellar-etl lee y transforma los datos de la red estelar en archivos JSON amigables OLAP.

Vale la pena señalar que la mayoría de los usuarios no necesitarán ponerse de pie y ejecutar su propia instancia stellar-etl. La Fundación Stellar Development proporciona acceso público a los datos de la red Stellar completamente transformados a través de los conjuntos de datos públicos y tablas en GCP BigQuery. Las instrucciones sobre cómo acceder a estos datos se pueden encontrar en la sección [Connecting](../../analyst-guide/connecting.mdx).

## ¿Por qué Run stellar-etl?

Ejecutar stellar-etl dentro de su propia infraestructura proporciona una serie de beneficios. Puedes:

- Tener un control operativo completo sin depender de la Fundación de Desarrollo Estelar para datos de red
- Ejecutar tuberías ETL/ELT modificadas que se ajusten a sus necesidades individuales de negocio
