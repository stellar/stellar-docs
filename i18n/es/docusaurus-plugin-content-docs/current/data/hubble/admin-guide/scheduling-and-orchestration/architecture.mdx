## Resumen de Arquete

<img src={stellar_etl_airflow_arch} width="300" />

En general estellar-etl-airflow corre por:

- Programando DAGs para ejecutar `stellar-etl` y subir los datos salidos a BigQuery
- Programar DAGs para ejecutar `stellar-dbt-public` usando los datos en BigQuery
  - Intentamos adherirnos a las mejores prácticas establecidas por [dbt docs](https://docs.getdbt.com/docs/build/projects)

Puede encontrar información más detallada sobre stellar-etl-airflow en el repositorio [stellar-etl-airflow](https://github.com/stellar/stellar-etl-airflow/tree/master).
