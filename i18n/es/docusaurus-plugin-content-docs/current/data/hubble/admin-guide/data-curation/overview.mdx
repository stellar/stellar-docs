La curación de datos en Hubble se realiza a través de [stellar-dbt-public](https://github.com/stellar/stellar-dbt-public). stellar-dbt-public transforma datos de red estelar crudos de BigQuery y tablas en agregados para análisis más fáciles de usar.

Vale la pena señalar que la mayoría de los usuarios no tendrán que ponerse de pie y ejecutar su propia instancia estelar-dbt-público. La Fundación Stellar Development proporciona acceso público a los datos de la red Stellar completamente transformados a través de los conjuntos de datos públicos y tablas en GCP BigQuery. Las instrucciones sobre cómo acceder a estos datos se pueden encontrar en la sección [Connecting](../../analyst-guide/connecting.mdx).

## ¿Por qué Run stellar-dbt-public?

Ejecutar stellar-dbt-public dentro de su propia infraestructura proporciona una serie de beneficios. Puedes:

- Tener un control operativo completo sin depender de la Fundación de Desarrollo Estelar para datos de red
- Ejecutar tuberías ETL/ELT modificadas que se ajusten a sus necesidades individuales de negocio
