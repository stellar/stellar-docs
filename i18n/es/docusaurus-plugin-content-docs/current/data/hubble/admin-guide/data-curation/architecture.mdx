## Resumen de Arquete

<img src={stellar_dbt_arch} width="300" />

En general stellar-dbt-public corre por:

- Seleccionando un modelo dbt para ejecutar
- Dentro del modelo de ejecución:
  - Las fuentes son referenciadas y usadas para crear tablas de staging
  - Las tablas provisionales se someten a varias transformaciones y se almacenan en tablas intermedias
  - Los toques de acabado y las juntas se hacen en las tablas intermedias que producen las tablas analíticas finales amigables con los inteligentes

Intentamos adherirnos a las mejores prácticas establecidas por [dbt docs](https://docs.getdbt.com/docs/build/projects)

Información más detallada sobre stellar-dbt-public y ejemplos puede encontrarse en el repositorio [stellar-dbt-public](https://github.com/stellar/stellar-dbt-public/tree/master).
