Cuando Horizon devuelve información sobre una ruta, utiliza el siguiente formato:

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- tipo_fuente
  - cadena
  - El tipo del activo de origen. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- código fuente
  - cadena
  - El código del activo fuente.
- emisor de recursos
  - cadena
  - La dirección Stellar del emisor del activo fuente.
- importe_origen
  - cadena
  - Coste estimado por hacer un pago de la cantidad de destino en esta ruta. Adecuado para su uso en el campo `sendMax` de una operación de pago de ruta.
- tipo de recurso de destino
  - cadena
  - El tipo del activo de destino. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- código de destino
  - cadena
  - El código del activo de destino.
- emisor de destino
  - cadena
  - La dirección estelar del emisor del activo de destino.
- monto de destino
  - cadena
  - La cantidad de destino especificada en la búsqueda que encontró esta ruta.
- ruta
  - array de objetos
  - Los activos intermediarios por los que se salta esta ruta.
    - asset_code
      - cadena
      - El código de este activo intermediario.
    - emisor de activo
      - cadena
      - La dirección estelar del emisor del activo intermediario.
    - asset_type
      - cadena
      - El tipo del activo intermediario. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.

</AttributeTable>
