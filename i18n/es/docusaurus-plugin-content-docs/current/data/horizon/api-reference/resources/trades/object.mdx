Cuando Horizon devuelve información sobre un intercambio, utiliza el siguiente formato:

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- id
  - cadena
  - Un identificador único para esta operación.
- ficha_paginando
  - número
  - Un valor del cursor para usar en [pagination](../../structure/pagination/README.mdx).
- cierrar_tiempo
  - cadena
  - Una cadena con formato ISO 8601 de cuando se cerró el libro de valores con esta operación.
- cuenta_base
  - cadena (opcional)
  - El ID de la cuenta de la parte base para esta operación.
- id_oferta base
  - cadena (opcional)
  - La base ofrece ID. Si esta oferta fue consumida de forma inmediata y completa, será un ID sinético.
- base_liquididad_pool_id
  - cadena (opcional)
  - La piscina base de liquidez ID. Si este intercambio fue ejecutado contra un fondo de liquidez.
- monto_base
  - cadena
  - La cantidad del activo base que fue movido de `base_account` a `counter_account`.
- tipo_base
  - cadena
  - El tipo del activo base. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- código de recurso
  - cadena
  - El código del activo base.
- emisor de recurso
  - cadena
  - La dirección estelar del emisor del activo base.
- contador_cuenta
  - cadena (opcional)
  - El ID de la cuenta de la parte base para esta operación.
- id_contador_oferta
  - cadena (opcional)
  - El contador ofrece ID. Si esta oferta fue consumida de forma inmediata y completa, será un ID sinético.
- contador_liquidez_pool_id
  - cadena (opcional)
  - La piscina de liquidez contra ID. Si este intercambio fue ejecutado contra un fondo de liquidez.
- monto_contador
  - cadena
  - La cantidad del activo contador que fue movido de `counter_account` a `base_account`.
- contador_tipo_activo
  - cadena
  - El tipo de activo del contador. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- código de contador_recurso
  - cadena
  - El código del activo del contador.
- contar_activo_emisor
  - cadena
  - La dirección estelar de la contra-emisora del activo.
- precio
  - objeto
  - Un objeto de numerador numérico y denominador que representa el precio de oferta original. Para derivar el precio, divida `n` entre `d`. `n` y `d` se representan como cadenas porque pueden ser enteros de 64 bits.
    - n
      - cadena
      - El numerador.
    - d
      - cadena
      - El denominador.
- base_es_vendedor
  - boolean
  - Indica con la fiesta es el vendedor.

</AttributeTable>
