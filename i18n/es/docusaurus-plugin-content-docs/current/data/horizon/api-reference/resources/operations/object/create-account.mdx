Crea y financia una nueva cuenta con el saldo inicial especificado.

Vea los [errores de 'Crear cuenta'](../../../errors/result-codes/operation-specific/create-account.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- saldo_inicial
  - cadena
  - La cantidad de XLM para enviar la cuenta recién creada.
- financiador
  - cadena
  - La cuenta que financia la nueva cuenta.
- cuenta
  - cadena
  - Una nueva cuenta que está financiada.

</AttributeTable>
