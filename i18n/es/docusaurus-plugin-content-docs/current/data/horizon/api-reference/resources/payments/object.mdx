Envía una cantidad en un activo específico a una cuenta de destino.

Vea los [errores de `Payment`](../../errors/result-codes/operation-specific/payment.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- asset_type
  - cadena
  - El tipo de activo que se está enviando. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- asset_code
  - cadena
  - El código para el activo que se está enviando. Aparece si el `asset_type` no es `native`.
- emisor de activo
  - cadena
  - La dirección Stellar del emisor del activo a ser enviado. Aparece si el `asset_type` no es `native`.
- de
  - cadena
  - Clave pública del remitente de pagos.
- a
  - cadena
  - La clave pública del destinatario del pago.
- monto
  - cadena
  - Cantidad enviada.

</AttributeTable>
