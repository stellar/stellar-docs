Actualiza la bandera “autorizada” de una línea de confianza existente. Esto debe ser invocado por el emisor del activo.

Vea los [errores `Permitir Trust`](../../../errors/result-codes/operation-specific/allow-trust.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- asset_type
  - cadena
  - El tipo de activo. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- asset_code
  - cadena
  - La dirección estelar del activo.
- emisor de activo
  - cadena
  - El código del activo.
- autorizar
  - int
  - Marca que indica si la línea de confianza está autorizada. 0 si la cuenta no está autorizada a transaccionar con el activo de ninguna manera. 1 si la cuenta está autorizada a transaccionar con el activo. 2 si la cuenta está autorizada a mantener órdenes, pero no a realizar otras transacciones.
- fideicomisario
  - cadena
  - La cuenta de emisión, o la cuenta de origen en esta instancia.
- trustor
  - cadena
  - La cuenta de confianza, o la cuenta que está siendo autorizada o no autorizada.

</AttributeTable>
