Cuando Horizon devuelve información sobre una oferta, utiliza el siguiente formato:

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- id
  - cadena
  - Un identificador único para esta oferta.
- ficha_paginando
  - número
  - Un valor del cursor para usar en [pagination](../../structure/pagination/README.mdx).
- vendedor
  - cadena
  - El ID de cuenta de la cuenta haciendo esta oferta.
- venta
  - código de activo
  - El activo que esta oferta quiere vender.
- comprar
  - código de activo
  - El activo que esta oferta quiere comprar.
- monto
  - cadena
  - La cantidad de 'vender' que la cuenta que realiza esta oferta está dispuesta a vender.
- precio_r
  - objeto
  - Una representación precisa del precio de compra y venta de los activos en oferta.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- precio
  - cadena
  - Cuántas unidades de 'comprar' se necesitan para obtener 1 unidad de 'vender'. Un número que representa la forma decimal de `price_r`.
- última modificación_ledger
  - entero
  - Número de secuencia del último libro en el que se modificó esta oferta.
- última modificación_hora
  - cadena
  - Una cadena con formato ISO 8601 de última modificación.
- patrocinador
  - cadena (opcional)
  - El identificador de cuenta del patrocinador que está pagando las reservas para esta oferta.

</AttributeTable>
