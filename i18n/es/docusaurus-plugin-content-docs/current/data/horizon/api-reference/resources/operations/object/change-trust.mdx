Crea, actualiza o elimina una línea de fideicomiso de la cuenta de origen al activo emitido por otra cuenta.

Vea los [errores `Cambiar Trust`](../../../errors/result-codes/operation-specific/change-trust.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- asset_type
  - cadena
  - El tipo de activo en el que se confía, uno de `native`, `credit_alphanum4`, `credit_alphanum12` o `liquidity_pool_shares`.
- asset_code
  - cadena
  - La dirección estelar del activo de confianza. (Solo presente para activos de crédito.)
- emisor de activo
  - cadena
  - El código para el activo de confianza. (Solo presente para activos de crédito.)
- límite
  - cadena
  - Limita la cantidad de un activo que puede contener la cuenta de origen.
- fideicomisario
  - cadena
  - La cuenta de emisión. (Solo presente para activos de crédito.)
- trustor
  - cadena
  - La cuenta de origen.
- liquidez_pool_id
  - cadena
  - La piscina de liquidez cuya línea de confianza está siendo modificada. (Solo presente para `asset_type == liquididada_pool_shares`.)

</AttributeTable>
