[Invokes](../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#invoke-host-function) una función Soroban de contrato inteligente.

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- función
  - cadena
  - El tipo de `InvokeHostFunctionOp` que puede ser uno de `HostFunctionTypeHostFunctionTypeInvokeContract`, `HostFunctionTypeHostFunctionTypeCreateContract`, o `HostFunctionTypeHostFunctionTypeUploadContractWasm`.
- parámetros
  - matriz
  - Una matriz de parámetros pasados a la función Soroban de contrato inteligente (sólo presente cuando el tipo de `InvokeHostFunctionOp` es `HostFunctionTypeHostFunctionTypeInvokeContract`).
    - tipo
      - cadena
      - La [type](../../../../../../learn/encyclopedia/contract-development/types/built-in-types.mdx) del parámetro de función.
    - valor
      - cadena
      - La codificación base64 del valor XDR del parámetro.
- dirección
  - cadena
  - La dirección del contrato recién creado (sólo presente cuando el tipo de `InvokeHostFunctionOp` es `HostFunctionTypeHostFunctionTypeCreateContract`).
- sal
  - cadena
  - La sal utilizada para crear el contrato (sólo presente cuando el tipo de `InvokeHostFunctionOp` es `HostFunctionTypeHostFunctionTypeCreateContract`).
- activo_balance_cambios
  - matriz
  - Una matriz de [Contrato de Activos Estelar](../../../../../../../.. tokens/stellar-asset-contract) balance de eventos que ocurrieron como un efecto secundario de la `InvokeHostFunctionOp` (sólo presente cuando el tipo de `InvokeHostFunctionOp` es `HostFunctionTypeHostFunctionTypeInvokeContract` _y_ la invocación está en un Contrato de Activos Estelares).
    - asset_type
      - cadena
      - El tipo de activo que se está enviando. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
    - código
      - cadena
      - El código para el activo que se está enviando. Aparece si el `asset_type` no es `native`.
    - emisor
      - cadena
      - La dirección Stellar del emisor del activo a ser enviado. Aparece si el `asset_type` no es `native`.
    - tipo
      - cadena
      - El tipo del [evento del Contrato de Activos Estelar](../../../../../../../tokens/stellar-asset-contract). Puede ser uno de `transfer`, `mint`, `clawback`, o `burn`.
    - de
      - cadena
      - La dirección estelar del remitente (puede ser un contrato o una cuenta Stellar). Este campo no está presente cuando `type` es `mint`.
    - a
      - cadena
      - La dirección estelar del destinatario (puede ser un contrato o una cuenta Stellar). Este campo sólo está presente cuando `type` es `transfer` o `mint`.
    - monto
      - cadena
      - La cantidad enviada.

</AttributeTable>
