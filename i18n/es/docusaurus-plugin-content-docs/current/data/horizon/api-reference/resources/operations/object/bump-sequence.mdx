Salta hacia adelante el número de secuencia de la cuenta de origen, permitiéndole invalidar cualquier transacción con un número de secuencia más pequeño.

Vea los [errores `Bump Sequence`](../../../errors/result-codes/operation-specific/bump-sequence.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- bump_to
  - cadena
  - El nuevo valor deseado para el número de secuencia de la cuenta fuente.

</AttributeTable>
