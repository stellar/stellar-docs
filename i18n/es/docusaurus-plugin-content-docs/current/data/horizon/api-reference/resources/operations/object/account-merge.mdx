Elimina la cuenta de origen del Stellar y transfiere los lumens de la cuenta de origen a otra cuenta.

Vea los [`Combinación de cuentas`](../../../errors/result-codes/operation-specific/account-merge.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- cuenta
  - cadena
  - La dirección estelar que se está eliminando.
- en
  - cadena
  - La dirección estelar que recibe los lumens de la cuenta eliminada.

</AttributeTable>
