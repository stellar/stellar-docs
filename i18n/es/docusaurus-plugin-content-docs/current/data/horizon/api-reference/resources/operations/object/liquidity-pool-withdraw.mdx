Retira reservas de activos de una piscina de liquidez mediante el canje de acciones de piscina.

Vea la [operación `Liquidity Pool Withdraw`](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#liquidity-pool-quit) para obtener parámetros, errores, etc.

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- liquidez_pool_id
  - cadena
  - La batería de liquidez asociada con este retiro
- reservas_min
  - matriz
  - Una matriz de objetos correspondiente a la cantidad mínima de cada reserva que debería haber sido retirada
    - asset
      - cadena
      - El activo en forma canónica (`Code:Emisor`)
    - monto
      - cadena
      - Un valor de punto flotante codificado como una cadena
- compartir
  - cadena
  - El número de acciones canjeadas por esta operación de retiro
- reservas_recibidas
  - matriz
  - Una matriz de objetos que representan cuánto de cada reserva terminó realmente retirado del pool
    - asset
      - cadena
      - El activo en forma canónica (`Code:Emisor`)
    - monto
      - cadena
      - Un valor de punto flotante codificado como una cadena

</AttributeTable>
