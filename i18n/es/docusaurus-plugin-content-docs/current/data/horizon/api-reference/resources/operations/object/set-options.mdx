Establece las banderas de una cuenta, destino inflacionario, signatarios y dominio doméstico.

Vea los [`Configurar opciones`](../../../errors/result-codes/operation-specific/set-options.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- clave_firmante
  - cadena
  - La clave pública del nuevo firmante.
- peso del firmante
  - número
  - El peso del nuevo firmante. Puede ir de `1` a `255`.
- peso de clave maestro
  - número
  - El peso de la clave maestra. Puede ir de `1` a `255`.
- umbral bajo
  - número
  - El peso de la suma para el umbral bajo.
- umbral medido
  - número
  - El peso de la suma para el umbral medio.
- umbral alto
  - número
  - El peso de la suma para el umbral alto.
- dominio_inicio
  - cadena
  - El dominio de inicio utilizado para el descubrimiento de archivos stellar.toml.
- set_banderas
  - matriz
  - La matriz de valores numéricos de banderas que se ha establecido en esta operación. Las opciones incluyen `1` para `AUTH_REQUIRED_FLAG`, `2` para `AUTH_REVOCABLE_FLAG`, y `4` para `AUTH_IMMUTABLE_FLAG`.
- set_banderas_s
  - matriz
  - El array de valores de cadenas de parámetros que se ha establecido en esta operación. Options include `AUTH_REQUIRED_FLAG`, `AUTH_REVOCABLE_FLAG`, and `AUTH_IMMUTABLE_FLAG`.
- limpiar_banderas
  - matriz
  - La matriz de valores numéricos de banderas que ha sido borrada en esta operación. Las opciones incluyen `1` para `AUTH_REQUIRED_FLAG`, `2` para `AUTH_REVOCABLE_FLAG`, y `4` para `AUTH_IMMUTABLE_FLAG`.
- limpiar_bandera_s
  - matriz
  - El array de valores de cadena de banderas que ha sido borrado en esta operación. Options include `AUTH_REQUIRED_FLAG`, `AUTH_REVOCABLE_FLAG`, and `AUTH_IMMUTABLE_FLAG`.

</AttributeTable>
