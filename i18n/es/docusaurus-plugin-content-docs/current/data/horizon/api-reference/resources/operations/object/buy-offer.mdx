Crea, actualiza o elimina una oferta de compra de activos operativos. Una oferta de compra especifica una cierta cantidad del activo de compra que debe ser vendido a cambio de la cantidad mínima del activo de venta.

Vea los [errores de `Manage Buy Ofer`](../../../errors/result-codes/operation-specific/manage-buy-offer.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- monto
  - cadena
  - La cantidad de `buying_asset` que la cuenta que realiza esta oferta está dispuesta a comprar.
- precio
  - cadena
  - Cuántas unidades de `buying_asset` se necesitan para obtener 1 unidad de `selling_asset`. Un número que representa la forma decimal de `price_r`.
- precio_r
  - objeto
  - Una representación precisa del precio de compra y venta de los activos en oferta.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- comprar_tipo_activo
  - cadena
  - El tipo para el activo de compra. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- comprar_activo_emisor
  - cadena
  - La dirección Stellar del emisor del activo de compra. Aparece si el `buying_asset_type` no es `native`.
- comprar_codigo activo
  - cadena
  - El código para el activo de compra. Aparece si el `buying_asset_type` no es `native`.
- selling_tipo_activo
  - cadena
  - El tipo de activo vendido. O sea `native`, `credit_alphanum4`, o `credit_alphanum12`.
- emisor de recursos
  - cadena
  - La dirección Stellar del emisor del activo de venta. Aparece si el `selling_asset_type` no es `native`.
- selling_código activo
  - cadena
  - El código para el activo de venta. Aparece si el `selling_asset_type` no es `native`.
- oferta_id
  - cadena
  - Un identificador único para esta oferta.

</AttributeTable>
