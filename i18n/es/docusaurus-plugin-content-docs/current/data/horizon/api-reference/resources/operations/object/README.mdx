Cada una de las operaciones de Stellar tiene formas de respuesta únicas. Debajo están los atributos que son comunes a través de los objetos de operación individual.

Vea los [errores genéricos de operación](../../../errors/result-codes/operations.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- id
  - número
  - Número de ID de la operación.
- ficha_paginando
  - cadena
  - Un valor del cursor para usar en [`pagination`](../../structure/pagination/README.mdx).
  - A cursor value for use in [pagination](../../../structure/pagination/README.mdx).
- type_i
  - número
  - Un número que indica el tipo de operación.
- tipo
  - cadena
  - El nombre del tipo de operación.
- hash de transacción
  - cadena
  - Un identificador único para la transacción a la que pertenece esta operación.
- transacción_exitosa
  - boolean
  - Indica si esta operación fue parte de una transacción exitosa.
- cuenta de origen
  - cadena
  - La cuenta que origina la operación.
- creado_en
  - cadena
  - La fecha de creación de esta operación.

</AttributeTable>
