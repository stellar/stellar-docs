Revocar el patrocinio de una entrada en el libro de notas.

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- cuenta_id
  - cadena (opcional)
  - El id de la cuenta que ya no está patrocinada.
- claimable_balance_id
  - cadena (opcional)
  - La identificación del saldo reclamable que ya no está patrocinada.
- datos_cuenta_id
  - cadena (opcional)
  - El id de la cuenta cuya entrada de datos ya no está patrocinada.
- data_name
  - cadena (opcional)
  - El nombre de la entrada de datos que ya no está patrocinada.
- oferta_id
  - cadena (opcional)
  - El id de la oferta que ya no está patrocinada.
- línea de confianza_id
  - cadena (opcional)
  - El id de la cuenta cuya línea de confianza ya no está patrocinada.
- trustline_asset
  - cadena (opcional)
  - El activo de la línea de confianza que ya no está patrocinado.
- identificador_cuenta
  - cadena (opcional)
  - El identificador de cuenta del firmante que ya no está patrocinado.
- clave_firmante
  - cadena (opcional)
  - El tipo de firmante que ya no está patrocinado.

</AttributeTable>
