Deposite reservas de activos en una piscina de liquidez.

Vea la [operación `Liquidity Pool Deposit`](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#liquidity-pool-deposit) para obtener parámetros, errores, etc.

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATA
  - DESCRIPCIÓN
- liquidez_pool_id
  - cadena
  - La piscina de liquidez asociada con este depósito
- reservas_máx
  - matriz
  - Una matriz de objetos que corresponden a la cantidad máxima de cada reserva que podría haber sido depositada
    - asset
      - cadena
      - El activo en forma canónica (`Code:Emisor`)
    - monto
      - cadena
      - Un valor de punto flotante codificado como una cadena
- min_precio
  - cadena
  - Un valor de punto flotante codificado como una cadena que indica el tipo de cambio mínimo para esta operación de depósito
- mín_precio_r
  - objeto
  - Una representación precisa (fraccional) del precio de compra y venta de los activos que se ofrecen.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- precio máximo
  - cadena
  - Un valor de punto flotante codificado como una cadena que indica el tipo de cambio máximo para esta operación de depósito
- precio máximo
  - objeto
  - Una representación precisa (fraccional) del precio de compra y venta de los activos que se ofrecen.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- reservas_depositadas
  - matriz
  - Una matriz de objetos que representan la cantidad de cada reserva terminó realmente depositada en el pool
    - asset
      - cadena
      - El activo en forma canónica (`Code:Emisor`)
    - monto
      - cadena
      - Un valor de punto flotante codificado como una cadena
- acciones_recibida
  - cadena
  - Un valor de punto flotante codificado como una cadena que representa el número de acciones de fondo recibidas para este depósito

</AttributeTable>
