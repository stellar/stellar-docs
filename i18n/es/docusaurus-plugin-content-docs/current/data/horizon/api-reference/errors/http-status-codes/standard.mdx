Estas respuestas son códigos de protocolo de Internet que describen problemas básicos con una transacción enviada.

<AttributeTable>

- CÓDIGO DE ESTATUS
  - VALO
  - DESCRIPCIÓN
- Ok
  - 200
  - La solicitud ha tenido éxito.
- Solicitud incorrecta
  - 400
  - La solicitud no es válida de alguna manera.
- No encontrado
  - 404
  - El recurso solicitado no existe.
- No implementado
  - 404
  - La solicitud no tiene un tipo de respuesta aceptable.
- No aceptable
  - 406
  - La solicitud no tiene un tipo de respuesta aceptable.
- Error interno del servidor
  - 500
  - Algo salió mal en el final del servidor Horizon.

</AttributeTable>

<ExampleResponse title="Example Response for a “Bad Request” Status Code">

```json
{
  "type": "https://stellar.org/horizon-errors/bad_request",
  "title": "Bad Request",
  "status": 400,
  "detail": "The request you sent was invalid in some way",
  "extras": {
    "invalid_field": "limit",
    "reason": "unparseable value"
  }
}
```

</ExampleResponse>
