Algunos errores sólo ocurren a nivel de Horizon y no son lanzados en Stellar Core. Estos son normalmente problemas con cómo se formó la transacción o un conflicto entre la composición de la transacción y cómo se configura el servidor Horizon.

Stellar utiliza códigos de respuesta HTTP convencionales para indicar el éxito o fallo de una petición API. En general:

- Los códigos de la gama 2xx indican éxito.
- Los códigos en el rango 4xx indican un error que falló dada la información proporcionada
- Los códigos en el rango 5xx indican un error con el servidor Horizon.

Hay dos tipos de Códigos de Estado: [Códigos de Estado](./standard.mdx) y [Códigos de Estado Específicos Horizon-Specificos](./Texton-specific/README.mdx).
