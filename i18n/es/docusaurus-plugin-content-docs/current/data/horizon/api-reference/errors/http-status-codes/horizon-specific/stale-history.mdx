Un servidor Horizon puede ser configurado para rechazar peticiones históricas cuando se sabe que la historia está más desactualizada que el umbral configurado.

En tales casos, el error `stale_history` ocurre y devuelve un [`503` código de error](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/503). Para resolver este error (siempre que sea el operador de la instancia de Horizon), asegúrese de que el sistema de ingestión se está ejecutando correctamente e importando nuevos contadores.

<ExampleResponse title="Example Response for a 'Stale History' Status Code">

```json
{
  "type": "https://stellar. rg/horizonte-errors/stale_history",
  "title": "Histórico DB Is Too Stale",
  "status": 503,
  "detail": "Esta instancia de horizonte está configurada para rechazar peticiones de clientes cuando puede determinar que la base de datos de historia está demasiado rezagada detrás de la instancia conectada de stellar-core. Si utiliza este servidor, asegúrese de que el sistema de ingestión está funcionando correctamente."
}
```

</ExampleResponse>
