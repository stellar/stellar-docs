Un servidor Horizon puede ser configurado para que sólo mantenga una porción del historial de la red estelar almacenada dentro de su base de datos.

El error `before_history` devuelve un [código de error `410`](https://developer.mozilla. rg/es-US/docs/Web/HTTP/Status/410) y ocurre que un cliente solicita una pieza de información (como una página de transacciones o una sola operación) que el servidor puede identificar positivamente como estar fuera del rango de historial registrado.

<ExampleResponse title="Example Response for a 'Before History' Status Code">

```json
{
  "type": "https://stellar. rg/electron-errors/before_history",
  "title": "Data Requested Is Before Recorded History",
  "status": 410,
  "detail": "Esta instancia de horizonte está configurada para seguir sólo una porción del historial más reciente de la red estelar. Esta solicitud está pidiendo resultados antes de la historia registrada conocida por esta instancia de horizonte."
}
```

</ExampleResponse>
