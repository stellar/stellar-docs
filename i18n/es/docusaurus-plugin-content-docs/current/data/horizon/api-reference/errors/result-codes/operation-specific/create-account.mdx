Estos son códigos de resultado que comunican el éxito (200) o fallo (400) específico de la operación `Crear cuenta`, que a menudo falla porque la nueva cuenta no cumple con la reserva mínima.

Aprende más sobre la [operación 'Crear cuenta'](../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#create-account).

<AttributeTable>

- RESULTAR CÓDIGO
  - STELLAR PROTOCOL CODE
  - DESCRIPCIÓN
- OpSuccess
  - CREAR CUENT_SUCCESS
  - El resultado del objeto interno es válido, la operación fue un éxito y se creó un cuento.
- OpMalformado
  - CREAR CUENT_MALFORMED
  - El destino no era válido.
- Optimizado
  - CREATE_CUENT_UNDERFUNDADO
  - La cuenta de origen que realiza el comando no tiene fondos suficientes para dar a la cuenta de destino la reserva mínima necesaria y mantener su propia reserva mínima.
- OpLowReserve
  - CREATE_CUENT_DESCRIPTION
  - La operación crearía una cuenta por debajo de la reserva mínima.
- Ya existe
  - CREATE_ACCOUNT_TITLE
  - La cuenta de destino ya existe.

</AttributeTable>
