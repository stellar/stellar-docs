El error `timeout` devuelve un [código de error `504`](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/504) y ocurre cuando:

- Horizon no ha recibido una confirmación del servidor de Stellar Core de que la transacción que está intentando enviar a la red fue incluida de forma oportuna, o
- Horizon no ha enviado una respuesta a un proxy reverso antes de que transcurra una cantidad de tiempo especificada.

El primer caso puede ocurrir porque no había espacio para su transacción para 3 contadores consecutivos. Esto es porque Stellar Core elimina cada transacción enviada de una cola. Para resolver esto puedes:

- Sigue reenviando la misma transacción (con el mismo número de secuencia) y espera hasta que finalmente se agregue a un nuevo mayor, o
- Aumente la cuota para priorizar la transacción.
