Estos son códigos de resultado que comunican éxito (200) o fallo (400) en el nivel de transacción: números de secuencia incorrectos, saldos insuficientes, tarifas insuficientes, etc.

<AttributeTable>

- RESULTAR CÓDIGO
  - STELLAR PROTOCOL CODE
  - DESCRIPCIÓN
- tx_éxito
  - SUCESS
  - La transacción tuvo éxito.
- tx_fallo
  - txFALLADO
  - Una de las operaciones falló (ninguna fue aplicada).
- tx_too_pronto
  - %1
  - El tiempo de cierre del libro de contabilidad era antes del minTime.
- tarde
  - %1
  - El tiempo de cierre del libro fue después del maxTime.
- operación faltante
  - %1
  - No se especificó ninguna operación
- seq_malo
  - %1
  - el número de secuencia no coincide con la cuenta de origen
- autenticidad incorrecta
  - %1
  - muy pocas firmas válidas / red incorrecta
- saldo insuficiente
  - BALANCE
  - la comisión traería la cuenta debajo de la reserva
- cuenta de fuente
  - no hay una cuenta
  - cuenta de origen no encontrada
- honorarios insuficientes
  - TxINSUFICIENT_FEE
  - comisión es demasiado pequeña
- tx_bad_auth_extra
  - txBAD_AUTH_EXTRA
  - firmas no utilizadas adjuntas a la transacción
- error interno
  - error
  - se ha producido un error desconocido

</AttributeTable>
