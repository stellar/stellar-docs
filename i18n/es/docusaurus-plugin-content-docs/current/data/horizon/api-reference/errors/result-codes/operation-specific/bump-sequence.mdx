Estos son códigos de resultado que comunican el éxito (200) o el fallo (400) específico de la operación `Bump Sequence`.

Aprende más sobre la [operación `Secuencia de volúmenes`](../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#bump-sequence).

<AttributeTable>

- OpSuccess
  - PLAYLIST_BTN
  - El número de secuencia ha sido golpeado.
- seq_malo
  - SEQ
  - El número de secuencia bumpTo especificado no es un número de secuencia válido. Debe estar entre 0 e INT64_MAX (9223372036854775807 o 0x7fffffffffff).

</AttributeTable>
