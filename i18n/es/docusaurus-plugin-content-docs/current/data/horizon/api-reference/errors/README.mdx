Después de procesar una solicitud, Horizon devuelve un éxito o una respuesta de error al cliente. Una respuesta exitosa devolverá un Código de Estado de 200, y una respuesta de error devolverá un Código de Estado en el rango de 4XX - 5XX junto con información adicional sobre por qué la solicitud no pudo completarse correctamente.

Hay dos categorías de errores: [códigos de estado HTTP](./http-status: codes/README.mdx) y [códigos de resultado](./result-codes/README.mdx). Los códigos de resultado sólo siguen un código de estado HTTP fallido (400).
