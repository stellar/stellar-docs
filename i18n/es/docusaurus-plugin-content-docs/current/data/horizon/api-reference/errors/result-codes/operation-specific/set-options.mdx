Estos son códigos de resultado que comunican el éxito (200) o fallo (400) específico de la operación `Set Options`.

Aprende más sobre la [operación 'Establecer opciones'](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#set-options).

<AttributeTable>

- OpSuccess
  - Objeto de objetos
  - Opciones configuradas con éxito.
- OpLowReserve
  - OPTION_BTN
  - Esta cuenta no tiene suficiente XLM para satisfacer el incremento mínimo de reserva XLM causado por la adición de una subentrada y aún así satisfacer sus pasivos de venta XLM. Por cada nuevo firmante añadido a una cuenta, la reserva mínima de XLM que la cuenta debe mantener aumenta.
- firma_too_muchos_firmantes
  - OPTIONS_BTN
  - 20 es el número máximo de firmantes que puede tener una cuenta, y añadir otro firmante excedería eso.
- banderas_bad_
  - PLAYLIST_POPUP_TITLE
  - Los parámetros establecidos y/o borrados no son válidos por sí mismos o en combinación.
- op_invalid_inflación
  - INFLATION
  - La cuenta de destino establecida en el campo de inflación no existe.
- op_cant_change
  - Causa de salida
  - Esta cuenta ya no puede cambiar la opción que quiere cambiar.
- marca desconocida
  - Banda desconocida
  - La cuenta está tratando de establecer una bandera que es desconocida.
- quedar_fuera del rango
  - OPTIONS_THRESHOLD
  - El valor de un peso o umbral de clave no es válido.
- firmante malo
  - OPTIONS_BAD_SIGNER
  - Cualquier firma adicional añadida a la cuenta no puede ser la clave maestra.
- dominio de inicio no válido
  - OPTIONS_INVALID_HOMAIN
  - El dominio del hogar está mal formado.

</AttributeTable>
