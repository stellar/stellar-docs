### Usar el Laboratorio

El método `simulateTransaction` te permite **simular una invocación de contrato inteligente** sin realmente enviarlo a la red. Es una herramienta poderosa para probar y depurar transacciones de forma segura.

Este punto final devuelve los **datos de transacción calculados**, **autorizaciones requeridas** y la **tarifa mínima de recursos**, ayudándote a entender cómo la red procesaría la transacción antes de que la envíes.

👉 [Simular una Transacción en el Laboratorio](https://lab.stellar.org/endpoints/rpc/simulate-transaction?$=network$id=testnet&label=Testnet&horizonUrl=https:////horizon-testnet.stellar.org&rpcUrl=https:////soroban-testnet.stellar.org&passphrase=Test%20SDF%20Network%20/;%20September%202015;;)
