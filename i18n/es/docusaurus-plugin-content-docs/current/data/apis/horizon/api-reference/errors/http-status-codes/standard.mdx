---
title: Códigos de Estado Estándar
sidebar_position: 10
---

Estas respuestas son códigos de protocolo de Internet que describen problemas básicos con una transacción enviada.

<AttributeTable>

- CÓDIGO DE ESTADO
  - VALOR
  - DESCRIPCIÓN
- Aceptar
  - 200
  - La solicitud ha tenido éxito.
- Solicitud Incorrecta
  - 400
  - La solicitud es inválida de alguna manera.
- No Encontrado
  - 404
  - El recurso solicitado no existe.
- No Implementado
  - 404
  - La solicitud no tiene un tipo de contenido de respuesta aceptable.
- No Aceptable
  - 406
  - La solicitud no tiene un tipo de contenido de respuesta aceptable.
- Error Interno del Servidor
  - 500
  - Algo salió mal en el servidor de Horizon.

</AttributeTable>

<ExampleResponse title="Respuesta de Ejemplo para un Código de Estado de Solicitud Incorrecta">

```json
{
  "type": "https://stellar.org/horizon-errors/bad_request",
  "title": "Bad Request",
  "status": 400,
  "detail": "The request you sent was invalid in some way",
  "extras": {
    "invalid_field": "limit",
    "reason": "unparseable value"
  }
}
```

</ExampleResponse>
