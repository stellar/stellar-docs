Estos son códigos de resultado que comunican éxito (200) o fracaso (400) específicos para la operación de `Gestionar oferta de compra`.

Obtén más información sobre la [operación `Gestionar oferta de compra`](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#manage-buy-offer).

<AttributeTable>

- Éxito de operación
  - MANAGE_BUY_OFFER_SUCCESS
  - La oferta se realizó con éxito.
- Operación malformada
  - MANAGE_BUY_OFFER_MALFORMED
  - La entrada es incorrecta y resultaría en una oferta inválida.
- op_sell_no_trust
  - MANAGE_BUY_OFFER_SELL_NO_TRUST
  - La cuenta que crea la oferta no tiene una línea de confianza para el activo que está vendiendo.
- op_buy_no_trust
  - MANAGE_BUY_OFFER_BUY_NO_TRUST
  - La cuenta que crea la oferta no tiene una línea de confianza para el activo que está comprando.
- no autorizado para vender
  - MANAGE_BUY_OFFER_SELL_NOT_AUTHORIZED
  - La cuenta que crea la oferta no está autorizada para vender este activo.
- no autorizado para comprar
  - MANAGE_BUY_OFFER_BUY_NOT_AUTHORIZED
  - La cuenta que crea la oferta no está autorizada para comprar este activo.
- Operación de línea completa
  - MANAGE_BUY_OFFER_LINE_FULL
  - La cuenta que crea la oferta no tiene límites suficientes para recibir compras y aún satisfacer sus obligaciones de compra.
- Operación subfundada
  - MANAGE_BUY_OFFER_UNDERFUNDED
  - La cuenta que crea la oferta no tiene límites suficientes para enviar ventas y aún satisfacer sus obligaciones de venta. Ten en cuenta que si vendes XLM, entonces la cuenta debe mantener adicionalmente su reserva mínima de XLM, que se calcula asumiendo que esta oferta no se ejecutará completamente de inmediato.
- op_cross_self
  - MANAGE_BUY_OFFER_CROSS_SELF
  - La cuenta tiene una oferta opuesta de precio igual o menor activa, por lo que la cuenta que crea esta oferta se cruzaría a sí misma inmediatamente.
- op_sell_no_issuer
  - MANAGE_BUY_OFFER_SELL_NO_ISSUER
  - El emisor del activo que se vende no existe.
- buy_no_issuer
  - MANAGE_BUY_OFFER_BUY_NO_ISSUER
  - El emisor del activo que se compra no existe.
- op_offer_not_found
  - MANAGE_BUY_OFFER_NOT_FOUND
  - No se puede encontrar una oferta con ese ID de oferta.
- Operación de reserva baja
  - MANAGE_BUY_OFFER_LOW_RESERVE
  - La cuenta que crea esta oferta no tiene suficientes XLM para satisfacer el aumento mínimo de reserva de XLM causado por la adición de una subentrada y aún satisfacer sus obligaciones de venta de XLM. Por cada oferta que una cuenta crea, la cantidad mínima de XLM que esa cuenta debe mantener aumentará.

</AttributeTable>
