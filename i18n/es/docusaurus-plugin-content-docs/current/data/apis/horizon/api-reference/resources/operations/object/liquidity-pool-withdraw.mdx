Retira reservas de activos de un fondo de liquidez al canjear acciones del fondo.

Consulta la operación [`Retirada de Fondos de Liquidez`](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#liquidity-pool-withdraw) para parámetros, errores, etc.

<AttributeTable>

- ATRIBUTE
  - TIPO DE DATO
  - DESCRIPCIÓN
- liquidity_pool_id
  - cadena
  - El fondo de liquidez asociado a esta retirada de fondos
- reserves_min
  - arreglo
  - Un arreglo de objetos que corresponden a la cantidad mínima de cada reserva que debería haber sido retirada
    - activo
      - cadena
      - El activo en forma canónica (`Código:Emisor`)
    - cantidad
      - cadena
      - Un valor de punto flotante codificado como cadena
- acciones
  - cadena
  - El número de acciones que fueron canjeadas por esta operación de retirada de fondos
- reserves_received
  - arreglo
  - Un arreglo de objetos que representan cuánto de cada reserva terminó siendo efectivamente retirada del fondo
    - activo
      - cadena
      - El activo en forma canónica (`Código:Emisor`)
    - cantidad
      - cadena
      - Un valor de punto flotante codificado como cadena

</AttributeTable>
