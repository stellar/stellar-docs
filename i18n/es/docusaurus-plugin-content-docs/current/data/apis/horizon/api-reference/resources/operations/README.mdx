Las operaciones son objetos que representan un cambio deseado en el ledger: pagos, ofertas para intercambiar divisas, cambios realizados en las opciones de la cuenta, etc. Las operaciones se envían a la red Stellar agrupadas en una Transacción.

Cada una de las operaciones de Stellar tiene un objeto de operación único.
