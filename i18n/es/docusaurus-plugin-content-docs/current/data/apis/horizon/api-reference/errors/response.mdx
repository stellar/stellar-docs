Cuando ocurre un error, Horizon responde con un documento JSON que contiene los atributos a continuación.

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- tipo
  - URL
  - El tipo de código de estado devuelto.
- título
  - cadena
  - Un breve título que describe el código de estado, que se puede usar para buscar más información sobre un error.
- estado
  - número
  - Un breve título que describe el código de estado, que se puede usar para buscar más información sobre un error.
- detalle
  - cadena
  - Un breve título que describe el código de estado, que se puede usar para buscar más información sobre un error.
- extras
  - mapa
  - Si el código de estado es `Transacción fallida`, este campo de extras muestra el código de resultado devuelto por Stellar Core que describe por qué falló la transacción.
    - envelope_xdr
      - cadena
      - Una representación codificada en base64 del TransactionEnvelope XDR cuya falla provocó esta respuesta.
    - result_xdr
      - cadena
      - Una representación codificada en base64 del TransactionResult XDR devuelto por stellar-core al enviar esta transacción.
    - result_codes.transaction
      - cadena
      - El código de resultado de la transacción devuelto por Stellar Core, que se puede usar para buscar más información sobre un error en la documentación.
    - result_codes.operations
      - array
      - Un array de códigos de resultado de la operación devueltos por Stellar Core, que se pueden usar para buscar más información sobre un error en la documentación.

</AttributeTable>
