---
title: Transacción Malformada
sidebar_position: 20
---

El error `transaction_malformed` devuelve un [código de error `400`](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/400) y ocurre cuando un cliente envía una transacción malformada.

Hay muchas maneras en las que una transacción podría estar malformada, incluyendo:

1. Enviaste una cadena vacía.
2. Tu cadena codificada en base64 no es válida.
3. Tu estructura [XDR](../../../structure/xdr.mdx) no es válida.
4. Tienes bytes sobrantes en tu estructura [XDR](../../../structure/xdr.mdx).
