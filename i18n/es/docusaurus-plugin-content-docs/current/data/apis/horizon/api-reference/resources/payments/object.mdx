---
title: El Objeto de Pago
order: 20
---

Envía una cantidad en un activo específico a una cuenta de destino.

Consulta los [`Payment` errores](../../errors/result-codes/operation-specific/payment.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATOS
  - DESCRIPCIÓN
- tipo_de_activo
  - cadena
  - El tipo de activo que se envía. Puede ser `nativo`, `credit_alphanum4` o `credit_alphanum12`.
- código_de_activo
  - cadena
  - El código del activo que se envía. Aparece si el `tipo_de_activo` no es `nativo`.
- emisor_de_activo
  - cadena
  - La dirección Stellar del emisor del activo enviado. Aparece si el `tipo_de_activo` no es `nativo`.
- de
  - cadena
  - La clave pública del remitente del pago.
- a
  - cadena
  - La clave pública del destinatario del pago.
- cantidad
  - cadena
  - Cantidad enviada.

</AttributeTable>
