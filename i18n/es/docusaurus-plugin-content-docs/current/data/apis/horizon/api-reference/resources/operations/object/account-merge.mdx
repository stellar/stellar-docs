Elimina la cuenta origen del Stellar y transfiere los lumens de la cuenta origen a otra cuenta.

Consulta los [errores de `Fusión de Cuenta`](../../../errors/result-codes/operation-specific/account-merge.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- cuenta
  - cadena
  - La dirección Stellar que se está eliminando.
- en
  - cadena
  - La dirección Stellar que recibe los lumens de la cuenta eliminada.

</AttributeTable>
