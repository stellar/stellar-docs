Actualiza el indicador “autorizado” de una línea de confianza existente. Esto debe ser llamado por el emisor del activo.

Consulta los [errores de `Permitir Confianza`](../../../errors/result-codes/operation-specific/allow-trust.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- tipo_de_activo
  - cadena
  - El tipo de activo. Puede ser `nativo`, `credit_alphanum4`, o `credit_alphanum12`.
- código_de_activo
  - cadena
  - La dirección Stellar del activo.
- emisor_de_activo
  - cadena
  - El código del activo.
- autorizar
  - int
  - Indicador que indica si la línea de confianza está autorizada. 0 si la cuenta no está autorizada para transaccionar con el activo de ninguna manera. 1 si la cuenta está autorizada para transaccionar con el activo. 2 si la cuenta está autorizada para mantener órdenes, pero no para realizar otras transacciones.
- fiduciario
  - cadena
  - La cuenta emisor, o cuenta origen en este caso.
- fiduciario
  - cadena
  - La cuenta fideicomitente, o la cuenta que está siendo autorizada o no autorizada.

</AttributeTable>
