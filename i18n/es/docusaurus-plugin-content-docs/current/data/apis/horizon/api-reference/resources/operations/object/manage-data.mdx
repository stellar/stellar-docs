---
title: Gestionar Objeto de Datos
order: 120
---

Establecer, modificar o eliminar una entrada de datos (par de nombre/valor) para una cuenta.

Consulta los [errores de `Gestionar Datos`](../../../errors/result-codes/operation-specific/manage-data.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- nombre
  - cadena
  - La clave para esta entrada de datos. Puede tener hasta 64 bytes de longitud. Si este es un nuevo `Nombre`, se agregará el par nombre/valor dado a la cuenta. Si este `Nombre` ya está presente, entonces se modificará el valor asociado.
- valor
  - cadena
  - Si está presente, entonces este valor se establecerá en el DataEntry. Puede tener hasta 64 bytes de longitud. Si no está presente, entonces se eliminará el `Nombre` existente.

</AttributeTable>
