Cuando Horizon devuelve información sobre un libro de órdenes, utiliza el siguiente formato:

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- ofertas
  - objeto
  - Los precios y las cantidades del lado comprador del par de activos.
    - precio_r
      - objeto
      - Una representación precisa del precio de la oferta del par de activos.
        - n
          - número
          - El numerador.
        - d
          - número
          - El denominador.
    - precio
      - cadena
      - El precio de la oferta del activo base expresado en el activo contra. Un número que representa la forma decimal de `precio_r`.
    - cantidad
      - cadena
      - La cantidad del activo contra que la cuenta que realiza esta oferta está dispuesta a comprar a este precio.
- ofertas
  - objeto
  - Los precios y las cantidades del lado vendedor del par de activos.
    - precio_r
      - objeto
      - Una representación precisa del precio de la oferta del par de activos.
        - n
          - número
          - El numerador.
        - d
          - número
          - El denominador.
    - precio
      - cadena
      - El precio de la oferta del activo base expresado en el activo contra. Un número que representa la forma decimal de `precio_r`.
    - cantidad
      - cadena
      - La cantidad del activo contra que la cuenta que realiza esta oferta está dispuesta a vender a este precio.
- base
  - objeto
  - Detalles sobre el activo base.
    - tipo_de_activo
      - cadena
      - El tipo para el activo base. Ya sea `nativo`, `credito_alphanum4`, o `credito_alphanum12`.
    - código_de_activo
      - cadena
      - El código para el activo base.
    - emisor_de_activo
      - cadena
      - La dirección Stellar del emisor del activo base.
- contra
  - objeto
  - Detalles sobre el activo contra.
    - tipo_de_activo
      - cadena
      - El tipo para el activo contra. Ya sea `nativo`, `credito_alphanum4`, o `credito_alphanum12`.
    - código_de_activo
      - cadena
      - El código para el activo contra.
    - emisor_de_activo
      - cadena
      - La dirección Stellar del emisor del activo contra.

</AttributeTable>
