Cuando una oferta se cumple total o parcialmente, ocurre un comercio. Los comercios también pueden ser causados por pagos de ruta exitosos, ya que los pagos de ruta implican cumplir ofertas.

Un comercio ocurre entre dos partes—`base` y `contraparte`. Cuál es cuál es arbitrario o determinado por la consulta que llama.

Aprende más sobre [operaciones](../../../../../../learn/glossary.mdx#decentralized-exchange) con intercambio descentralizado.
