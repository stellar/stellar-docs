Los Códigos de Resultado describen por qué una transacción u operación falló en Stellar Core y se comunican en el campo “extras” de una respuesta de Horizon cuando se devuelve el Código de Estado “Transacción Fallida”.

En el campo “extras”, los errores devueltos se denominan “Códigos de Resultado” y son la abstracción de Horizon de los “Códigos del Protocolo Stellar”, que son códigos más específicos disponibles en el XDR. Los Códigos de Resultado son la forma en que Horizon normaliza los Códigos del Protocolo Stellar.

Hay tres tipos de Códigos de Resultado: [Códigos de Resultado de Transacción](./transactions.mdx), [Códigos de Resultado de Operación](./operations.mdx) y [Códigos de Resultado Específicos de Operación](./operation-specific/README.mdx).
