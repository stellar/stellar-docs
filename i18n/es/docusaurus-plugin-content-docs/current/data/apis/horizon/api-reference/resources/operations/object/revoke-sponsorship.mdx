Revocar el patrocinio de una entrada del ledger.

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATOS
  - DESCRIPCIÓN
- account_id
  - string (opcional)
  - El id de la cuenta que ya no está patrocinada.
- claimable_balance_id
  - string (opcional)
  - El id del saldo reclamable que ya no está patrocinado.
- data_account_id
  - string (opcional)
  - El id de la cuenta cuya entrada de datos ya no está patrocinada.
- data_name
  - string (opcional)
  - El nombre de la entrada de datos que ya no está patrocinada.
- offer_id
  - string (opcional)
  - El id de la oferta que ya no está patrocinada.
- trustline_account_id
  - string (opcional)
  - El id de la cuenta cuya trustline ya no está patrocinada.
- trustline_asset
  - string (opcional)
  - El activo de la trustline que ya no está patrocinado.
- signer_account_id
  - string (opcional)
  - El id de la cuenta del firmante que ya no está patrocinado.
- signer_key
  - string (opcional)
  - El tipo de firmante que ya no está patrocinado.

</AttributeTable>
