Crea, actualiza o elimina una oferta de compra para intercambiar activos. Una oferta de compra especifica una cierta cantidad del activo comprado que debe ser vendida a cambio de la cantidad mínima del activo vendido.

Consulta los [`errores de Gestionar Oferta de Compra`](../../../errors/result-codes/operation-specific/manage-buy-offer.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- cantidad
  - cadena
  - La cantidad de `buying_asset` que la cuenta que realiza esta oferta está dispuesta a comprar.
- precio
  - cadena
  - Cuántas unidades de `buying_asset` se necesitan para obtener 1 unidad de `selling_asset`. Un número que representa la forma decimal de `price_r`.
- precio_r
  - objeto
  - Una representación precisa del precio de compra y venta de los activos en oferta.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- tipo_de_activo_compra
  - cadena
  - El tipo del activo comprado. Puede ser `native`, `credit_alphanum4` o `credit_alphanum12`.
- emisora_de_activo_compra
  - cadena
  - La dirección Stellar del emisor del activo comprado. Aparece si el `buying_asset_type` no es `native`.
- código_de_activo_compra
  - cadena
  - El código del activo comprado. Aparece si el `buying_asset_type` no es `native`.
- tipo_de_activo_venta
  - cadena
  - El tipo del activo vendido. Puede ser `native`, `credit_alphanum4` o `credit_alphanum12`.
- emisora_de_activo_venta
  - cadena
  - La dirección Stellar del emisor del activo vendido. Aparece si el `selling_asset_type` no es `native`.
- código_de_activo_venta
  - cadena
  - El código para el activo vendido. Aparece si el `selling_asset_type` no es `native`.
- id_de_oferta
  - cadena
  - Un identificador único para esta oferta.

</AttributeTable>
