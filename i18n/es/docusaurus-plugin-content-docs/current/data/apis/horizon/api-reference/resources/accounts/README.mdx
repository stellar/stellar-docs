Los usuarios interactúan con la red Stellar a través de cuentas. Todo lo demás en el ledger—activos, ofertas, líneas de confianza, etc.—pertenece a las cuentas, y las cuentas deben autorizar todos los cambios en el ledger a través de transacciones firmadas.

El [ledger más reciente](../../structure/consistency.mdx) conocido por Horizon se incluye como un encabezado HTTP en la respuesta.

Aprende más sobre [cuentas](../../../../../../learn/glossary.mdx#account).
