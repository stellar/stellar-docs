Cuando Horizon devuelve información sobre una agregación de operaciones, utiliza el siguiente formato:

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- marca de tiempo
  - cadena
  - Hora de inicio para esta agregación de operaciones. Representado como milisegundos desde la época.
- contador_de_operaciones
  - entero
  - Número total de operaciones agregadas.
- volumen_base
  - cadena
  - Volumen total del activo base.
- volumen_contraparte
  - cadena
  - Volumen total del activo contra.
- promedio
  - cadena
  - Precio promedio ponderado del activo contra en términos del activo base.
- alto
  - cadena
  - El precio más alto durante este período de tiempo.
- alto_r
  - objeto
  - El precio más alto durante este período de tiempo como número racional.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- bajo
  - cadena
  - El precio más bajo durante este período de tiempo.
- bajo_r
  - objeto
  - El precio más bajo durante este período de tiempo como número racional.
    - n
      - número
      - El numerador.
    - p
      - número
      - El denominador.
- abrir
  - cadena
  - El precio como se ve en la primera operación agregada.
- abrir_r
  - objeto
  - El precio como se ve en la primera operación agregada como número racional.
    - n
      - número
      - El numerador.
    - p
      - número
      - El denominador.
- cerrar
  - cadena
  - El precio como se ve en la última operación agregada.
- cerrar_r
  - objeto
  - El precio como se ve en la última operación agregada como número racional.
    - n
      - número
      - El numerador.
    - p
      - número
      - El denominador.

</AttributeTable>
