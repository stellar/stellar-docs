Los Fondos de Liquidez proporcionan una forma simple y no interactiva de intercambiar grandes cantidades de capital y permiten altos volúmenes de negociación.

El [ledger más reciente](../../structure/consistency.mdx) conocido por Horizon se incluye como un encabezado HTTP en la respuesta.
