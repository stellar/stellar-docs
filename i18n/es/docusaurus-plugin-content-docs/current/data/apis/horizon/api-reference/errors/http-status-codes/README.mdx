Algunos errores solo ocurren a nivel de Horizon y no se lanzan en Stellar Core. Estos suelen ser incidencias con la forma en que se formó la transacción o un conflicto entre la composición de la transacción y cómo está configurado el servidor de Horizon.

Stellar utiliza códigos de respuesta HTTP convencionales para indicar el éxito o fracaso de una solicitud de API. En general:

- Los códigos en el rango 2xx indican éxito.
- Los códigos en el rango 4xx indican un error que falló dada la información proporcionada
- Los códigos en el rango 5xx indican un error con el servidor de Horizon.

Hay dos tipos de Códigos de Estado: [Códigos de Estado Estándar](./standard.mdx) y [Códigos de Estado Específicos de Horizon](./horizon-specific/README.mdx).
