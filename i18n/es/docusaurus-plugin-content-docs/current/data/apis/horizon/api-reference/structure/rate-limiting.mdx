---
title: Limitación de tasa
sidebar_position: 40
---

Los límites de tasa de Horizon se aplican por dirección IP. Se puede configurar a través de la opción `PER_HOUR_RATE_LIMIT` y, por defecto, se establece en 3600 solicitudes por hora. Se recomienda que los operadores de Horizon ajusten este valor según sus capacidades infraestructurales individuales y necesidades de uso.

Este límite se puede desactivar completamente estableciendo el parámetro en `0`.

Cuando un cliente excede este límite, Horizon devolverá un error `429 Too Many Requests`.

Mientras se transmite, cada actualización del flujo cuenta como una solicitud y contra el límite de tasa asignado a un cliente.
