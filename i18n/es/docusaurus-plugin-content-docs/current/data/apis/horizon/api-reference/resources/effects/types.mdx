Hay ocho grupos de tipos de efecto. Cada tipo de efecto tiene su propio conjunto de atributos.

- Cuenta eliminada

- Cuenta acreditada
  - saltar

- Cuenta debitada
  - saltar

- Umbrales de Cuenta Actualizados
  - saltar

- Dominio de Casa de Cuenta Actualizado
  - saltar

- Banderas de Cuenta Actualizadas
  - saltar

- Destino de Inflación de Cuenta Actualizado

</AttributeTable>
