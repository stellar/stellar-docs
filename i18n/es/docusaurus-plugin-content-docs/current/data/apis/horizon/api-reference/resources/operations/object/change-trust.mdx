Crea, actualiza o elimina una línea de confianza de la cuenta de origen a un activo emitido de otra cuenta.

Consulta los [`errores de Cambiar Confianza`](../../../errors/result-codes/operation-specific/change-trust.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- tipo_de_activo
  - cadena
  - El tipo de activo que se está confiando, uno de `nativo`, `credit_alphanum4`, `credit_alphanum12` o `fondos_de_liquidez`.
- código_de_activo
  - cadena
  - La dirección Stellar del activo que se está confiando. (Solo presente para activos de crédito.)
- emisor_de_activo
  - cadena
  - El código del activo que se está confiando. (Solo presente para activos de crédito.)
- límite
  - cadena
  - Limita la cantidad de un activo que puede poseer la cuenta de origen.
- fiduciario
  - cadena
  - La cuenta emisora. (Solo presente para activos de crédito.)
- fiduciante
  - cadena
  - La cuenta de origen.
- id_de_fondos_de_liquidez
  - cadena
  - El fondo de liquidez cuya línea de confianza se está modificando. (Solo presente para `asset_type == liquidity_pool_shares`.)

</AttributeTable>
