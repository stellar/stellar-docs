Cada una de las operaciones de Stellar tiene formas de respuesta únicas. A continuación se presentan los atributos que son comunes en los objetos de operación individuales.

Consulta los [errores genéricos de Operación](../../../errors/result-codes/operations.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- id
  - número
  - El número de identificación de la operación.
- token_de_paginación
  - cadena
  - Un valor de cursor para usar en [paginación](../../../structure/pagination/README.mdx).
- tipo_i
  - número
  - Un número que indica el tipo de operación.
- tipo
  - cadena
  - El nombre del tipo de operación.
- hash_de_transacción
  - cadena
  - Un identificador único para la transacción a la que pertenece esta operación.
- transacción_exitosa
  - booleano
  - Indica si esta operación fue parte de una transacción exitosa.
- cuenta_origen
  - cadena
  - La cuenta que origina la operación.
- creado_en
  - cadena
  - La fecha en que se creó esta operación.

</AttributeTable>
