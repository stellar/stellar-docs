Estos son los Códigos de Resultado que comunican éxito (200) o fracaso (400) a nivel de operación: sin cuenta de origen, demasiadas subentradas, etc.

<AttributeTable>

- CÓDIGO DE RESULTADO
  - CÓDIGO DEL PROTOCOLO STELLAR
  - DESCRIPCIÓN
- op_inner
  - opINNER
  - El resultado del objeto interno es válido y la operación fue un éxito.
- op_bad_auth
  - opBAD_AUTH
  - Hay muy pocas firmas válidas, o la transacción fue enviada a la red equivocada.
- op_no_source_account
  - opNO_ACCOUNT
  - No se encontró la cuenta de origen.
- op_not_supported
  - opNOT_SUPPORTED
  - La operación no es admitida en este momento.
- op_too_many_subentries
  - opTOO_MANY_SUBENTRIES
  - Número máximo de subentradas (1000) ya alcanzado
- op_exceeded_work_limit
  - opEXCEEDED_WORK_LIMIT
  - La operación hizo demasiado trabajo

</AttributeTable>
