Crea una oferta que no consumirá una contraoferta que iguale exactamente esta oferta. Esto es útil para ofertas destinadas a ser intercambios 1:1 para pagos de ruta. Utiliza Gestionar Oferta de Venta para gestionar esta oferta después de usar esta operación para crearla.

Consulta los [`errores de Crear Oferta de Venta Pasiva`](../../../errors/result-codes/operation-specific/create-passive-sell-offer.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- cantidad
  - cadena
  - La cantidad de `selling_asset` que la cuenta que hace esta oferta está dispuesta a vender.
- precio
  - cadena
  - Cuántas unidades de `selling_asset` se necesitan para obtener 1 unidad de `buying_asset`. Un número que representa la forma decimal de `price_r`.
- precio_r
  - objeto
  - Una representación precisa del precio de compra y venta de los activos en oferta.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- tipo_de_activo_compra
  - cadena
  - El tipo para el activo de compra. Puede ser `native`, `credit_alphanum4` o `credit_alphanum12`.
- emisor_de_activo_compra
  - cadena
  - La dirección Stellar del emisor del activo de compra. Aparece si el `tipo_de_activo_compra` no es `native`.
- código_de_activo_compra
  - cadena
  - El código para el activo de compra. Aparece si el `tipo_de_activo_compra` no es `native`.
- tipo_de_activo_venta
  - cadena
  - El tipo para el activo de venta. Puede ser `native`, `credit_alphanum4` o `credit_alphanum12`.
- emisor_de_activo_venta
  - cadena
  - La dirección Stellar del emisor del activo de venta. Aparece si el `tipo_de_activo_venta` no es `native`.
- código_de_activo_venta
  - cadena
  - El código para el activo de venta. Aparece si el `tipo_de_activo_venta` no es `native`.
- id_de_oferta
  - cadena
  - Un identificador único para esta oferta.

</AttributeTable>
