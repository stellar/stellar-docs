---
title: Configurar el objeto de opciones
order: 80
---

Establece las banderas de una cuenta, el destino de inflación, los firmantes y el dominio principal.

Consulta los [errores de `Configurar opciones`](../../../errors/result-codes/operation-specific/set-options.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- clave_firmante
  - cadena
  - La clave pública del nuevo firmante.
- peso_firmante
  - número
  - El peso del nuevo firmante. Puede ir de `1` a `255`.
- peso_clave_maestra
  - número
  - El peso de la clave maestra. Puede ir de `1` a `255`.
- umbral_bajo
  - número
  - El peso total para el umbral bajo.
- umbral_medio
  - número
  - El peso total para el umbral medio.
- umbral_alto
  - número
  - El peso total para el umbral alto.
- dominio_principal
  - cadena
  - El dominio principal utilizado para el descubrimiento del archivo stellar.toml.
- establecer_bandas
  - arreglo
  - El arreglo de valores numéricos de las banderas que se han establecido en esta operación. Las opciones incluyen `1` para `AUTH_REQUIRED_FLAG`, `2` para `AUTH_REVOCABLE_FLAG` y `4` para `AUTH_IMMUTABLE_FLAG`.
- establecer_bandas_s
  - arreglo
  - El arreglo de valores de cadena de las banderas que se han establecido en esta operación. Las opciones incluyen `AUTH_REQUIRED_FLAG`, `AUTH_REVOCABLE_FLAG` y `AUTH_IMMUTABLE_FLAG`.
- borrar_bandas
  - arreglo
  - El arreglo de valores numéricos de las banderas que se han eliminado en esta operación. Las opciones incluyen `1` para `AUTH_REQUIRED_FLAG`, `2` para `AUTH_REVOCABLE_FLAG` y `4` para `AUTH_IMMUTABLE_FLAG`.
- borrar_bandas_s
  - arreglo
  - El arreglo de valores de cadena de las banderas que se han eliminado en esta operación. Las opciones incluyen `AUTH_REQUIRED_FLAG`, `AUTH_REVOCABLE_FLAG` y `AUTH_IMMUTABLE_FLAG`.

</AttributeTable>
