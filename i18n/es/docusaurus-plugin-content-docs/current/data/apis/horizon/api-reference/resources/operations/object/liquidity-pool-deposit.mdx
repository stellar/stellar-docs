Depositar reservas de activos en un fondo de liquidez.

Consulta la [operación `Depósito en Fondo de Liquidez`](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#liquidity-pool-deposit) para parámetros, errores, etc.

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- liquidity_pool_id
  - cadena
  - El fondo de liquidez asociado con este depósito
- reserves_max
  - arreglo
  - Un arreglo de objetos que corresponden a la cantidad máxima de cada reserva que podría haberse depositado
    - activo
      - cadena
      - El activo en forma canónica (`Código:Emisor`)
    - cantidad
      - cadena
      - Un valor de punto flotante codificado como una cadena
- min_price
  - cadena
  - Un valor de punto flotante codificado como una cadena que indica la tasa de cambio mínima para esta operación de depósito
- min_price_r
  - objeto
  - Una representación precisa (fraccional) del precio de compra y venta de los activos ofrecidos.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- max_price
  - cadena
  - Un valor de punto flotante codificado como una cadena que indica la tasa de cambio máxima para esta operación de depósito
- max_price_r
  - objeto
  - Una representación precisa (fraccional) del precio de compra y venta de los activos ofrecidos.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- reserves_deposited
  - arreglo
  - Un arreglo de objetos que representan cuánto de cada reserva terminó siendo realmente depositado en el fondo
    - activo
      - cadena
      - El activo en forma canónica (`Código:Emisor`)
    - cantidad
      - cadena
      - Un valor de punto flotante codificado como una cadena
- shares_received
  - cadena
  - Un valor de punto flotante codificado como una cadena que representa el número de acciones del fondo recibidas por este depósito

</AttributeTable>
