Estos son códigos de resultado que comunican éxito (200) o fracaso (400) específicos de la operación de `Cambio de Confianza`.

Aprende más sobre la [operación de `Cambio de Confianza`](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#change-trust)

<AttributeTable>

- Éxito de Operación
  - CAMBIO_DE_CONFIANZA_EXITO
  - La confianza ha sido cambiada exitosamente
- Operación Malformada
  - CHANGE_TRUST_MALFORMED
  - La entrada a esta operación es inválida.
- Operación Sin Emisor
  - CHANGE_TRUST_NO_ISSUER
  - No se puede encontrar el emisor del activo.
- operación_límite_inválido
  - CHANGE_TRUST_INVALID_LIMIT
  - El límite no es suficiente para mantener el saldo actual de la línea de confianza y aún así satisfacer sus obligaciones de compra.
- Operación de Reserva Baja
  - CHANGE_TRUST_LOW_RESERVE
  - Esta cuenta no tiene suficiente XLM para satisfacer el aumento mínimo de reserva de XLM causado por agregar una subentrada y aún satisfacer sus obligaciones de venta de XLM. Por cada nueva línea de confianza agregada a una cuenta, la reserva mínima de XLM que esa cuenta debe mantener aumenta.
- operación_autoconfianza_no_permitida
  - CHANGE_TRUST_SELF_NOT_ALLOWED
  - La cuenta fuente intentó crear una línea de confianza para sí misma, lo cual no está permitido.

</AttributeTable>
