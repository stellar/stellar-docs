---
title: Códigos de Resultado de Transacción
sidebar_position: 10
---

Estos son los Códigos de Resultado que comunican éxito (200) o fracaso (400) a nivel de transacción: números de secuencia erróneos, saldos insuficientes, tarifas insuficientes, etc.

<AttributeTable>

- CÓDIGO DE RESULTADO
  - CÓDIGO DEL PROTOCOLO STELLAR
  - DESCRIPCIÓN
- tx_success
  - txSUCCESS
  - La transacción se completó con éxito.
- tx_failed
  - txFAILED
  - Una de las operaciones falló (ninguna fue aplicada).
- tx_too_early
  - txTOO_EARLY
  - El closeTime del ledger fue antes del minTime.
- tx_too_late
  - txTOO_LATE
  - El closeTime del ledger fue después del maxTime.
- tx_missing_operation
  - txMISSING_OPERATION
  - No se especificó ninguna operación
- tx_bad_seq
  - txBAD_SEQ
  - el número de secuencia no iguala a la cuenta de origen
- tx_bad_auth
  - txBAD_AUTH
  - muy pocas firmas válidas / red incorrecta
- tx_insufficient_balance
  - txINSUFFICIENT_BALANCE
  - la tarifa dejaría la cuenta por debajo de la reserva
- tx_no_source_account
  - txNO_ACCOUNT
  - no se encontró la cuenta de origen
- tx_insufficient_fee
  - txINSUFFICIENT_FEE
  - la tarifa es demasiado baja
- tx_bad_auth_extra
  - txBAD_AUTH_EXTRA
  - firmas no utilizadas adjuntas a la transacción
- tx_internal_error
  - txINTERNAL_ERROR
  - se produjo un error desconocido

</AttributeTable>
