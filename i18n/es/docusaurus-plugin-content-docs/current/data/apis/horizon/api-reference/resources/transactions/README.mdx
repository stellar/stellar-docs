:::info

El meta de transacción `result_meta_xdr` será eliminado de la API Horizon alojada por SDF (`horizon.stellar.org`) en el tercer trimestre de 2024. En lugar de usar Horizon para acceder a los metadatos de las transacciones, los desarrolladores pueden acceder a ellos mediante el punto final [`getTransactions`](../../../../rpc/api-reference/methods/getTransactions.mdx) en el RPC de Stellar.

:::

Las transacciones son comandos que modifican el estado del ledger y consisten en una o más operaciones.
