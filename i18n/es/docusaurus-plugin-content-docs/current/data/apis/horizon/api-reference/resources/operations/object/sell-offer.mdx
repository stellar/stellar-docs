---
title: Gestionar objeto de oferta de venta
order: 50
---

Crea, actualiza o elimina una oferta de venta para intercambiar activos. Una oferta de venta especifica una cierta cantidad del activo en venta que debe ser vendida a cambio de la máxima cantidad del activo en compra.

Consulta los [`errores de gestionar oferta de venta`](../../../errors/result-codes/operation-specific/manage-sell-offer.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- cantidad
  - cadena
  - La cantidad de `selling_asset` que la cuenta que hace esta oferta está dispuesta a vender.
- precio
  - cadena
  - Cuántas unidades de `selling_asset` se necesitan para obtener 1 unidad de `buying_asset`. Un número que representa la forma decimal de `price_r`.
- precio_r
  - objeto
  - Una representación precisa del precio de compra y venta de los activos en oferta.
    - n
      - número
      - El numerador.
    - d
      - número
      - El denominador.
- tipo_de_activo_compra
  - cadena
  - El tipo del activo en compra. Puede ser `native`, `credit_alphanum4` o `credit_alphanum12`.
- emisor_activo_compra
  - cadena
  - La dirección Stellar del emisor del activo en compra. Aparece si el `buying_asset_type` no es `native`.
- código_activo_compra
  - cadena
  - El código del activo en compra. Aparece si el `buying_asset_type` no es `native`.
- tipo_de_activo_venta
  - cadena
  - El tipo del activo en venta. Puede ser `native`, `credit_alphanum4` o `credit_alphanum12`.
- emisor_activo_venta
  - cadena
  - La dirección Stellar del emisor del activo en venta. Aparece si el `selling_asset_type` no es `native`.
- código_activo_venta
  - cadena
  - El código del activo en venta. Aparece si el `selling_asset_type` no es `native`.
- id_de_oferta
  - cadena
  - Un identificador único para esta oferta.

</AttributeTable>
