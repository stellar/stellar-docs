Después de procesar una solicitud, Horizon devuelve una respuesta de éxito o de error al cliente. Una respuesta de éxito devolverá un código de estado de 200, y una respuesta de error devolverá un código de estado en el rango de 4XX - 5XX junto con información adicional sobre por qué la solicitud no pudo completarse con éxito.

Hay dos categorías de errores: [Códigos de Estado HTTP](./http-status-codes/README.mdx) y [Códigos de Resultado](./result-codes/README.mdx). Los Códigos de Resultado solo siguen un Código de Estado HTTP de Transacción Fallida (400).
