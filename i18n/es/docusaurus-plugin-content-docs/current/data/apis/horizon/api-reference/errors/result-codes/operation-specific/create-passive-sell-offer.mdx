Estos son códigos de resultado que comunican éxito (200) o fracaso (400) específicos de la operación `Crear Oferta de Venta Pasiva`.

Conoce más sobre la [operación `Crear Oferta de Venta Pasiva`](../../../../../../../learn/fundamentals/transactions/list-of-operations.mdx#create-passive-sell-offer).

<AttributeTable>

- OpSuccess
  - CREAR_OFERTA_DE_VENTA_PASSIVA_EXITO
  - La oferta de venta pasiva se ha colocado con éxito.
- OpMalformed
  - MANAGE_SELL_OFFER_MALFORMED
  - La entrada es incorrecta y resultaría en una oferta inválida.
- op_sell_no_trust
  - MANAGE_SELL_OFFER_SELL_NO_TRUST
  - La cuenta que crea la oferta no tiene un trustline para el activo que está vendiendo.
- op_buy_no_trust
  - MANAGE_SELL_OFFER_BUY_NO_TRUST
  - La cuenta que crea la oferta no tiene un trustline para el activo que está comprando.
- sell_not_authorized
  - MANAGE_SELL_OFFER_SELL_NOT_AUTHORIZED
  - La cuenta que crea la oferta no está autorizada para vender este activo.
- buy_not_authorized
  - MANAGE_SELL_OFFER_BUY_NOT_AUTHORIZED
  - La cuenta que crea la oferta no está autorizada para comprar este activo.
- OpLineFull
  - MANAGE_SELL_OFFER_LINE_FULL
  - La cuenta que crea la oferta no tiene los límites suficientes para recibir compras y cumplir con sus responsabilidades de compra.
- OpUnderfunded
  - MANAGE_SELL_OFFER_UNDERFUNDED
  - La cuenta que crea la oferta no tiene los límites suficientes para enviar ventas y cumplir con sus responsabilidades de venta. Ten en cuenta que si estás vendiendo XLM, la cuenta también debe mantener su reserva mínima de XLM, que se calcula asumiendo que esta oferta no se ejecutará completamente de inmediato.
- op_cross_self
  - MANAGE_SELL_OFFER_CROSS_SELF
  - La cuenta tiene una oferta opuesta de precio igual o menor activa, por lo que la cuenta que crea esta oferta inmediatamente se cruzaría a sí misma.
- op_sell_no_issuer
  - MANAGE_SELL_OFFER_SELL_NO_ISSUER
  - El emisor del activo de venta no existe.
- buy_no_issuer
  - MANAGE_SELL_OFFER_BUY_NO_ISSUER
  - El emisor del activo de compra no existe.
- op_offer_not_found
  - MANAGE_SELL_OFFER_NOT_FOUND
  - No se puede encontrar una oferta con ese offerID.
- OpLowReserve
  - MANAGE_SELL_OFFER_LOW_RESERVE
  - La cuenta que crea esta oferta no tiene suficiente XLM para satisfacer el aumento mínimo de reserva de XLM causado por agregar una subentrada y aún cumplir con sus responsabilidades de venta de XLM. Por cada oferta que crea una cuenta, la cantidad mínima de XLM que esa cuenta debe mantener aumentará.

</AttributeTable>
