Crea y financia una nueva cuenta con el saldo inicial especificado.

Consulta los [`errores de crear cuentas`](../../../errors/result-codes/operation-specific/create-account.mdx).

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATO
  - DESCRIPCIÓN
- saldo_inicial
  - cadena
  - La cantidad de XLM a enviar a la cuenta recién creada.
- financista
  - cadena
  - La cuenta que financia la nueva cuenta.
- cuenta
  - cadena
  - Una nueva cuenta que está financiada.

</AttributeTable>
