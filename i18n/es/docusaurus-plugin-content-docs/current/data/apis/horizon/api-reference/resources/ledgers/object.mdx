Cuando Horizon devuelve información sobre un ledger, utiliza el siguiente formato:

<AttributeTable>

- ATRIBUTO
  - TIPO DE DATOS
  - DESCRIPCIÓN
- id
  - cadena
  - Un identificador único para este ledger.
- token_de_paginación
  - número
  - Un valor de cursor para usar en [paginación](../../structure/pagination/README.mdx).
- hash
  - cadena
  - Un hash SHA-256 codificado en hexadecimal de la forma codificada en [XDR](../../../../../../learn/fundamentals/data-format/xdr.mdx) de este libro mayor.
- prev_hash
  - cadena
  - El hash del ledger inmediatamente anterior a este ledger.
- secuencia
  - número
  - El número de secuencia de este ledger y el parámetro usado en las llamadas de Horizon que requieren un número de ledger.
- cuenta_de_transacciones_exitosas
  - número
  - El número de transacciones exitosas en este ledger.
- cuenta_de_transacciones_fallidas
  - número
  - El número de transacciones fallidas en este ledger.
- cuenta_de_operaciones
  - número
  - El número de operaciones aplicadas en este ledger.
- cuenta_de_operaciones_tx_set
  - número
  - El número total de operaciones en el conjunto de transacciones (incluidas las transacciones fallidas).
- cerrado_en
  - cadena
  - Una cadena formateada según [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) que indica cuándo se cerró este ledger.
- monedas_totales
  - cadena
  - El número total de lumens en circulación.
- pool_de_comisiones
  - cadena
  - La suma de todas las tarifas de transacción.
- comisión_base_en_stroops
  - número
  - La tarifa que la red cobra por operación en una transacción.
- reserva_base_en_stroops
  - número
  - La reserva que la red utiliza al calcular el saldo mínimo de una cuenta.
- tamaño_max_tx_set
  - número
  - El número máximo de operaciones que los validadores han acordado procesar en un ledger dado. Desde el Protocolo 11, la capacidad del ledger se ha medido en operaciones en lugar de transacciones. Para más información sobre esa decisión, consulta [CAP-5](https://github.com/stellar/stellar-protocol/blob/master/core/cap-0005.md#repurposing-ledgerheadermaxtxsetsize).
- versión_del_protocolo
  - número
  - La versión del protocolo que la red Stellar estaba ejecutando cuando se comprometió este ledger.
- header_xdr
  - cadena
  - Una cadena codificada base64 de la estructura xdr `LedgerHeader` raw para este ledger.

</AttributeTable>
