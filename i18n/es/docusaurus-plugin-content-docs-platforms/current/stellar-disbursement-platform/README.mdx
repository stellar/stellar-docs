# Desembolso estelar de la plataforma

La Plataforma de Desembolso Estelar (SDP) es una herramienta creada para que las organizaciones realicen pagos masivos a un grupo de destinatarios a través de la red Stellar.

En esta sección, encontrarás una [Guía de Administración](./admin-guide/README.mdx) que te enseñará cómo ejecutar la Plataforma de Desembolso Stellar, así como una [Referencia API](./api-reference/resources/README.mdx).
