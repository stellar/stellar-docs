Un desembolso es un grupo de pagos enviados a múltiples individuos a la vez. Un usuario SDP con el rol apropiado activa un nuevo desembolso a través del panel de control SDP subiendo una lista de receptores y cantidades. Cuando el receptor ha vinculado su cartera al SDP, el pago comienza automáticamente. Los usuarios de SDP pueden rastrear sus desembolsos en tiempo real a través del panel de control SDP. Cada desembolso debe tener un nombre único definido por la organización.
