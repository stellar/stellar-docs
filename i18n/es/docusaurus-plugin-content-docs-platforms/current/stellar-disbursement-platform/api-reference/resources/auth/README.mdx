La autenticación controla el proceso de inicio de sesión/cierre de sesión para todos los usuarios SDP, así como el proceso de actualización de token. La autenticación utiliza un enfoque JWT firmado con una clave privada ES256.

<EndpointsTable title="Endpoints">

|      |                                                |
| ---- | ---------------------------------------------- |
| POST | [/login](../log-in.api.mdx)                    |
| POST | [/refresh-token](../refresh-token.api.mdx)     |
| POST | [/mfa](../autenticate-mfa.api.mdx)             |
| POST | [/forgot-password](../forgot-password.api.mdx) |
| POST | [/reset-password](../reset-password.api.mdx)   |

</EndpointsTable>
