Los extremos de los perfiles gestionan el proceso de obtener y actualizar información individual del perfil. La información del perfil se establece cuando la cuenta es creada y puede ser actualizada por el usuario en la página de perfil del panel de control SDP. Nota: los perfiles nunca se refieren a los receptores de fondos.
