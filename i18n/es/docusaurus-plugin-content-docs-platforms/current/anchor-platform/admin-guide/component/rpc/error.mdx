:::tip

También referenciaremos una variable `$transaction_id`. Esta es una identificación de la transacción que está siendo devuelta desde la Plataforma de Anclaje a petición de un retiro o depósito inicial. Puede obtener el ID de la transacción conectando el monedero de prueba a su instancia local de plataforma de anclaje.

:::
