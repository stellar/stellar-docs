Para habilitar la autenticación de clave API, modifica tu archivo `dev.env`:

<CodeExample>

```bash
# dev.env
PLATFORM_SERVER_AUTH_TYPE=api_key
# Se utilizará como clave API
SECRET_PLATFORM_API_AUTH_SECRET="tu clave API que utilizará el servidor empresarial"
```

</CodeExample>

Después de que esté habilitado, todas las peticiones deben contener una cabecera `X-Api-Key` válida, establecida en la clave API configurada.
