Dirección única es un par único de dirección estelar + memo que se asigna a una nueva transacción [SEP-31].

Al crear una transacción SEP-31, la Plataforma de Anclaje genera automáticamente un par de dirección/memo. Al monitorear la cadena Stellar (por ejemplo, a través de Payment Observer), esta dirección única se utiliza para mapear las transferencias en cadena a la transacción SEP.

Además, es posible definir su propio generador de direcciones (por ejemplo, puede delegarlo en su servicio de custodia). Este callback se utiliza para generar dichas direcciones.
