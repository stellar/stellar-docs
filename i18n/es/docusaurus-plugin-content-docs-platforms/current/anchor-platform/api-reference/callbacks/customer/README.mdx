La Plataforma de Anchor no almacena los datos del cliente KYC. En su lugar, la Plataforma de Anclas reenvía estos datos y los solicita del negocio.

Las llamadas al cliente están diseñadas para este propósito. Actualmente, las solicitudes y respuestas para estos extremos son casi idénticas a la [especificación de la API de KYC SEP-12][sep-12].
