# Admin Guide Overview

This Admin Guide provides comprehensive documentation for setting up, configuring, and managing the Anchor Platform. Whether you're getting started with your first deployment or looking to optimize an existing installation, this guide covers everything you need to know.

## What You'll Learn

The Admin Guide covers:

- **Getting Started** — Initial setup and deployment instructions
- **Architecture** — Understanding the system architecture and components
- **Configuration** — Configuration options for the platform features
- **Component Documentation** — In-depth guides for observers, RPC interfaces, and security
- **Events & Integration** — Event delivery, webhooks, and integration patterns
- **SEP Implementation Guides** — Guides for implementing specific Stellar Ecosystem Proposals

## Getting Started

If you're new to the Anchor Platform, start with the [Getting Started guide](./getting-started.mdx) to set up your first deployment. For a high-level understanding of the system architecture, see the [Architecture documentation](./architecture.mdx).

## Key Topics

- **Security** — Learn about API keys, JWT authentication, and security best practices
- **RPC Interface** — Understand the RPC request/response patterns and error handling
- **Event System** — Configure and integrate with the event delivery system
- **Observers** — Monitor and track transaction statuses and platform events
