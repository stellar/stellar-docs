---
title: What is new in version 4.0
sidebar_position: 30
---

Anchor Platform version 4.0 introduces enhanced capabilities with the following features:

- Support for deposit and withdraw to and from contract accounts for SEP-6 and SEP-24 transactions.
- Support for Stellar contract accounts via the Stellar Asset Contract (SAC), enabling more robust asset management. Discover more details [here](https://developers.stellar.org/docs/tokens/stellar-asset-contract).
- Integration with the Stellar RPC server for improved performance and efficiency. Find out more [here](https://developers.stellar.org/docs/data/apis/rpc).
- If SEP-45 (web authentcation) or contract accounts are enabled, the RPC server connection must be provided.
