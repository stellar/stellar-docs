Platform team demonstrated Galexie, a part of CDP(Composable Data Platform):

1. Galexie
     a. Data Extraction: Extracts raw ledger data from the Stellar network
     b. Compression: Compresses raw data for efficient storage
     c. Storage Options: Supports various storage solutions, starting with Google Cloud Storage (GCS)
     d. Modes of Operation: Can operate in either batch mode or streaming mode
2. Composable Data Platform 
     a. Flexible Datastore:  Multiple options for physical data storage layers
     b. Galexie:  Used to extract, compress and export data to your chosen Datastore
     c. Transform:  Structure data in a model suitable to your application
3. Pluggable Data Pipelines
     a. Workflows:  Create ETL(extract, transform, load) pipelines  
     b. Streaming:  Fast, lightweight streaming data
