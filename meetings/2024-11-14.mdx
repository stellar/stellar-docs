## [Public Discussion](https://discord.com/events/897514728459468821/1304859059425382553/1306725344870400000)

This session featured a walkthrough of the new Stellar Lab, with a focus on recent improvements aimed at streamlining the developer experience when working with Stellar and Soroban.

The demo explored how Stellar Lab supports common development workflows, including account setup, transaction creation, inspection, and interaction with Soroban RPC, highlighting both usability improvements and underlying technical capabilities.

### Key Topics

1. Activate the MultiSig exercise
2. Stellar Wallets Kit
3. Create and fund an account
4. Save transaction function
5. XDR-to-JSON mapping
6. RPC methods, including `simulateTransaction`
