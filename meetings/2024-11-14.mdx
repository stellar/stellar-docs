### [Agenda Thread](https://discord.com/events/897514728459468821/1304859059425382553/1306725344870400000)

In this week's developer meeting, Jeesun demoed the new Stellar Lab, highlighting improved features designed to improve the developer experience.

The team discussed the new Stellar Lab tech stack and walked through these demos:

1. Activate the MultiSig exercise
2. Stellar Wallets Kit
3. Create and fund an account
4. Save transaction function
5. XDR-to-JSON mapping
6. RPC methods, including `simulateTransaction`
